# 紫微斗数命盘排盘方法：从生辰到星盘的完整步骤

紫微斗数作为中国命理学的瑰宝，其排盘方法看似繁复，实则有严密的逻辑体系。掌握排盘，是学习紫微斗数的第一步，也是理解命盘结构的基础。本文将系统讲解从生辰信息到完整命盘的排盘全过程。

## 一、排盘的基本准备

### 1.1 所需信息

排一张紫微斗数命盘，需要以下基本信息：

| 信息项 | 说明 |
|--------|------|
| 出生年 | 农历年份，用于定天干地支 |
| 出生月 | 农历月份，用于安命宫 |
| 出生日 | 农历日期，用于安命宫 |
| 出生时 | 时辰（子时至亥时），用于安命宫 |
| 性别 | 阴阳，用于定大运顺逆 |

> **重要提示**：紫微斗数使用的是农历（阴历），而非公历（阳历）。如手头只有公历日期，需先转换为农历。同时需注意闰月的处理——闰月上半月归上月，下半月归下月。

### 1.2 十二宫位框架

紫微斗数的命盘由十二个宫位组成，排列在固定的位置上：

```
        巳    午    未    申
    辰                  酉
    卯                  戌
    寅    丑    子    亥
```

十二宫位的名称与含义：

| 宫位 | 名称 | 核心含义 |
|------|------|----------|
| 第一宫 | 命宫 | 性格、外貌、人生总体格局 |
| 第二宫 | 兄弟宫 | 兄弟姐妹关系、合作 |
| 第三宫 | 夫妻宫 | 婚姻、感情、配偶 |
| 第四宫 | 子女宫 | 子女、桃花、生育 |
| 第五宫 | 财帛宫 | 财运、收入、理财 |
| 第六宫 | 疾厄宫 | 健康、灾厄 |
| 第七宫 | 迁移宫 | 出行、外出、社会活动 |
| 第八宫 | 交友宫（奴仆宫） | 朋友、下属、人际 |
| 第九宫 | 官禄宫（事业宫） | 事业、学业、工作 |
| 第十宫 | 田宅宫 | 不动产、家庭环境 |
| 第十一宫 | 福德宫 | 精神生活、内心世界 |
| 第十二宫 | 父母宫 | 父母、长辈、庇护 |

## 二、定命宫与身宫

### 2.1 安命宫法

安命宫是排盘的第一步，也是最关键的步骤之一。其方法为：

**口诀**：寅上起正月，顺数至生月，再从生月起子时，逆数至生时。

具体步骤：

1. 从寅宫起正月
2. 顺数至出生的农历月份，此时停留的宫位为"月令宫"
3. 从月令宫起子时
4. **逆数**至出生时辰，所得宫位即为命宫

**示例**：农历三月初五午时生人
- 寅宫起正月，卯宫为二月，辰宫为三月
- 从辰宫起子时，逆数：辰子→卯丑→寅寅→丑卯→子辰→亥巳→戌午
- 命宫在**戌宫**

### 2.2 安身宫法

身宫的安法与命宫类似，唯一区别是第四步为**顺数**：

**口诀**：寅上起正月，顺数至生月，再从生月起子时，**顺数**至生时。

接上例：
- 辰宫起子时，顺数：辰子→巳丑→午寅→未卯→申辰→酉巳→戌午
- 身宫在**戌宫**

> 命身同宫者，其人性格与行为高度一致，人生方向较为明确。

### 2.3 安十二宫

命宫确定后，其余十一宫按固定顺序逆时针排列：

命宫→兄弟→夫妻→子女→财帛→疾厄→迁移→交友→官禄→田宅→福德→父母

## 三、定天干地支

### 3.1 宫位地支

十二宫位的地支是固定的，从寅宫开始逆时针排列：

> 寅、卯、辰、巳、午、未、申、酉、戌、亥、子、丑

### 3.2 宫位天干（五虎遁）

宫位天干根据出生年的天干来定，称为"五虎遁"法：

| 年干 | 寅宫天干 | 口诀 |
|------|----------|------|
| 甲己 | 丙寅 | 甲己之年丙作首 |
| 乙庚 | 戊寅 | 乙庚之岁戊为头 |
| 丙辛 | 庚寅 | 丙辛之岁庚寅起 |
| 丁壬 | 壬寅 | 丁壬壬寅顺水流 |
| 戊癸 | 甲寅 | 戊癸之年甲寅首 |

从寅宫的天干开始，按天干顺序（甲乙丙丁戊己庚辛壬癸）顺排，即可得到每个宫位的天干。

## 四、安十四主星

### 4.1 紫微星系（六颗）

紫微星系的主星由出生日的日数决定：

**定紫微星**：
1. 将农历日数除以30，取商（月数）和余数（日数）
2. 用余数查表确定紫微星所在宫位

紫微星定盘口诀表：

| 日数范围 | 紫微所在 |
|----------|----------|
| 初一至初六 | 对应特定宫位 |
| 初七至十二 | 对应特定宫位 |
| ... | ... |

> 实际操作中，通常使用查表法或软件辅助。手工排盘时可参照"紫微星定盘表"。

紫微星确定后，其余五颗星按固定顺序排列：

| 主星 | 与紫微的相对位置 |
|------|------------------|
| 紫微 | 基准星 |
| 天机 | 紫微逆1宫 |
| 太阳 | 紫微逆3宫 |
| 武曲 | 紫微逆4宫 |
| 天同 | 紫微逆5宫 |
| 廉贞 | 紫微逆8宫 |

### 4.2 天府星系（八颗）

天府星与紫微星呈镜像关系——紫微在寅，天府在申；紫微在卯，天府在未，以此类推。

口诀：**紫微与天府合起来是寅申线上的对称分布**。

天府确定后，其余七颗星按固定顺序排列：

| 主星 | 与天府的相对位置 |
|------|------------------|
| 天府 | 基准星 |
| 太阴 | 天府顺1宫 |
| 贪狼 | 天府顺2宫 |
| 巨门 | 天府顺3宫 |
| 天相 | 天府顺4宫 |
| 天梁 | 天府顺5宫 |
| 七杀 | 天府顺6宫 |
| 破军 | 天府顺10宫 |

### 4.3 主星分布规律

十四主星在命盘中的分布有固定规律：

- **紫微在子午**：命盘结构最对称
- **紫微在寅申**：紫微与天府同宫或对照
- **紫微在辰戌**：天罗地网位，紫微独坐

## 七、排大运与流年

### 7.1 大运排法

大运每十年一换，其起法如下：

1. **阳男阴女**：大运顺行（从命宫起运，顺时针排列）
2. **阴男阳女**：大运逆行（从命宫起运，逆时针排列）

> 阳年：年干为甲丙戊庚壬；阴年：年干为乙丁己辛癸

大运的起始年龄根据命宫与出生月的间距确定。

### 7.2 流年排法

流年每年一换，按照地支顺序排列：

- 流年地支与当年生肖对应
- 流年天干按年干排
- 流年命宫从流年地支起，逆数至流年月份，再逆数至流年时辰

## 八、排盘实例演示

### 完整排盘示例

**命主信息**：甲子年三月初五午时生，男命

**步骤汇总**：

1. ✅ 安命宫：戌宫
2. ✅ 安身宫：戌宫（命身同宫）
3. ✅ 十二宫：命戌→兄弟酉→夫妻申→子女未→财帛午→疾厄巳→迁移辰→交友卯→官禄寅→田宅丑→福德子→父母亥
4. ✅ 五虎遁：甲年→寅宫丙寅，卯宫丁卯，辰宫戊辰...戌宫甲戌
5. ✅ 安紫微：初三月初五→查表得紫微在午
6. ✅ 安天府：与紫微对照在申
7. ✅ 安六吉六煞：按规则排入
8. ✅ 定四化：甲年→廉贞化禄、破军化权、武曲化科、太阳化忌
9. ✅ 排大运：甲年男命→阳男→大运顺行

## 学习要点

1. **安命宫是排盘的根基**，务必掌握"起月逆时"的方法
2. **十四主星的分布有固定规律**，紫微星系逆排、天府星系顺排
3. **五虎遁法**是确定宫位天干的关键，需熟记口诀
4. **四化表必须背诵**，它是命盘动态分析的核心
5. **大运顺逆**取决于年干阴阳与性别
6. 初学阶段建议使用排盘软件验证手工排盘结果
7. 排盘只是工具，**解盘才是紫微斗数的精髓**——排盘准确是解盘正确的前提
8. 注意闰月处理、子时归属（早晚子时）等特殊情况
