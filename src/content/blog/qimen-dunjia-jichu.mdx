# 奇门遁甲基础知识

奇门遁甲，与太乙神数、六壬神课并称"三式"，是中国传统预测学中最玄奥的学问，有"帝王之学"的美誉。相传黄帝战蚩尤时得九天玄女传授奇门遁甲之术，历代帝王将相多有所研究。

---

## 奇门遁甲的核心概念

### "奇、门、遁、甲"的含义

| 字 | 含义 |
|----|------|
| **奇** | 三奇——乙、丙、丁，为吉星 |
| **门** | 八门——开、休、生等，主人事吉凶 |
| **遁** | 隐藏、隐遁 |
| **甲** | 天干之首，代表尊贵，隐于六仪之下 |

> 甲为十干之首，最为尊贵，不直接出现，隐于戊、己、庚、辛、壬、癸六仪之下，故称"遁甲"。

## 奇门遁甲的基本要素

### 一、三奇

| 三奇 | 天干 | 五行 | 含义 |
|------|------|------|------|
| **乙奇** | 乙 | 木 | 文臣、谋略、和缓 |
| **丙奇** | 丙 | 火 | 武将、光明、热烈 |
| **丁奇** | 丁 | 火 | 女将、文书、柔明 |

三奇在奇门遁甲中代表吉祥的力量。

---

### 二、六仪

| 六仪 | 天干 | 五行 |
|------|------|------|
| **戊** | 戊 | 土 |
| **己** | 己 | 土 |
| **庚** | 庚 | 金 |
| **辛** | 辛 | 金 |
| **壬** | 壬 | 水 |
| **癸** | 癸 | 水 |

六仪用于隐藏"甲"，形成六十甲子的循环。

### 三、八门

八门是奇门遁甲判断人事吉凶的核心要素：

| 八门 | 方位 | 吉凶 | 含义 |
|------|------|------|------|
| **开门** | 西北 | 吉 | 开拓、疏通、顺利 |
| **休门** | 北 | 吉 | 休息、休养、稳定 |
| **生门** | 东北 | 吉 | 生机、生长、财源 |
| **伤门** | 东 | 凶 | 伤害、破败、损失 |
| **杜门** | 东南 | 中/凶 | 阻塞、隐藏、困难 |
| **景门** | 南 | 中 | 光明、展示、虚幻 |
| **死门** | 西南 | 凶 | 停滞、死亡、终结 |
| **惊门** | 西 | 凶 | 惊恐、动荡、不安 |

> 吉门：开、休、生
> 凶门：伤、杜、死、惊
> 中平门：景

## 奇门遁甲的排盘

### 时间格局

奇门遁甲以时间为基础排盘：

| 类型 | 说明 |
|------|------|
| 年家奇门 | 以年为一局 |
| 月家奇门 | 以月为一局 |
| 日家奇门 | 以日为一局 |
| 时家奇门 | 以时辰为一局（最常用） |

### 排盘要素

一个完整的奇门盘包含：

1. **地盘**：固定不变的九宫格局
2. **天盘**：时辰天干加临的九宫格局
3. **人盘**：八门的排布
4. **神盘**：八神的排布

### 阴遁与阳遁

| 类型 | 时间 | 特点 |
|------|------|------|
| 阳遁 | 冬至后夏至前 | 顺排，阳长阴消 |
| 阴遁 | 夏至后冬至前 | 逆排，阴长阳消 |

## 奇门遁甲的判断方法

### 基本原则

1. **以时干为主**：时干代表所测之事
2. **三奇为吉**：逢三奇多主吉利
3. **吉门为助**：吉门临之更添吉象
4. **凶门为阻**：凶门临之多有阻滞
5. **格局综合**：星门神仪综合判断

### 判断要素

| 要素 | 作用 |
|------|------|
| 时干与宫位关系 | 判断事务本身吉凶 |
| 八门的状态 | 判断人事顺利与否 |
| 九星的旺衰 | 判断天时助力 |
| 八神的临位 | 判断神秘助力 |
| 格局组合 | 综合判断整体运势 |

## 学习奇门遁甲的建议

1. **打好基础**：先掌握干支五行、八卦九宫
2. **理解逻辑**：奇门有一定的排盘规则，理解比记忆重要
3. **循序渐进**：从时家奇门入手，逐步深入研究
4. **多练习**：多排盘、多分析，培养直觉
5. **辩证看待**：奇门遁甲是工具，不可过度迷信

---

> **学习要点**：奇门遁甲体系复杂，初学时不必贪多求全。建议先理解基本概念——三奇、六仪、八门、九星、八神，再学习排版方法。奇门遁甲重在"格局"的判断，格局的组合千变万化，需要长期积累经验。记住：奇门遁甲的核心是"趋吉避凶"，而非玄之又玄的神秘主义。
