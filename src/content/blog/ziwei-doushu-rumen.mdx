## 一、紫微斗数概述

紫微斗数是中国传统命理学中最精密的推命术之一，以紫微星为核心，配合百余颗星曜，在十二宫位中推演人生运势。

### 1. 历史渊源

| 时期 | 发展 |
|------|------|
| 相传起源 | 宋代陈希夷（陈抟）所创 |
| 宫廷秘传 | 明清时期皇室专用 |
| 现代普及 | 近数十年广泛流传 |

### 2. 紫微斗数的特点

> 紫微斗数被称为"帝王之学"，其特点是：

- **精密细致**：时间精确到时辰（两小时）
- **对应明确**：每颗星都有具体含义
- **体系完整**：涵盖人生各个层面
- **形象直观**：命盘如剧本，星曜如演员

### 3. 与八字的比较

| 对比项 | 紫微斗数 | 八字 |
|--------|----------|------|
| 基础元素 | 星曜 | 干支 |
| 时间单位 | 时辰 | 年月日时 |
| 体系特色 | 重形象 | 重数理 |
| 学习难度 | 入门易精通难 | 入门难精通易 |

## 二、命盘结构

### 1. 十二宫位

紫微斗数将人生划分为十二个领域：

| 宫位 | 名称 | 主管事项 |
|------|------|----------|
| 命宫 | 核心宫位 | 性格、命运、寿命 |
| 兄弟宫 | 手足宫 | 兄弟姐妹、同事朋友 |
| 夫妻宫 | 配偶宫 | 婚姻、配偶、感情 |
| 子女宫 | 子息宫 | 子女、晚辈、生育 |
| 财帛宫 | 财运宫 | 财富、收入、投资 |
| 疾厄宫 | 健康宫 | 身体、疾病、意外 |
| 迁移宫 | 外出宫 | 出行、在外运势 |
| 仆役宫 | 交友宫 | 朋友、下属、合作 |
| 官禄宫 | 事业宫 | 事业、学业、功名 |
| 田宅宫 | 不动产宫 | 房产、家宅、祖业 |
| 福德宫 | 精神宫 | 精神、内心、享受 |
| 父母宫 | 长上宫 | 父母、长辈、上司 |

### 2. 身宫概念

身宫是除命宫外的重要参考点：

> 身宫可能落在命宫、夫妻宫、财帛宫、迁移宫、官禄宫、福德宫之一，代表人生重点关注领域。

| 身宫位置 | 人生重心 |
|----------|----------|
| 命宫 | 自我发展 |
| 夫妻宫 | 感情婚姻 |
| 财帛宫 | 财富积累 |
| 迁移宫 | 外出发展 |
| 官禄宫 | 事业成就 |
| 福德宫 | 精神享受 |

### 3. 三方四正

每宫都有三方四正关系：

| 关系 | 定义 | 作用 |
|------|------|------|
| 三方 | 本宫加左右两邻宫 | 同类事项 |
| 四正 | 对宫（相隔六宫） | 对冲影响 |

示例：命宫的三方四正
- 三方：命宫、财帛宫、官禄宫
- 四正：命宫、迁移宫相对

星曜在不同宫位有不同亮度：

| 状态 | 含义 | 影响 |
|------|------|------|
| 庙 | 最旺 | 吉星增吉，凶星减凶 |
| 旺 | 次旺 | 正面影响 |
| 得地 | 平和 | 正常发挥 |
| 平 | 一般 | 基本发挥 |
| 不得地 | 较弱 | 需助 |
| 落陷 | 最弱 | 吉星减吉，凶星增凶 |
