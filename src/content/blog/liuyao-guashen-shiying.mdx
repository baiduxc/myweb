# 六爻卦身与世应：卦象定位的核心要素

在六爻预测中，卦身、世爻、应爻是定位卦象的核心要素。它们如同地图上的坐标点，帮助我们确定预测对象的主体与客体、自己与他人。本文将系统讲解这三个核心概念的含义与应用方法。

## 一、世爻与应爻的基本概念

### 1. 世爻的含义

世爻，又称"世身"、"世位"，代表卦象中的主体：

| 世爻代表 | 具体含义 |
|----------|----------|
| 求测者本人 | 占卦的人 |
| 主体事物 | 所测之事的主要方 |
| 自己一方 | 在人际关系中代表自己 |
| 主导力量 | 在事件中起主导作用的一方 |

> 《增删卜易》云："世为自己，应为他人。"世爻是预测的核心，是判断吉凶的主要依据。

### 2. 应爻的含义

应爻，又称"应身"、"应位"，代表卦象中的客体：

| 应爻代表 | 具体含义 |
|----------|----------|
| 对方他人 | 与求测者相对的一方 |
| 客体事物 | 所测之事的被动方 |
| 对应方 | 在人际关系中代表对方 |
| 配合力量 | 在事件中起配合作用的一方 |

### 3. 世应的关系

世爻与应爻的位置关系：

| 位置距离 | 含义 |
|----------|------|
| 初爻与四爻 | 距隔三爻，初应四 |
| 二爻与五爻 | 距隔三爻，二应五 |
| 三爻与六爻 | 距隔三爻，三应六 |

世应之间隔三爻，象征着一种对称与呼应的关系。

## 三、卦身的概念与计算

### 1. 卦身的定义

卦身，又称"月卦身"，是根据月建推算出的卦中特殊爻位：

| 卦身含义 | 作用 |
|----------|------|
| 卦象的灵魂 | 代表卦象的核心能量 |
| 事情的要点 | 代表所测事情的关键 |
| 求测者的心声 | 反映内心真实想法 |

### 2. 卦身的计算方法

卦身推算口诀：

> "阴世月卦从初起，阳世还从第四求。"

具体计算方法：

| 世爻阴阳 | 计算方法 |
|----------|----------|
| 阴世爻 | 从初爻顺数至月建数 |
| 阳世爻 | 从四爻顺数至月建数 |

**月建数对照表：**

| 月建 | 对应数字 |
|------|----------|
| 子月 | 1 |
| 丑月 | 2 |
| 寅月 | 3 |
| 卯月 | 4 |
| 辰月 | 5 |
| 巳月 | 6 |
| 午月 | 7 |
| 未月 | 8 |
| 申月 | 9 |
| 酉月 | 10 |
| 戌月 | 11 |
| 亥月 | 12 |

### 3. 卦身计算实例

**例：卯月占卦，世爻在二爻为阴**

| 计算步骤 | 结果 |
|----------|------|
| 判断世爻阴阳 | 二爻为阴，阴世从初起 |
| 从初爻数至卯月数 | 初爻=1，二爻=2，三爻=3，四爻=4（卯月为4） |
| 卦身位置 | 四爻为卦身 |

**例：巳月占卦，世爻在五爻为阳**

| 计算步骤 | 结果 |
|----------|------|
| 判断世爻阴阳 | 五爻为阳，阳世从四起 |
| 从四爻数至巳月数 | 四爻=4，五爻=5，六爻=6（巳月为6） |
| 卦身位置 | 六爻为卦身 |

## 四、世应卦身的实际应用

### 1. 世爻的判断要点

| 世爻状态 | 判断原则 |
|----------|----------|
| 世爻旺相 | 利于自己，事情顺利 |
| 世爻休囚 | 不利自己，事情艰难 |
| 世爻空亡 | 自己犹豫，事情不定 |
| 世爻冲破 | 自己受损，事情有变 |
| 世爻持身 | 事情与本人密切相关 |

### 2. 应爻的判断要点

| 应爻状态 | 判断原则 |
|----------|----------|
| 应爻旺相 | 对方有力，可得帮助 |
| 应爻休囚 | 对方无力，难以依靠 |
| 应爻空亡 | 对方不实，事不落实 |
| 应爻冲破 | 对方有变，关系破裂 |
| 应爻生世 | 对方助己，有贵人 |

### 3. 世应关系的判断

| 世应关系 | 断事原则 |
|----------|----------|
| 世应相生 | 双方和谐，事易成 |
| 世应相克 | 双方对立，事难成 |
| 世应相冲 | 双方冲突，必有争 |
| 世应相合 | 双方合作，事可成 |
| 世强应弱 | 自己主动，占据主导 |
| 应强世弱 | 对方主动，处于被动 |

### 4. 卦身的应用方法

| 卦身状态 | 判断原则 |
|----------|----------|
| 卦身旺相 | 事情有力，易成功 |
| 卦身休囚 | 事情无力，难成功 |
| 卦身空亡 | 事情虚假，不落实 |
| 卦身发动 | 事情有变，需注意 |
| 卦身与世相同 | 事情与自己密切相关 |
| 卦身与世不同 | 事情另有隐情 |

## 五、不同事项的世应取法

### 1. 婚姻预测

| 事项 | 世爻代表 | 应爻代表 |
|------|----------|----------|
| 婚姻占问 | 求测者本人 | 婚姻对象 |
| 男占 | 男方自己 | 妻子女方 |
| 女占 | 女方自己 | 丈夫男方 |

**判断要点：**
- 世应相生相合，婚姻和谐
- 世应相冲相克，婚姻不顺
- 应爻旺相，对象条件好
- 应爻空亡，对象不实或无缘

### 2. 求财预测

| 事项 | 世爻代表 | 应爻代表 |
|------|----------|----------|
| 求财占问 | 求财者本人 | 财源财主 |
| 经营占问 | 经营者 | 合作方/市场 |

**判断要点：**
- 世爻旺相，有求财能力
- 财爻生世，财运亨通
- 应爻生世，财源助力
- 世应相冲，财有阻碍

### 3. 求职预测

| 事项 | 世爻代表 | 应爻代表 |
|------|----------|----------|
| 求职占问 | 求职者本人 | 用人单位 |
| 考试占问 | 考生自己 | 考试结果 |

**判断要点：**
- 世爻旺相，有胜任能力
- 应爻旺相，单位条件好
- 官爻生世，职位可得
- 世应相合，双方契合

### 4. 疾病预测

| 事项 | 世爻代表 | 应爻代表 |
|------|----------|----------|
| 疾病占问 | 患者本人 | 疾病本身 |
| 代人占问 | 占卦者 | 患者本人 |

**判断要点：**
- 世爻休囚，体质虚弱
- 世爻旺相，有康复能力
- 官鬼为病，需看官爻
- 应爻凶险，疾病严重

### 5. 出行预测

| 事项 | 世爻代表 | 应爻代表 |
|------|----------|----------|
| 出行占问 | 出行者本人 | 目的地 |
| 行人占问 | 占卦者 | 行人本人 |

**判断要点：**
- 世爻旺相，出行顺利
- 应爻凶险，目的地有问题
- 世应相冲，出行受阻
- 世空应实，出行不实或推迟

## 六、世应卦身的高级应用

### 1. 世应飞伏

当世应爻位有伏神时：

| 情况 | 判断原则 |
|------|----------|
| 世下伏神 | 暗中有隐藏因素影响自己 |
| 应下伏神 | 对方另有隐情 |
| 伏神吉 | 隐藏有利因素 |
| 伏神凶 | 隐藏不利因素 |

### 2. 世应进退

世应爻的进退判断：

| 进退状态 | 含义 |
|----------|------|
| 世进 | 自己向前发展 |
| 世退 | 自己退缩观望 |
| 应进 | 对方积极进取 |
| 应退 | 对方退缩消极 |

### 3. 世应变爻

当世应爻有变爻时：

| 变爻情况 | 判断原则 |
|----------|----------|
| 世爻变吉 | 自己有好转 |
| 世爻变凶 | 自己有变化不利 |
| 应爻变吉 | 对方有转变向好 |
| 应爻变凶 | 对方有转变不利 |
| 世应互变 | 双方关系转变 |

## 学习要点总结

1. **世爻代表主体自己**：是判断吉凶的核心，世旺有利自己，世休不利自己。

2. **应爻代表客体对方**：是对应力量的体现，应旺对方有力，应休对方无力。

3. **世应隔三爻对称呼应**：初应四、二应五、三应六，形成对称关系。

4. **卦身是卦象的灵魂**：根据世爻阴阳和月建推算，代表事情的核心要点。

5. **不同事项世应取法不同**：婚姻、求财、求职、疾病等各有不同的世应代表。

6. **世应关系决定事情走向**：相生相合为顺，相冲相克为难，综合判断才能准确。
