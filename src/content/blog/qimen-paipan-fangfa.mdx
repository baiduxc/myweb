## 一、奇门排盘概述

奇门遁甲排盘是将天地人三盘有机结合的过程，需要掌握时辰、节气、局数、宫位等要素的准确计算。

### 排盘的基本框架

| 盘类 | 内容 | 作用 |
|------|------|------|
| 天盘 | 三奇六仪 | 表示天时 |
| 地盘 | 三奇六仪 | 表示地利 |
| 人盘 | 八门 | 表示人和 |
| 神盘 | 八神 | 表示神助 |

## 三、排盘完整步骤

### 步骤一：确定时间要素

| 要素 | 内容 |
|------|------|
| 年 | 如2026年 |
| 月 | 如4月 |
| 日 | 如29日 |
| 时 | 如8时 |
| 日干支 | 需查万年历 |
| 时干支 | 根据"五鼠遁"推算 |

### 步骤二：确定局数

| 方法 | 步骤 |
|------|------|
| 1 | 确定节气 |
| 2 | 判断阴阳遁 |
| 3 | 确定上中下元 |
| 4 | 查表得出局数 |

### 步骤三：地盘排布

**阳遁排法**：戊己庚辛壬癸丁丙乙，依次顺排入宫。

| 局数 | 坎一宫起 |
|------|----------|
| 一局 | 戊起坎一 |
| 二局 | 戊起坤二 |
| 三局 | 戊起震三 |
| ...依此类推 |

**阴遁排法**：逆排六仪，顺序不同。

### 步骤四：天盘排布

根据时干定值符，值符加临地盘时干所在宫位：

| 天盘排布原则 |
|---------------|
| 值符随时干 |
| 天盘顺转（阳遁） |
| 天盘逆转（阴遁） |

### 步骤五：人盘排布（八门）

| 八门排布原则 |
|---------------|
| 值使随时支 |
| 顺排（阳遁） |
| 逆排（阴遁） |

### 步骤六：神盘排布（八神）

| 神盘排布原则 |
|---------------|
| 值符随天盘值符 |
| 顺布（阳遁） |
| 逆布（阴遁） |

## 六、排盘的注意事项

### 1. 时间准确性

| 注意点 | 说明 |
|--------|------|
| 真太阳时 | 需按当地真太阳时计算 |
| 节气交接 | 节气交接时需特别注意 |
| 时辰分界 | 23点为子时开始 |

### 2. 置闰与拆补

| 方法 | 特点 |
|------|------|
| 置闰法 | 传统经典，超神接气 |
| 拆补法 | 现代常用，按节气分割 |

### 3. 排盘检验

| 检验项目 | 方法 |
|----------|------|
| 局数正确 | 核对节气与元数 |
| 六仪顺序 | 检查戊己庚辛壬癸顺序 |
| 值符正确 | 确认旬首对应 |
| 方位正确 | 核对九宫方位 |

## 七、常见排盘误区

### 误区一：时间计算错误

| 问题 | 解决方法 |
|------|----------|
| 阴历阳历混淆 | 统一使用阳历 |
| 时辰计算错误 | 使用五鼠遁口诀 |
| 节气判断偏差 | 参考万年历 |

### 误区二：阴阳遁混淆

| 问题 | 解决方法 |
|------|----------|
| 记错分界 | 冬至后阳遁，夏至后阴遁 |
| 排盘方向相反 | 阳遁顺排，阴遁逆排 |

### 误区三：旬首判断错误

| 问题 | 解决方法 |
|------|----------|
| 旬首不准 | 熟记六旬：甲子戊、甲戌己、甲申庚、甲午辛、甲辰壬、甲寅癸 |
