用神是六爻预测的核心枢纽，取准用神是一切断卦分析的前提。用神取错，则后续的旺衰、生克、动静分析全部偏离，断卦自然不准。

## 用神的概念

用神即代表**所问之事**的那个爻（六亲之一）。

> 问何事，取何爻为用神，这是六爻预测的第一原则。

### 六亲与用神对应

| 六亲 | 基本含义 | 代表事项 |
|------|----------|----------|
| **父母** | 文书、庇护 | 房屋、文书、考试、合同、长辈 |
| **兄弟** | 同辈、竞争 | 朋友、同事、竞争对手、合伙人 |
| **妻财** | 财物、欲望 | 钱财、生意、利润、妻子、下属 |
| **子孙** | 子息、福禄 | 子女、晚辈、医药、解忧、福气 |
| **官鬼** | 官府、约束 | 官司、工作、丈夫、盗贼、疾病 |

| 讨债 | 妻财爻 | 财爻旺相有气则能讨回 |
| 投资 | 妻财爻 | 财爻生合世爻则有利 |

| 考试 | 父母爻 | 父母为文书成绩 |
| 创业 | 官鬼为主，妻财为辅 | 官为事业，财为利润 |

## 用神取法详解

### 世爻与用神

| 情况 | 用神取法 |
|------|----------|
| 为自己问事 | 世爻为自己，另取用神 |
| 代人问事 | 按关系取六亲为用神 |
| 世爻即为用神 | 问自身安危时，世爻即为用神 |

### 用神两现

当卦中用神出现两个以上时的处理：

| 情况 | 取法 |
|------|------|
| 一静一动 | 取动爻为用神 |
| 两爻皆动 | 取旺相者为用神 |
| 两爻皆静 | 取旺相或临世爻者为用神 |
| 用神不上卦 | 看伏神（本宫六亲伏藏） |

### 用神不上卦

若用神不在主卦中出现，需看**伏神**：

1. 找出本宫卦中对应的六亲
2. 伏在本卦某爻之下
3. 伏神旺相且得飞神生扶，则有用
4. 伏神休囚且被飞神克压，则无用

## 用神旺衰判断

取准用神后，需判断用神的旺衰：

| 判断维度 | 说明 |
|----------|------|
| 月建生克 | 月令为提纲，得月令生扶为旺 |
| 日辰生克 | 日辰为辅助，得日生扶为有气 |
| 动爻生克 | 动爻对用神的生克作用 |
| 变爻影响 | 用神发动变出之爻的生克 |

### 旺衰等级

| 等级 | 条件 | 断语 |
|------|------|------|
| 旺相 | 得月令、得日辰 | 有力可成 |
| 休囚 | 失月令、不得日 | 力量不足 |
| 旬空 | 用神落空亡 | 暂时无力，出空有用 |
| 入墓 | 用神入日墓/月墓 | 被困住，冲墓则出 |
| 伏藏 | 用神不上卦 | 暗中存在，待引动 |

## 用神与原神忌神

### 三神关系

| 神 | 含义 | 作用 |
|----|------|------|
| **用神** | 代表所问之事 | 核心分析对象 |
| **原神** | 生用神之爻 | 帮助用神，宜旺不宜衰 |
| **忌神** | 克用神之爻 | 伤害用神，宜衰不宜旺 |

### 吉凶判断

| 情况 | 吉凶 |
|------|------|
| 用神旺相+原神旺相 | 大吉 |
| 用神旺相+忌神旺相 | 先吉后凶 |
| 用神休囚+原神旺相 | 待时而吉 |
| 用神休囚+忌神旺相 | 大凶 |
| 用神旺相+原神休囚 | 减吉 |
| 用神休囚+原神休囚 | 无力难成 |

---

> **学习要点**：取用神是六爻断卦的第一步也是最关键的一步。初学时务必把各类事项的用神取法记熟，遇到复杂情况时，先想清楚"我到底问的是什么"，再对应取用。取准用神后，原神忌神的判断就是顺理成章的事了。
