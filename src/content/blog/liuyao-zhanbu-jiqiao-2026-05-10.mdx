# 六爻占卜实战技巧：从起卦到断卦的完整流程

六爻占卜是中国传统预测学中应用最广泛的方法之一，以其灵活、准确、实用著称。本文将详细介绍六爻占卜的完整流程和实战技巧。

---

## 一、六爻占卜的基本原理

### 1.1 什么是六爻

六爻占卜，又称"纳甲筮法"、"火珠林法"，是用三枚铜钱起卦，通过分析卦象来预测事物发展。

> 六爻以"用神"为核心，分析世应、五行、六亲等关系。

### 1.2 六爻的特点

| 特点 | 说明 |
|------|------|
| 灵活性 | 可以预测任何具体事项 |
| 准确性 | 理论成熟，应验率高 |
| 实用性 | 起卦简单，分析直观 |
| 系统性 | 有完整的理论体系 |

### 1.3 六爻的基本构成

| 要素 | 说明 |
|------|------|
| 卦象 | 六十四卦之一 |
| 爻象 | 六个爻，各有阴阳 |
| 地支 | 每爻配一地支 |
| 五行 | 每爻有五行属性 |
| 六亲 | 根据卦宫确定六亲 |
| 世应 | 世爻代表自己，应爻代表对方 |

## 二、起卦方法

### 2.1 铜钱起卦法

**准备工作**：
- 三枚铜钱（或硬币）
- 记录纸笔
- 安静的环境

**起卦步骤**：

> 心念专一，想着要问的事情，将三枚铜钱合在手中摇动，然后掷出。

**记录方法**：

| 铜钱组合 | 爻象 | 记录 |
|----------|------|------|
| 一背二面 | 少阳 | — |
| 二背一面 | 少阴 | -- |
| 三背 | 老阳 | O（阳动） |
| 三面 | 老阴 | ×（阴动） |

**摇卦六次**：从下往上记录，第一次为初爻，第六次为上爻。

### 2.2 时间起卦法

**方法**：
- 年支数 + 月数 + 日数，除以8，余数为上卦
- 年支数 + 月数 + 日数 + 时数，除以8，余数为下卦
- 总数除以6，余数为动爻

### 2.3 数字起卦法

**方法**：
- 用任意两个数字
- 第一个数字除以8，余数为上卦
- 第二个数字除以8，余数为下卦
- 两数之和除以6，余数为动爻

## 四、断卦核心要素

### 4.1 用神的概念

**用神**：代表所占之事的六亲。

| 占问事项 | 用神 |
|----------|------|
| 求财 | 妻财 |
| 考试 | 父母 |
| 求职 | 官鬼 |
| 疾病 | 官鬼（症）、子孙（药） |
| 婚姻 | 妻财（男测）、官鬼（女测） |
| 子女 | 子孙 |
| 出行 | 父母 |

### 4.2 世应的作用

| 爻位 | 代表 | 作用 |
|------|------|------|
| 世爻 | 自己、我方 | 断卦的立足点 |
| 应爻 | 对方、他方 | 断卦的参考点 |

**世应关系**：
- 世生应：我方主动，付出
- 应生世：对方主动，我方受益
- 世克应：我方占优
- 应克世：对方占优
- 世应相合：双方和谐

### 4.3 日月的作用

**月建**：旺相休囚的判断标准

| 状态 | 说明 |
|------|------|
| 旺 | 被月建生扶 |
| 相 | 与月建同类 |
| 休 | 生月建者 |
| 囚 | 克月建者 |
| 死 | 被月建克制 |

**日辰**：决定爻的生克权

- 日辰生扶：增力
- 日辰克制：减力
- 爻逢日冲：暗动或冲散

### 4.4 动爻的作用

**动爻**：发生变化的爻

| 动爻状态 | 含义 |
|----------|------|
| 动而化进 | 增强 |
| 动而化退 | 减弱 |
| 动而化回头生 | 得到支持 |
| 动而化回头克 | 受到制约 |
| 动而化空化破 | 变化无效 |

## 五、断卦流程

### 5.1 标准断卦流程

```
第一步：核实卦象，确认无误
第二步：确定用神
第三步：看用神的旺衰
第四步：分析世应关系
第五步：看动爻对用神的影响
第六步：综合判断吉凶
第七步：推断应期
```

### 5.2 吉凶判断标准

| 用神状态 | 判断 |
|----------|------|
| 用神旺相有生扶 | 吉 |
| 用神休囚受克 | 凶 |
| 用神伏藏不现 | 有待寻找 |
| 用神旬空月破 | 需等填实 |

### 5.3 应期推断

**应期**：预测结果应验的时间

| 情况 | 应期推断 |
|------|----------|
| 用神休囚 | 待旺相之时 |
| 用神旬空 | 待出空之时 |
| 用神伏藏 | 待冲开之时 |
| 用神被合 | 待冲开之时 |
| 用神被冲 | 待合住之时 |

## 六、实战案例分析

### 6.1 求财案例

**问题**：某人问近期投资能否获利？

**卦象**：
```
公历时间：某年某月某日
农历时间：某月某日
干支：甲子年 丁丑月 戊寅日
卦象：风天小畜
```

**分析步骤**：

1. **确定用神**：求财以妻财为用
2. **查找用神**：卦中妻财在二爻
3. **分析旺衰**：妻财丑土，月建丑土旺
4. **看动爻**：无动爻
5. **世应关系**：世爻代表自己，得月令生扶
6. **综合判断**：用神旺相，有财可求

**结论**：近期投资可以获利。

### 6.2 求职案例

**问题**：某人问能否考上公务员？

**卦象分析**：

1. **用神**：考试以父母爻为用（代表文书）
2. **官鬼**：也需看官鬼（代表职位）
3. **旺衰分析**：用神得月令生扶
4. **动爻影响**：看是否有动爻相生
5. **综合判断**：用神旺相，有利

## 七、六爻占卜的注意事项

### 7.1 起卦注意事项

| 注意点 | 说明 |
|--------|------|
| 心念专一 | 不要心不在焉 |
| 问题明确 | 问题要具体 |
| 时间适宜 | 不在极端天气起卦 |
| 环境安静 | 避免干扰 |

### 7.2 断卦注意事项

| 注意点 | 说明 |
|--------|------|
| 用神准确 | 不同事项用神不同 |
| 综合分析 | 不能只看一点 |
| 因人而异 | 结合问卦人情况 |
| 客观理性 | 不夸大不贬低 |

### 7.3 常见误区

| 误区 | 正确做法 |
|------|----------|
| 死记硬背 | 理解原理 |
| 孤立看爻 | 整体分析 |
| 忽视日月 | 日月为判断依据 |
| 过于迷信 | 保持理性 |
