# 六爻预测进阶技法：卦象变化的深层解读

掌握基础的六爻预测后，进阶技法是提升预测准确率的关键。进神退神、反吟伏吟、三合三会、冲合刑害等概念，是六爻预测的深层密码。本文将系统讲解这些进阶技法。

## 一、进神与退神

### 1.1 进神退神的定义

**进神**：卦爻由休囚变为旺相，或由弱转强，称为进神。

**退神**：卦爻由旺相变为休囚，或由强转弱，称为退神。

> 进神代表事物向好发展，退神代表事物向坏发展。

### 1.2 进神的判断标准

| 情况 | 判断 | 断语 |
|------|------|------|
| 爻逢长生 | 进神 | 事情开始好转 |
| 爻逢帝旺 | 进神 | 事情达到顶峰 |
| 爻逢临官 | 进神 | 事业有成 |
| 月建生合 | 进神 | 贵人相助 |
| 日辰生合 | 进神 | 近期好转 |

### 1.3 退神的判断标准

| 情况 | 判断 | 断语 |
|------|------|------|
| 爻逢死绝 | 退神 | 事情走向衰败 |
| 爻逢墓库 | 退神 | 事情停滞不前 |
| 月建冲克 | 退神 | 阻碍加重 |
| 日辰冲克 | 退神 | 近期恶化 |

### 1.4 进退神的实战应用

**测事业**：
- 官鬼爻进神：事业上升，升职有望
- 官鬼爻退神：事业下滑，需防失职

**测财运**：
- 妻财爻进神：财运好转，收入增加
- 妻财爻退神：财运下降，需节俭

**测婚姻**：
- 应爻进神：对方态度好转
- 应爻退神：对方热情减退

## 二、反吟与伏吟

### 2.1 反吟卦

**定义**：卦变后，六爻爻位全部变为对冲之爻，称为反吟卦。

例如：乾卦变坤卦（六爻全变对冲）

**反吟卦特征**：

| 特征 | 说明 |
|------|------|
| 内外反对 | 内卦与外卦变化相反 |
| 事情反复 | 主事情反复无常 |
| 好事变坏 | 原本好事可能变坏 |
| 坏事变好 | 原本坏事可能变好 |

**反吟卦断语**：

| 测事 | 断语 |
|------|------|
| 测事业 | 事业反复，需防变故 |
| 测婚姻 | 婚姻反复，需防反悔 |
| 测出行 | 出行不顺，需防反复 |
| 测疾病 | 病情反复，需防复发 |

### 2.2 伏吟卦

**定义**：卦变后，六爻爻位不变，但地支变化，称为伏吟卦。

例如：乾卦变乾卦（地支变化但卦象不变）

**伏吟卦特征**：

| 特征 | 说明 |
|------|------|
| 爻位不变 | 变爻后仍在原位 |
| 忧郁呻吟 | 主内心郁闷、压抑 |
| 事情迟滞 | 主事情进展缓慢 |
| 内心矛盾 | 主内心反复纠结 |

**伏吟卦断语**：

| 测事 | 断语 |
|------|------|
| 测事业 | 事业停滞，内心煎熬 |
| 测婚姻 | 婚姻不顺，内心矛盾 |
| 测疾病 | 病情缠绵，难愈 |
| 测出行 | 出行受阻，反复不定 |

### 2.3 反吟与伏吟的化解

| 情况 | 化解方法 |
|------|----------|
| 反吟卦 | 需灵活应变，不要执着 |
| 伏吟卦 | 需调整心态，不要焦虑 |
| 吉事逢吟 | 吉中有忧 |
| 凶事逢吟 | 凶中有变 |

## 三、三合与三会

### 3.1 三合局

**定义**：三个地支合成一个五行局，称为三合局。

| 三合局 | 地支 | 化成 |
|--------|------|------|
| 水局 | 申子辰 | 水 |
| 木局 | 亥卯未 | 木 |
| 火局 | 寅午戌 | 火 |
| 金局 | 巳酉丑 | 金 |

**三合局的条件**：
1. 三个地支必须齐全
2. 三合局必须有一支在卦中发动
3. 三合局需有中神（子、卯、午、酉）临值或发动

**三合局的作用**：

| 测事 | 三合局断语 |
|------|-----------|
| 测事业 | 三合官鬼，事业大成 |
| 测财运 | 三合妻财，财源广进 |
| 测婚姻 | 三合应爻，婚姻美满 |
| 测合作 | 三合有利合作 |

### 3.2 三会局

**定义**：三个同方地支聚在一起，称为三会局。

| 三会局 | 地支 | 方位 |
|--------|------|------|
| 东方木局 | 寅卯辰 | 东 |
| 南方火局 | 巳午未 | 南 |
| 西方金局 | 申酉戌 | 西 |
| 北方水局 | 亥子丑 | 北 |

**三会局的作用**：
- 力量比三合局更强
- 代表方位之气的汇聚
- 三会局成，主大事可成

### 3.3 三合三会的实战

**案例分析**：

测合伙做生意，得卦：

卦中有申、子两支，另有辰土发动：
- 申子辰三合水局
- 水为财星（若财星属水）
- 断：合伙可成，财运亨通

## 四、冲合刑害详解

### 4.1 冲

**定义**：两个地支相互对冲，称为冲。

| 六冲 | 说明 |
|------|------|
| 子午冲 | 水火相冲 |
| 丑未冲 | 土土相冲 |
| 寅申冲 | 木金相冲 |
| 卯酉冲 | 木金相冲 |
| 辰戌冲 | 土土相冲 |
| 巳亥冲 | 火水相冲 |

**冲的效应**：

| 测事 | 冲断语 |
|------|--------|
| 冲用神 | 用神被冲，事情有变 |
| 冲原神 | 原神被冲，根基不稳 |
| 冲忌神 | 忌神被冲，灾祸可免 |
| 月冲 | 长期有变 |
| 日冲 | 近期有变 |
| 动冲 | 突发有变 |

**冲的化解**：

| 情况 | 化解 |
|------|------|
| 冲用神 | 需有合来解冲 |
| 冲忌神 | 冲去忌神是好事 |
| 日月同冲 | 需小心谨慎 |

### 4.2 合

**定义**：两个或三个地支相合，称为合。

**六合**：

| 六合 | 化成 |
|------|------|
| 子丑合 | 土 |
| 寅亥合 | 木 |
| 卯戌合 | 火 |
| 辰酉合 | 金 |
| 巳申合 | 水 |
| 午未合 | 火/土 |

**合的效应**：

| 测事 | 合断语 |
|------|--------|
| 合用神 | 用神得合，事情顺利 |
| 合原神 | 原神得合，根基稳固 |
| 合忌神 | 忌神被合，灾祸减轻 |
| 静合 | 自然成就 |
| 动合 | 主动追求 |

**冲与合的关系**：

> 合可解冲，冲可破合
> 冲合相遇，先冲后合或先合后冲，需看何者有力

## 五、特殊卦象

### 5.1 归魂卦

**定义**：外卦变为内卦的对立面，最后又变回来的卦。

**特征**：
- 主事情反复
- 主归家、回归
- 测出行主难行

### 5.2 游魂卦

**定义**：卦象游走不定，变爻后内外卦互换。

**特征**：
- 主心神不宁
- 主出行、游走
- 测安定主难安

### 5.3 桃花卦

**定义**：卦中桃花星（子、午、卯、酉）多现。

**特征**：
- 主感情之事
- 主异性缘
- 测婚姻需细看

## 六、进阶技法综合应用

### 6.1 复杂案例一

**测合伙做生意**

卦象：内卦申子辰三合水局，外卦巳酉丑三合金局

**分析**：
- 内卦三合水局，主合伙顺利
- 外卦三合金局，金生水，外助内
- 两局相生，合伙大吉

**断语**：合伙可成，生意兴隆。

### 6.2 复杂案例二

**测婚姻**

卦象：用神妻财被月建冲，但有日辰合

**分析**：
- 月冲用神，有阻碍
- 日合用神，有转机
- 冲合相遇，需看日辰力量

**断语**：婚姻有阻碍，但可化解，最终可成。

## 学习要点

1. **进神退神**判断事情的吉凶走向
2. **反吟伏吟**预测事情的反复与停滞
3. **三合三会**代表事情的大势所趋
4. **冲合刑害**是爻与爻之间的关系，需细细分析
5. **冲可解合，合可解冲**，需看力量大小
6. 进阶技法需**综合运用**，不可执一而定
7. **实战练习**是提升预测准确率的关键
8. 复杂卦象需**层层分析**，从整体到局部
