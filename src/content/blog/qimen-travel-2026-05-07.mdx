## 奇门遁甲与出行预测

出行预测是奇门遁甲的重要应用领域之一。古人远行多需翻山越岭、长途跋涉，风险较大，因此出行前往往起局占测，趋吉避凶。在现代，出行预测同样适用，无论是商务出差、旅游度假还是日常通勤，都可以通过奇门盘局判断行程吉凶。

### 出行预测的核心要素

| 要素 | 对应符号 | 命理含义 |
|------|----------|----------|
| 出行人 | 日干、年命 | 判断当事人状态 |
| 出行目的地 | 目的地方位 | 用神落宫 |
| 出行工具 | 景门、九天 | 交通工具状态 |
| 旅途安全 | 八门、九星 | 顺利与否 |
| 出行时机 | 时干、值符 | 最佳出发时间 |

## 用神选取方法

### 主客关系定用神

在出行预测中，首先要分清主客关系：

**以日干为出行人**：
- 日干代表求测者本人的状态
- 日干落宫的吉凶判断出行顺利程度
- 日干所临星门神反映出行状态

**以时干为出行之事**：
- 时干代表出行的具体事件
- 时干与日干的关系判断成败
- 时干临吉门吉格，出行顺利

> 日干为己时干事，主客分明事可推。
> 出行先看日干落，再看时干定是非。

根据出行目的地方向选取用神：

| 方位 | 对应宫位 | 五行属性 |
|------|----------|----------|
| 东方 | 震三宫 | 木 |
| 东南 | 巽四宫 | 木 |
| 南方 | 离九宫 | 火 |
| 西南 | 坤二宫 | 土 |
| 西方 | 兑七宫 | 金 |
| 西北 | 乾六宫 | 金 |
| 北方 | 坎一宫 | 水 |
| 东北 | 艮八宫 | 土 |

## 吉凶判断方法

### 八门与出行

| 八门 | 出行判断 |
|------|----------|
| 开门 | 出行大吉，事业顺利 |
| 休门 | 出行平稳，适合休息 |
| 生门 | 出行吉利，有收获 |
| 伤门 | 出行有风险，可能受伤 |
| 杜门 | 出行受阻，不宜远行 |
| 景门 | 出行顺利，有美景 |
| 死门 | 出行大凶，有危险 |
| 惊门 | 出行惊恐，多是非 |

**出行吉利格局**：
- 日干临开门、生门、休门
- 日干临吉星（天辅、天心、天任）
- 日干临值符、六合

**出行凶险格局**：
- 日干临死门、惊门、伤门
- 日干临凶星（天芮、天柱）
- 日干入墓、击刑、门迫

## 具体出行类型分析

### 商务出差

商务出差主要关注事业运势：

| 判断要点 | 具体内容 |
|----------|----------|
| 开门状态 | 代表事业，开门吉则事业顺 |
| 时干落宫 | 代表具体事项成败 |
| 日干旺衰 | 代表个人状态和能力 |
| 目的地宫 | 目的地方位的吉凶 |

**商务出差吉利标志**：
- 开门临吉星吉格
- 日干生开门，事业有成
- 时干临生门，有收获
- 目的地宫临吉门

### 旅游度假

旅游度假主要关注旅途愉快和安全：

| 判断要点 | 具体内容 |
|----------|----------|
| 景门状态 | 代表旅游、景观 |
| 休门状态 | 代表休息、娱乐 |
| 九天状态 | 代表出行、远方 |
| 杜门状态 | 代表阻碍、不通 |

**旅游吉利标志**：
- 景门临吉星，风景优美
- 休门临吉格，休息充分
- 九天临吉门，旅途顺利
- 杜门空亡，无阻碍

### 探亲访友

探亲访友主要关注人际关系：

**吉利标志**：
- 六合临吉门，人际关系好
- 日干生时干，主动积极
- 时干临开门、生门，访友顺利
- 目的地宫临吉星

### 求医看病

求医出行主要关注健康运势：

| 判断要点 | 具体内容 |
|----------|----------|
| 天芮状态 | 代表疾病，凶则病情重 |
| 生门状态 | 代表生机，吉则康复有望 |
| 死门状态 | 代表凶险，临则危险 |
| 天心状态 | 代表医生，吉则遇良医 |

## 出行安全防范

### 判断出行风险

**高风险标志**：
- 日干临死门、伤门
- 日干入墓、击刑
- 白虎临日干
- 日干落空亡

**应对措施**：
- 改变出行时间
- 更换出行方式
- 增加同行人数
- 做好安全准备

### 出行择时

**出行最佳时机**：
- 日干临值符时
- 时干临吉门时
- 出行方向宫临吉星时

**出行忌讳时机**：
- 大时凶日
- 五不遇时
- 时干入墓时

## 实战案例分析

### 案例一：商务出差

某商人欲赴外地谈生意，起局占测：

- 日干丙火落兑宫，临开门
- 开门属金，丙火克金
- 时干辛金落震宫，临生门
- 目的地在西方，兑宫有吉星

**分析**：
- 日干临开门，事业之门打开
- 丙火克开门，说明需要努力
- 时干临生门，此行有收获
- 目的地宫临吉星，顺利

**结果**：此行成功签下订单。

### 案例二：旅游出行

某家庭计划假期出游，起局占测：

- 日干戊土落坎宫，临杜门
- 杜门主受阻，但空亡
- 景门落离宫，临天辅星
- 九天落巽宫，临开门

**分析**：
- 杜门空亡，阻碍消除
- 景门临吉星，风景优美
- 九天临开门，旅途顺利

**结果**：旅途顺利，风景优美，全家满意。

## 学习要点总结

1. 出行预测以日干为出行人，时干为出行之事
2. 八门反映出行的顺利程度，吉门宜出行
3. 九星代表出行的性质和状态
4. 格局吉凶直接影响出行成败
5. 出行类型不同，关注重点各有侧重
6. 凶险格局应及时采取化解措施
7. 出行择时可趋吉避凶
