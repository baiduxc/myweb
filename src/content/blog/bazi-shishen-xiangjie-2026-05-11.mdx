## 十神概述

十神是八字命理学中最重要的分析工具之一，它是根据日主（日干）与其他干支的五行关系推导出来的。十神代表了人生中的各种人事关系和社会现象，是连接八字与现实的桥梁。掌握十神是学好八字的关键。

<Note title="十神的由来">
十神由日主与其他干支的生克关系产生。同我者为比劫，生我者为印绶，我生者为食伤，我克者为财星，克我者为官杀。
</Note>

## 十神的基本概念

### 十神推导规则

| 日主关系 | 阴阳相同 | 阴阳不同 |
|----------|----------|----------|
| 同我 | 比肩 | 劫财 |
| 生我 | 偏印 | 正印 |
| 我生 | 食神 | 伤官 |
| 我克 | 偏财 | 正财 |
| 克我 | 偏官 | 正官 |

## 十神详解

### 一、比肩

**定义**：与日主同阴阳同五行

**代表事物**：

| 象征 | 说明 |
|------|------|
| 自我 | 自己、本我 |
| 兄弟 | 兄弟姐妹、同行 |
| 竞争 | 竞争对手、合伙人 |
| 体力 | 体力、意志力 |

**性格特点**：

| 正面 | 负面 |
|------|------|
| 自信、独立 | 固执、自负 |
| 意志坚定 | 独断专行 |
| 乐于助人 | 好管闲事 |

<Tip title="比肩特点">
比肩代表自我和同类。比肩旺的人性格独立、有主见，但也容易固执己见。比肩在命局中的作用取决于日主强弱。
</Tip>

### 二、劫财

**定义**：与日主异阴阳同五行

**代表事物**：

| 象征 | 说明 |
|------|------|
| 兄弟 | 兄弟姐妹、朋友 |
| 夺取 | 夺取、损失 |
| 风险 | 冒险、投机 |

**性格特点**：

| 正面 | 负面 |
|------|------|
| 豪爽大方 | 冲动消费 |
| 重情重义 | 好赌投机 |
| 有魄力 | 缺乏规划 |

<Warning title="劫财注意事项">
劫财有夺财之意，命局劫财旺而无制，容易破财、被骗。但劫财也可为日主分担压力，具体情况具体分析。
</Warning>

### 三、食神

**定义**：日主所生，阴阳相同

**代表事物**：

| 象征 | 说明 |
|------|------|
| 才华 | 艺术、才能 |
| 子女 | 晚辈、下属 |
| 口福 | 饮食、享受 |

**性格特点**：

| 正面 | 负面 |
|------|------|
| 温和善良 | 懒散安逸 |
| 有艺术天赋 | 缺乏进取 |
| 知足常乐 | 依赖性强 |

### 四、伤官

**定义**：日主所生，阴阳不同

**代表事物**：

| 象征 | 说明 |
|------|------|
| 才华 | 聪明才智 |
| 口才 | 表达能力 |
| 叛逆 | 反抗精神 |

**性格特点**：

| 正面 | 负面 |
|------|------|
| 聪明伶俐 | 傲慢自负 |
| 才华横溢 | 恃才傲物 |
| 有创造力 | 好高骛远 |

<Warning title="伤官见官">
伤官克正官，"伤官见官，为祸百端"。伤官旺的人要注意言行，避免招惹是非。但伤官生财也是好格局。
</Warning>

<Warning title="七杀要点">
七杀攻身，需要制化。有制化为权，无制化为灾。七杀喜食神制、印星化，忌无制无化。
</Warning>

> "枭神夺食，灾祸难免。"——古语

## 十神的组合与判断

### 十神的组合

| 组合 | 含义 |
|------|------|
| 官印相生 | 学业有成、仕途顺遂 |
| 财官双美 | 富贵双全 |
| 食神生财 | 靠才华求财 |
| 伤官见官 | 是非口舌 |
| 枭神夺食 | 灾祸不顺 |

## 学习要点总结

1. **十神由日主与其他干支的五行关系产生**，是连接八字与现实的桥梁
2. **每个十神有特定含义**，代表不同的人事关系
3. **十神有正负两面性**，旺衰不同表现不同
4. **十神的组合形成格局**，需要综合分析
5. **十神分析是八字预测的核心**，需要大量实践
