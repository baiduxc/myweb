## 一、紫微星系详解

### 1. 紫微星（帝星）

**基本特质**：
- 五行：阴土
- 化气：尊贵
- 性质：帝星、领导星

| 方面 | 表现 |
|------|------|
| 正面特质 | 尊贵、领导、统筹、威严 |
| 负面特质 | 傲慢、主观、好面子 |
| 庙旺位置 | 午、巳宫 |
| 落陷位置 | 亥、子宫 |

**紫微星的组合**：

| 组合 | 影响 |
|------|------|
| 紫微+天府 | 紫府同宫，帝王格局 |
| 紫微+天相 | 紫相同宫，辅弼之相 |
| 紫微+贪狼 | 桃花偏重，领袖魅力 |
| 紫微+破军 | 开创力强，起伏较大 |

### 2. 天机星（智慧星）

**基本特质**：
- 五行：阴木
- 化气：善
- 性质：智星、谋略星

| 方面 | 表现 |
|------|------|
| 正面特质 | 聪明、谋略、灵活、善良 |
| 负面特质 | 多虑、犹豫、善变 |
| 庙旺位置 | 丑、卯、未宫 |
| 落陷位置 | 戌、辰宫 |

**天机星的人生建议**：
> "天机善变，最宜策划参谋，不宜独当一面。"

## 二、太阳星系详解

### 3. 太阳星（光明星）

**基本特质**：
- 五行：阳火
- 化气：贵
- 性质：官禄主、父星

| 方面 | 表现 |
|------|------|
| 正面特质 | 光明、正直、外向、无私 |
| 负面特质 | 急躁、冲动、好高骛远 |
| 庙旺位置 | 卯、辰、巳、午宫 |
| 落陷位置 | 戌、亥、子宫 |

**太阳的亮度变化**：

| 宫位 | 状态 | 影响 |
|------|------|------|
| 卯宫 | 日出 | 光芒初现，前程似锦 |
| 辰巳午宫 | 日丽中天 | 最旺，贵显 |
| 酉戌宫 | 日落 | 光芒渐弱，宜低调 |
| 亥子宫 | 入夜 | 落陷，需努力发光 |

### 4. 太阴星（财星）

**基本特质**：
- 五行：阴水
- 化气：富
- 性质：财星、母星

| 方面 | 表现 |
|------|------|
| 正面特质 | 内敛、细腻、温柔、积财 |
| 负面特质 | 消极、敏感、多愁 |
| 庙旺位置 | 酉、戌、亥、子宫 |
| 落陷位置 | 卯、辰、巳、午宫 |

> 太阴与太阳相反，太阳主贵不主富，太阴主富不主贵。
