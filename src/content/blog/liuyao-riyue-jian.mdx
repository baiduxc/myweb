# 六爻日月建的作用：时空条件对卦象的影响

在六爻预测中，日建和月建是判断爻之旺衰的首要标准，被称为"日月为君主，爻为臣民"。日月建对爻的影响深远，直接决定了事情的成败吉凶。

## 一、日建与月建的基本概念

### 1. 日建（日辰）

**日建**是指起卦当日天干地支中的地支。

| 内容 | 说明 |
|------|------|
| 来源 | 起卦当日的日支 |
| 作用 | 对当日之事有主宰权 |
| 时效 | 主当日的影响，性质为短期 |
| 特性 | 可生可克，可冲可合 |

### 2. 月建（月令）

**月建**是指起卦当月的地支。

| 内容 | 说明 |
|------|------|
| 来源 | 起卦当月的月支 |
| 作用 | 对当月之事有主宰权 |
| 时效 | 主当月的影响，性质为中期 |
| 特性 | 决定爻的基本旺衰状态 |

### 3. 日月建的地位

> 《增删卜易》云："月建乃万卦之提纲，日辰为六爻之主宰。"

| 因素 | 地位 | 作用时间 |
|------|------|----------|
| 月建 | 提纲 | 管一月之事 |
| 日建 | 主宰 | 管一日之事 |

日月建共同构成了卦象分析的时空背景。

## 二、月建的作用

### 1. 月建定旺衰

月建是判断爻之旺衰的首要标准。

| 爻的状态 | 条件 | 旺衰程度 |
|----------|------|----------|
| 月破 | 爻与月建相冲 | 最衰 |
| 月克 | 月建克爻 | 衰 |
| 月生 | 月建生爻 | 旺 |
| 月比 | 爻与月建比和 | 旺 |

### 2. 月破详解

**月破**是指爻与月建相冲，此时爻处于最衰弱的状态。

**月破的判断规则**：

| 月建 | 月破之爻 | 含义 |
|------|----------|------|
| 寅 | 申 | 申金被破 |
| 卯 | 酉 | 酉金被破 |
| 辰 | 戌 | 戌土被破 |
| 巳 | 亥 | 亥水被破 |
| 午 | 子 | 子水被破 |
| 未 | 丑 | 丑土被破 |
| 申 | 寅 | 寅木被破 |
| 酉 | 卯 | 卯木被破 |
| 戌 | 辰 | 辰土被破 |
| 亥 | 巳 | 巳火被破 |
| 子 | 午 | 午火被破 |
| 丑 | 未 | 未土被破 |

### 3. 月破的影响

月破之爻：

| 影响 | 结果 |
|------|------|
| 动爻月破 | 动而无力，难成其事 |
| 静爻月破 | 极弱，毫无作为 |
| 用神月破 | 大凶，诸事难成 |

> 但月破之爻若得日建生合，或有动爻来生，则可"逢冲则散，逢合则聚"，有救则不为凶。

## 三、日建的作用

### 1. 日建可生克冲合

日建对爻的影响最为直接：

| 作用 | 含义 | 影响 |
|------|------|------|
| 日生 | 日建生爻 | 为吉，爻得力 |
| 日克 | 日建克爻 | 为凶，爻受损 |
| 日冲 | 日建冲爻 | 为暗动或日破 |
| 日合 | 日建与爻相合 | 为合住或合起 |

### 2. 暗动与日破

日建冲爻时，需要区分暗动和日破：

| 情况 | 条件 | 结果 |
|------|------|------|
| 暗动 | 爻旺而被日冲 | 越冲越动，反而为吉 |
| 日破 | 爻衰而被日冲 | 冲而即破，为凶 |

**判断标准**：
- 爻得月建生扶，而有日冲 → 暗动
- 爻被月建克破，而又有日冲 → 日破

### 3. 日建的特殊作用

日建有月建所不具备的特殊作用：

| 特殊作用 | 说明 |
|----------|------|
| 主当日 | 日建对当日之事有直接决定权 |
| 可解月破 | 日建生合月破之爻，可以解破 |
| 可破合 | 日建冲合住之爻，可以破合 |
| 临卦可成 | 若日建临卦中之爻，此爻当日有力 |

## 四、日月建的共同作用

### 1. 日月生扶

**日月生扶**是最理想的状态：

| 情况 | 含义 | 结果 |
|------|------|------|
| 日月同生 | 日建月建同时生爻 | 最旺 |
| 日月一生一克 | 一方生、一方克 | 旺衰相抵，需详查 |
| 日月同克 | 日建月建同时克爻 | 最衰 |

### 2. 日月冲克的矛盾

当月建与日建对爻的作用相反时：

| 矛盾情况 | 分析 |
|----------|------|
| 月破日生 | 虽破得救，仍有转机 |
| 月生日破 | 得月失日，先好后差 |
| 月克日生 | 有难有救，先困后成 |
| 月生日克 | 有福有难，先顺后逆 |

### 3. 日月空亡的判断

日月建不入空亡，但要注意：

| 情况 | 判断 |
|------|------|
| 日建为空亡 | 日建不入空，仍有力 |
| 月建为空亡 | 月建不入空，仍有力 |
| 爻值空亡被日月冲 | 可以出空 |

## 五、日月建对动爻的作用

### 1. 动爻逢日月

动爻是变化的关键，受日月影响：

| 情况 | 影响 |
|------|------|
| 动爻得月建 | 动而有力，变化显著 |
| 动爻值月破 | 动而无力，变化无效 |
| 动爻得日建 | 当日触发，速成速败 |
| 动爻值日破 | 当日难成，需待他时 |

### 2. 动变回头生克

动爻变出之爻，也受日月影响：

| 变化 | 日月影响 |
|------|----------|
| 动变回头生 | 变爻得日月则生力强 |
| 动变回头克 | 变爻得日月则克力强 |
| 变爻被日月破 | 生克无力 |

### 3. 进神退神的判断

日月建影响进神退神：

| 进退神 | 日月影响 |
|--------|----------|
| 进神逢日月生扶 | 进而有成 |
| 进神逢日月克破 | 进而不利 |
| 退神逢日月生扶 | 退而有力 |
| 退神逢日月克破 | 退而受损 |

| 子孙月破 | 医药无效 | 求医不利 |

## 学习要点总结

1. **月建为提纲**，决定爻的基本旺衰状态，管一月之事

2. **日建为主宰**，可直接生克冲合爻，管一日之事

3. **月破**是爻与月建相冲，为最衰状态，但可解

4. **暗动与日破**：爻旺被日冲为暗动，爻衰被日冲为日破

5. 日月共同作用时，需分析**生克关系**和**力量对比**

6. 动爻受日月影响更大，动变回头生克也需考虑日月

日月建是六爻预测的核心参考，分析任何卦象都必须首先考虑日月建对各爻的影响。日月为君主，爻为臣民，君令臣从，方能吉凶分明。
