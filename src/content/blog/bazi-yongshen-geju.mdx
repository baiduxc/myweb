# 八字用神格局进阶：从入门到精通的格局判断

用神是八字命理的灵魂，格局是用神的载体。格局不明，用神无依；用神不准，断命必谬。本文将深入讲解八字用神格局的进阶判断方法。

## 一、用神的核心概念

### 1.1 什么是用神

用神，即八字中对日主起平衡、调节作用的神（五行）。

> **用神者，命之根本，运之枢纽**

用神的作用：
- 扶抑日主：弱者扶之，强者抑之
- 调候气候：寒者暖之，燥者润之
- 通关化合：调和矛盾

### 1.2 用神的分类

| 分类 | 说明 | 适用情况 |
|------|------|----------|
| 扶抑用神 | 日主偏弱则扶，偏强则抑 | 常见格局 |
| 调候用神 | 寒冬需暖，盛夏需凉 | 调候为先 |
| 通关用神 | 两种相克五行需调和 | 相战格局 |
| 病药用神 | 有病方为贵，无伤不是奇 | 特殊格局 |

### 1.3 用神的选取原则

**第一步：判断日主强弱**

| 日主状态 | 特征 | 取用方向 |
|----------|------|----------|
| 身强 | 得令+得地+得势 | 抑之（克泄耗） |
| 身弱 | 失令+失地+失势 | 扶之（生扶） |
| 中和 | 力量均衡 | 随运而变 |

**第二步：看格局**

| 格局类型 | 取用方法 |
|----------|----------|
| 正格 | 扶抑为主 |
| 从格 | 顺其气势 |
| 化气格 | 从化神论 |

**第三步：看调候**

| 月令 | 调候需求 |
|------|----------|
| 子月（冬） | 需火暖局 |
| 丑月（冬） | 需火暖局 |
| 寅月（初春） | 需火暖局 |
| 午月（夏） | 需水调候 |
| 未月（夏） | 需水调候 |
| 申月（秋） | 需火温水 |

## 二、正格详解

### 2.1 正格的定义

正格，即日主有根有气，或强或弱，但不至于从化，需用扶抑之法取用的格局。

### 2.2 正格的分类

**身强格**：

| 情况 | 特征 | 用神 |
|------|------|------|
| 比劫重重 | 日主极强 | 食伤泄秀或官杀制身 |
| 印旺身强 | 印多为病 | 财星坏印 |
| 官杀弱 | 克制无力 | 食伤生财 |

**身弱格**：

| 情况 | 特征 | 用神 |
|------|------|------|
| 财多身弱 | 财耗身力 | 比劫帮身 |
| 官杀重身弱 | 克身太过 | 印星化杀 |
| 食伤多身弱 | 泄身太过 | 印星制食 |

### 2.3 身强格实例

**例：庚金日主，生于申月，比劫重重**

```
年柱：辛酉（比劫）
月柱：丙申（比劫）
日柱：庚申（比劫）
时柱：戊寅（偏印）
```

**分析**：
- 庚金生于申月，当令得时
- 地支三申一酉，比劫林立
- 日主极强，需抑

**取用**：
- 用神：水（食伤泄秀）
- 喜神：木（财星）
- 忌神：金（比劫）、土（印星）

### 2.4 身弱格实例

**例：甲木日主，生于申月，官杀重重**

```
年柱：庚申（七杀）
月柱：甲申（七杀）
日柱：甲子（印星）
时柱：丙寅（食神）
```

**分析**：
- 甲木生于申月，失令
- 庚申年、甲申月，官杀重重
- 甲木坐子水，有印相生
- 日主偏弱，需扶

**取用**：
- 用神：水（印星化杀）
- 喜神：木（比劫帮身）
- 忌神：金（官杀）、土（财星坏印）

## 三、变格详解

### 3.1 从格

**定义**：日主无根无助，不得已从其旺势。

**从格分类**：

| 从格类型 | 条件 | 用神 |
|----------|------|------|
| 从财格 | 财星当令，日主无根 | 财星、食伤 |
| 从杀格 | 官杀当令，日主无根 | 官杀、财星 |
| 从儿格 | 食伤当令，日主无根 | 食伤、财星 |
| 从势格 | 众势专一，日主从之 | 旺者为用 |

**从财格实例**：

```
年柱：己巳（财星）
月柱：丙戌（财星）
日柱：乙酉（日主）
时柱：己卯（财星）
```

**分析**：
- 乙木生于戌月，财星当令
- 地支巳酉戌金局，天干丙己财星
- 乙木无根（卯被酉冲），无力抗争
- 从财格成

**取用**：
- 用神：土（财星）、火（食伤生财）
- 忌神：木（比劫）、水（印星）

### 3.2 化气格

**定义**：日干与月干或时干合化成功，化为他神。

**化气格分类**：

| 化格类型 | 合化条件 | 用神 |
|----------|----------|------|
| 化木格 | 甲己合化木 | 木、水 |
| 化火格 | 乙庚合化火 | 火、木 |
| 化土格 | 丙辛合化土 | 土、火 |
| 化金格 | 丁壬合化金 | 金、土 |
| 化水格 | 戊癸合化水 | 水、金 |

**化土格实例**：

```
年柱：癸巳
月柱：丙辰
日柱：丙辰
时柱：辛卯
```

**分析**：
- 丙辛相合，生于辰月
- 辰为湿土，利于化土
- 化土格成

**取用**：
- 用神：土（化神）
- 喜神：火（生助化神）
- 忌神：木（克化神）

### 3.3 建禄格与羊刃格

**建禄格**：

| 条件 | 说明 |
|------|------|
| 日主坐禄 | 甲日见寅，乙日见卯 |
| 月令为禄 | 甲生寅月，乙生卯月 |
| 特点 | 身旺有力，可任财官 |

**取用**：
- 身旺用财官
- 喜食伤泄秀
- 忌比劫再助

**羊刃格**：

| 条件 | 说明 |
|------|------|
| 日主坐刃 | 甲日见卯，丙日见午 |
| 月令为刃 | 甲生卯月，丙生午月 |
| 特点 | 阳刚过盛，需制化 |

**取用**：
- 喜官杀制刃
- 喜食伤泄刃
- 忌财生杀太过

## 四、格局成败

### 4.1 格局成功的条件

| 条件 | 说明 |
|------|------|
| 用神有力 | 用神得令得地得势 |
| 用神不伤 | 用神不受冲克 |
| 格局纯粹 | 没有破格之物 |
| 行运配合 | 大运帮扶用神 |

### 4.2 格局失败的原因

| 原因 | 说明 |
|------|------|
| 用神无力 | 用神失令休囚 |
| 用神受伤 | 用神被冲克刑害 |
| 格局杂乱 | 喜忌混杂不清 |
| 行运悖逆 | 大运克泄用神 |

### 4.3 格局高低判断

| 等级 | 条件 | 人生表现 |
|------|------|----------|
| 上格 | 用神有力+格局纯粹+运助 | 富贵双全 |
| 中上格 | 用神有力+格局稍杂 | 有成有败 |
| 中格 | 用神有力+行运不佳 | 有才难展 |
| 中下格 | 用神无力+格局杂乱 | 一生平庸 |
| 下格 | 无用神+格局破败 | 多灾多难 |

## 学习要点

1. **用神是八字的核心**，取准用神等于成功一半
2. **正格用扶抑法**，**变格用顺势法**
3. **调候为先**，寒暖燥湿需首先考虑
4. **格局成败**决定了人生的层次高低
5. 用神需**结合行运**看吉凶
6. 身强身弱是**相对概念**，需综合判断
7. 学习格局需**多看实例**，从实践中总结经验
8. 用神格局的最终目的是**指导人生、趋吉避凶**
