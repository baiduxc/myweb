空亡又称旬空、落空，是六爻预测中判断爻位力量强弱与应期早晚的重要技法。空亡之爻有其特殊的表现规律，掌握空亡的断法，可以显著提升断卦的精准度。

## 空亡的基本概念

### 何为空亡

空亡是根据六十甲子的排列规律，每个旬中有两支轮空：

> 六十甲子分为六旬，每旬十日，各有两支旬空。

**六旬空亡表：**

| 旬首 | 干支组合 | 空亡地支 |
|------|----------|----------|
| 甲子旬 | 甲子至癸酉 | 戌、亥 |
| 甲戌旬 | 甲戌至癸未 | 申、酉 |
| 甲申旬 | 甲申至癸巳 | 午、未 |
| 甲午旬 | 甲午至癸卯 | 辰、巳 |
| 甲辰旬 | 甲辰至癸丑 | 寅、卯 |
| 甲寅旬 | 甲寅至癸亥 | 子、丑 |

### 空亡的查法

以日干支为准，查该旬所空之支：

**示例：**
- 日柱为甲子，属甲子旬，则戌亥空亡
- 卦中有戌爻或亥爻，即为空亡
- 日柱为丙寅，属甲子旬，戌亥仍空亡

## 空亡爻的性质

### 空亡爻的基本特征

| 特征 | 说明 |
|------|------|
| 力量减弱 | 空爻对其他爻的作用力降低 |
| 应期延迟 | 事情落在空爻上，往往应期延后 |
| 有待填补 | 空亡需要出空、填实才有力 |
| 因事而异 | 吉事空亡为不吉利，凶事空亡反为吉 |

### 空亡与六亲的关系

| 空爻六亲 | 空亡含义 |
|----------|----------|
| 父母空 | 文书不就、长辈有灾 |
| 兄弟空 | 合作不成、朋友无力 |
| 妻财空 | 财源不聚、妻室不利 |
| 子孙空 | 子女缘分薄、求财无源 |
| 官鬼空 | �职不牢、功名不就 |

> 空亡须结合卦中具体情况判断，不可机械套用。

## 空亡的填补

### 出空与填实

空亡爻可通过以下方式填实：

| 方式 | 条件 | 效果 |
|------|------|------|
| 旬空出空 | 时间进入下一旬 | 自然填实 |
| 日建填实 | 日支与空爻相同 | 当日出空 |
| 月建填实 | 月支与空爻相同 | 本月有效 |
| 动爻填实 | 动爻变出空爻 | 化空填实 |
| 冲空填实 | 被他爻冲起 | 冲则不空 |

### 实例分析

**卦例：问财运**

卦得：妻财爻为用神，落空亡

**分析：**
1. 用神空亡——财运暂不落实
2. 若有日建临财——当日可得
3. 若有月建临财——月内有财
4. 待出空之日——财运兑现

## 真空与假空

### 真空

真正的空亡，无填实可能：

| 情况 | 说明 |
|------|------|
| 爻无根无生 | 空爻无助，难以填实 |
| 死绝之地空 | 爻位死绝，空则更弱 |
| 月破又空 | 破空并见，力量极弱 |

### 假空

表面的空亡，实际可填实：

| 情况 | 说明 |
|------|------|
| 有日月生扶 | 虽空得力，逢填即应 |
| 动爻化空 | 动出为真，不以此论 |
| 空下坐库 | 有根不真空 |

**判断口诀：**

> "旺不空，动不空，有日月建帮助不空，动而化空不空，空而生合不空。"

## 空亡在不同预测中的应用

### 一、财运预测

| 财爻状态 | 判断 |
|----------|------|
| 财旺不空 | 财运亨通 |
| 财衰而空 | 无财可求 |
| 财空有生 | 待时而得 |
| 财空遇冲 | 反可得财 |

### 二、感情预测

| 妻财/官鬼状态 | 判断 |
|---------------|------|
| 用神空亡 | 缘分未到、感情不稳 |
| 用神空有填 | 等待时机、日后可成 |
| 世应俱空 | 双方无心、难有结果 |

### 三、健康预测

| 病灶爻状态 | 判断 |
|------------|------|
| 鬼爻空亡 | 病情不稳定或暂缓 |
| 鬼旺不空 | 病重难治 |
| 鬼空有填 | 需防复发 |

## 空亡的应期判断

### 出空应期推断

| 应期类型 | 断法 |
|----------|------|
| 日时出空 | 日支填实时应验 |
| 月令出空 | 月支填实时应验 |
| 流年出空 | 流年地支填实时应验 |
| 变爻出空 | 动变之爻填实时应验 |

> 空亡的应期往往延迟于正常情况，须有填实条件方可应验。

## 学习要点总结

1. **旬空查法**：以日干支查旬空，戌亥、申酉、午未、辰巳、寅卯、子丑六组
2. **空爻特性**：力量减弱、应期延迟、需待填实
3. **真假之分**：旺不空、动不空、有扶不空
4. **填实方式**：日填、月填、动变填、冲填
5. **吉凶两面**：吉事空则不吉，凶事空则反吉
6. **应期推后**：空亡之爻本身应期延迟，需等填实方应
