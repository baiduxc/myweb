## 一、阴阳概述

阴阳学说是中国古代哲学的核心，用以解释宇宙间一切事物的变化规律。

### 1. 阴阳的定义

> "一阴一阳之谓道。"——《周易·系辞》

阴阳是宇宙间两种对立统一的基本力量：

| 属性 | 阳 | 阴 |
|------|-----|-----|
| 性质 | 刚健、主动、明亮 | 柔顺、被动、晦暗 |
| 方位 | 上、外、左、南 | 下、内、右、北 |
| 时间 | 昼、春夏、生 | 夜、秋冬、杀 |
| 形态 | 热、燥、明、升 | 寒、湿、暗、降 |
| 人物 | 男、君、父 | 女、臣、母 |

### 2. 阴阳的特性

| 特性 | 解释 |
|------|------|
| 普遍性 | 万物皆有阴阳 |
| 相对性 | 阴阳相互依存，相互比较 |
| 可分性 | 阴中有阳，阳中有阴 |
| 转化性 | 阴可变阳，阳可变阴 |

## 二、阴阳的基本关系

### 1. 对立制约

> 阴阳双方相互制约，维持动态平衡。

| 对立表现 | 说明 |
|----------|------|
| 水火对立 | 寒热、湿燥 |
| 动静对立 | 动为阳，静为阴 |
| 升降对立 | 升为阳，降为阴 |

### 2. 互根互用

> "孤阴不生，独阳不长。"

阴阳相互依存，缺一不可：

| 关系 | 说明 |
|------|------|
| 阴为阳根 | 阳以阴为根基 |
| 阳为阴用 | 阴需要阳的推动 |
| 无阴则阳无根 | 阳会耗散 |
| 无阳则阴不化 | 阴会停滞 |

### 3. 消长平衡

> 阴阳不断消长，维持相对平衡。

| 消长变化 | 阶段 |
|----------|------|
| 阳长阴消 | 春夏、上午 |
| 阴长阳消 | 秋冬、下午 |
| 平衡状态 | 健康状态 |
| 失衡状态 | 疾病状态 |

### 4. 相互转化

> "阳极生阴，阴极生阳。"

| 转化条件 | 转化结果 |
|----------|----------|
| 阳极 | 生阴 |
| 阴极 | 生阳 |
| 突变 | 质变 |

## 三、五行与阴阳的融合

### 1. 五行的阴阳划分

| 五行 | 阳 | 阴 |
|------|-----|-----|
| 木 | 甲木（大林） | 乙木（花草） |
| 火 | 丙火（太阳） | 丁火（灯烛） |
| 土 | 戊土（城墙） | 己土（田园） |
| 金 | 庚金（斧钺） | 辛金（珠玉） |
| 水 | 壬水（江河） | 癸水（雨露） |

### 2. 阳五行与阴五行的特点

| 类别 | 特点 | 象征 |
|------|------|------|
| 阳木 | 刚直、向上、突破 | 大树、栋梁 |
| 阴木 | 曲柔、依附、延伸 | 藤萝、花草 |
| 阳火 | 明亮、外向、大公 | 太阳、烈火 |
| 阴火 | 幽暗、内向、深思熟虑 | 灯火、蜡烛 |
| 阳土 | 厚重、承载、防范 | 城墙、堤坝 |
| 阴土 | 滋养、包容、随和 | 田园、细土 |
| 阳金 | 刚毅、果断、肃杀 | 刀剑、斧钺 |
| 阴金 | 细腻、精巧、敏感 | 珠宝、首饰 |
| 阳水 | 奔腾、广阔、自由 | 江河、大海 |
| 阴水 | 涓秀、幽静、细腻 | 露珠、清泉 |
