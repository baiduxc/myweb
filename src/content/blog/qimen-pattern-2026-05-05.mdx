奇门遁甲格局是盘局分析的核心内容，格局的好坏直接决定事情成败。掌握格局的判断要领，是学习奇门遁甲的关键一步。

## 一、格局的基本概念

### 1. 什么是格局

格局是天盘与地盘之间的特定组合关系：

| 要素 | 说明 |
|-----|-----|
| 天盘 | 加在九宫之上的天干 |
| 地盘 | 九宫原始方位的天干 |
| 门 | 所临的八门 |
| 星 | 所临的九星 |
| 神 | 所临的八神 |

### 2. 格局的重要性

格局是判断事情成败的第一要素，好格局往往能逢凶化吉，坏格局则可能好景不长。

## 二、主要吉格详解

### 1. 青龙回首（吉）

**构成条件**：
- 天盘甲子加地盘戊
- 或天盘戊加地盘甲子

**格局象义**：

| 方面 | 象义 |
|-----|-----|
| 事业 | 有贵人相助，事业顺利 |
| 财运 | 财源广进，求财顺利 |
| 婚姻 | 姻缘美满，夫妻和睦 |

### 2. 飞鸟跌穴（大吉）

**构成条件**：
- 天盘丙加地盘戊

**格局象义**：

| 方面 | 象义 |
|-----|-----|
| 事业 | 一飞冲天，事业有成 |
| 考试 | 金榜题名，学业顺遂 |
| 投资 | 投资获利，一举成功 |

### 3. 天遁（吉）

**构成条件**：天盘乙加地盘丙

**格局象义**：谋事顺利，有贵人暗助

### 4. 地遁（吉）

**构成条件**：天盘乙加地盘丁

**格局象义**：宜静不宜动，利于养精蓄锐

### 5. 人遁（吉）

**构成条件**：天盘乙加地盘己

**格局象义**：人际关系和睦，利于合作

### 6. 神遁（吉）

**构成条件**：天盘丙加地盘丁

**格局象义**：有神灵护佑之象，遇险能化吉

## 三、主要凶格详解

### 1. 青龙折足（凶）

**构成条件**：天盘庚加地盘辛

**格局象义**：

| 方面 | 象义 |
|-----|-----|
| 事业 | 事业受阻，步履维艰 |
| 出行 | 交通不利，需防意外 |
| 健康 | 腿脚受伤，行动不便 |

### 2. 白虎猖狂（大凶）

**构成条件**：天盘庚加地盘乙

**格局象义**：

| 方面 | 象义 |
|-----|-----|
| 官司 | 官非口舌，后果严重 |
| 健康 | 血光之灾，手术之象 |
| 婚姻 | 婚姻破裂，夫妻相克 |

### 3. 太白入荧（凶）

**构成条件**：天盘庚加地盘丙

**格局象义**：官非破财，盗贼入侵

### 4. 荧入太白（凶）

**构成条件**：天盘丙加地盘庚

**格局象义**：口舌官非，争斗冲突

## 五、格局判断的要点

### 1. 格局与门星的关系

| 情况 | 判断 |
|-----|-----|
| 吉格临吉门 | 大吉 |
| 吉格临凶门 | 吉力减弱 |
| 凶格临吉门 | 凶力减弱 |
| 凶格临凶门 | 大凶 |

### 2. 格局与落宫的关系

| 情况 | 判断 |
|-----|-----|
| 吉格落旺宫 | 吉上加吉 |
| 吉格落衰宫 | 吉力减半 |

| 凶格落衰宫 | 凶上加凶 |

## 学习要点

1. 记住构成：熟记各格局的天干组合
2. 理解象义：明白格局代表的意义
3. 综合判断：结合门星神宫位分析
4. 旺衰考量：格局配合落宫旺衰
5. 灵活应用：不拘泥于书本，灵活判断
6. 实践验证：大量案例验证格局应验

## 结语

奇门遁甲格局是盘局分析的核心技能。好的格局往往预示着事情顺利，坏的格局则提醒我们谨慎行事。但格局不是绝对的，还需综合判断。

格局虽定，人事可变——命理预测的意义在于趋吉避凶，而非宿命论。
