## 八卦概述

八卦是《周易》的基础符号系统，由阳爻"—"和阴爻"--"三叠而成，代表宇宙间八种基本现象和力量。

## 乾卦：天行健

### 自然象征

乾卦象征天空、天体、纯阳之气：

> "大哉乾元，万物资始，乃统天。"——《周易·乾卦·彖辞》

**乾卦特性：**

| 特性 | 表现 |
|------|------|
| 刚健 | 强劲有力，不屈不挠 |
| 主导 | 领导力量，统领万物 |
| 进取 | 向上发展，积极进取 |
| 创造 | 生生不息，万物之源 |

### 人事象征

| 象征对象 | 含义说明 |
|----------|----------|
| 父亲 | 家庭中的权威人物 |
| 君王 | 国家统治者 |
| 男人 | 成年男性 |
| 领导 | 单位负责人 |

### 身体部位

乾卦对应人体的头部、骨骼、肺部。

## 学习要点总结

1. **八卦本质**：八种基本力量和现象的符号化表达

2. **双重属性**：每卦都有阴阳属性，分阳卦和阴卦

3. **多层象征**：自然、人事、身体、方位等多重含义

4. **动态变化**：八卦不是静态的，而是动态流转的

5. **实践应用**：理解八卦象征是学习易学的基础
