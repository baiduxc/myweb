## 六合的基本概念

六合是六爻预测中极为重要的格局之一，它揭示了事物之间的和谐、合作与相互吸引的力量。在六爻卦象中，六合分为天干六合、地支六合两种，其中以地支六合在断卦中应用最为广泛。

### 地支六合组合

六爻中的地支六合共有六组：

| 六合组合 | 合化结果 | 基本象义 |
|----------|----------|----------|
| 子丑合 | 化土 | 泥土相合，包容厚重 |
| 寅亥合 | 化木 | 生发之气，生机勃勃 |
| 卯戌合 | 化火 | 燃烧之象，光明热烈 |
| 辰酉合 | 化金 | 藏金之土，实质收获 |
| 巳申合 | 化水 | 流动变化，智慧通达 |
| 午未合 | 化土 | 烈日熔土，坚韧稳固 |

> 子丑合寅亥合，卯戌辰酉巳申合。
> 午未相合化成土，六合入卦主和合。

### 六合的核心意义

在断卦实践中，六合的基本含义包括：

1. **和谐与协作**：双方或多方达成共识，合作顺利
2. **牵绊与联结**：事情可能有所拖延，但不会断绝
3. **助力与支持**：获得他人帮助，增强自身力量
4. **成局与转化**：原本分散的力量形成整体

## 六合的成局条件

### 静爻六合

静卦中的六合判断相对直接：

- 卦中有两个动爻形成六合，为动合
- 卦中静爻与日辰或月建相合，为时合
- 卦中有三合局（三支六合），为合局

**静爻六合的要点**：

| 条件 | 吉凶判断 |
|------|----------|
| 用神得合 | 吉，主得人助、事成 |
| 忌神得合 | 凶，主仇人得势、阻碍大 |
| 动爻合静爻 | 动者主动寻求结合 |
| 静爻自合 | 内在潜力，需待时而动 |

### 动爻六合

动爻形成六合时，其力量和应验更为明显：

**动爻六合的几种情况**：

1. **本卦动合**：原卦中两动爻相合
2. **变卦得合**：动爻变出之爻与原爻相合
3. **日月建合**：动爻与日辰或月建六合

> 动而逢合多应验，合处逢冲事难成。
> 用神得合最为吉，忌神逢合祸不轻。

### 三合局

六爻中的三合局是最强的合局形式：

| 三合组合 | 合局结果 | 基本性质 |
|----------|----------|----------|
| 申子辰 | 水局 | 智慧、流动、北方 |
| 寅午戌 | 火局 | 热情、光明、南方 |
| 巳酉丑 | 金局 | 决断、收获、西方 |
| 亥卯未 | 木局 | 仁慈、生发、东方 |

**三合局的成局条件**：

1. 卦中三支全现，且有一支为动爻
2. 两支在卦中，一支为日辰或月建
3. 三支虽静，但日月建参与构成

## 六合吉凶判断

### 用神逢六合

用神是断卦的核心，用神逢合的吉凶判断：

| 情况 | 吉凶 | 具体表现 |
|------|------|----------|
| 用神旺相逢合 | 大吉 | 事必成，得贵人助力 |
| 用神衰弱逢合 | 先凶后吉 | 初虽有阻，最终得助 |
| 用神被忌神合 | 凶 | 受制于人，难展宏图 |
| 用神自刑又逢合 | 吉凶参半 | 先苦后甜，曲折前进 |

### 六合在不同事项中的应用

**占婚姻**：
- 用神逢合，婚姻可成，感情和谐
- 世应相合，双方情投意合
- 忌神合用神，可能有第三者

**占财运**：
- 财爻逢合，财源稳定，合作生财
- 财爻合世，财来就我，容易获利
- 财爻合应，财归他人，求财需借力

**占事业**：
- 官爻逢合，得领导赏识，职场顺利
- 子孙合官，可能离职或创业
- 兄弟合官，有同事助力也需防范竞争

**占诉讼**：
- 六合入卦，多主和解或调解
- 用神逢合，有利于己方
- 忌神逢合，对方有贵人相助

## 六合的应验时间

六合入卦的应验时间有其特殊规律：

### 应验时限

| 合局类型 | 一般应验时间 |
|----------|--------------|
| 月建合 | 当月应验 |
| 日辰合 | 当日或近期应验 |
| 卦内合 | 根据动爻旺衰判断 |
| 三合局 | 需待合局完整的时辰或月份 |

### 合处逢冲

六合最怕冲破，合处逢冲的判断：

- 月破冲合，合局无力
- 日冲合爻，合而复散
- 动爻冲合，先合后散

> 合处逢冲多反复，合而不化力难全。
- 若逢月破日辰冲，好事多磨需耐心。

## 实战案例分析

### 案例一：占婚姻成否

某男占问与女友能否成婚，得卦如下：

- 世持妻财寅木
- 应持官鬼酉金
- 寅亥六合在卦中

**分析**：
- 世爻持财，说明求测者对女方感情深厚
- 寅亥相合，双方感情基础良好
- 财爻旺相，婚姻可成

**结果**：当年亥月双方成婚。

### 案例二：占生意合作

某商占问与合作伙伴的生意前景：

- 卦中申子辰三支全现
- 财爻子水为用神
- 日辰为辰土

**分析**：
- 申子辰三合水局，合作力量强
- 财爻为合局中心，主财源汇聚
- 合作顺利，前景看好

**结果**：当年申月正式签约，合作成功。

## 六合与六冲的关系

### 基本对比

| 特征 | 六合 | 六冲 |
|------|------|------|
| 性质 | 和谐、凝聚 | 冲突、散开 |
| 速度 | 缓慢、渐进 | 迅速、突变 |
| 稳定性 | 稳定、持久 | 不稳、变化 |
| 适用 | 婚姻、合作、和解 | 分手、诉讼、破财 |

### 合冲并见

当卦中既有六合又有六冲时：

1. **先合后冲**：初期顺利，后期有变
2. **先冲后合**：初期波折，后期和谐
3. **合冲并行**：事有反复，需综合判断

## 断卦要诀

运用六合断卦的核心要领：

1. 辨明用神忌神，合局对各方的利弊
2. 判断合局强弱，旺相逢合力大
3. 注意合化方向，化出五行对卦象的影响
4. 观察冲破情况，合处逢冲需谨慎判断
5. 结合卦名卦意，综合分析六合同其他因素

## 学习要点总结

1. 六合共有六组，每组都有特定的合化结果和象义
2. 六合主和谐合作，但需区分用神忌神逢合的不同效果
3. 三合局力量最强，需掌握申子辰、寅午戌等四组
4. 六合的应验时间与月建、日辰密切相关
5. 合处逢冲是六合格局的重要变数，需特别关注
6. 六合与六冲性质相反，断卦时需综合权衡
