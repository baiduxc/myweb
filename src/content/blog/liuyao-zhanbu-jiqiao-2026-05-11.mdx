## 六爻占卜概述

六爻占卜是中国古代最为普及的预测方法之一，它以《周易》六十四卦为基础，通过摇卦或时间起卦，结合五行生克、六亲关系、世应位置等要素，对所问之事进行预测判断。六爻占卜操作简便、判断精准，深受民间喜爱。

<Note title="六爻占卜的特点">
六爻占卜以易经八卦为理论基础，以五行生克为判断依据，具有简单易学、判断精准、适用广泛的特点，被称为"小六壬"。
</Note>

## 六爻占卜的基本概念

### 一、六十四卦

六十四卦由八卦两两相重而成，每卦六爻，共三百八十四爻。

| 卦的组成 | 说明 |
|----------|------|
| 上卦 | 上三爻，代表外卦 |
| 下卦 | 下三爻，代表内卦 |
| 阳爻 | 用"—"或"九"表示 |
| 阴爻 | 用"--"或"六"表示 |

### 二、纳甲与地支

六爻占卜需要为每爻配上地支，称为"纳甲"：

| 八卦 | 纳甲地支 |
|------|----------|
| 乾卦 | 子、寅、辰、午、申、戌 |
| 坤卦 | 未、巳、卯、丑、亥、酉 |
| 震卦 | 子、寅、辰、午、申、戌 |
| 巽卦 | 丑、亥、酉、未、巳、卯 |
| 坎卦 | 寅、辰、午、申、戌、子 |
| 离卦 | 卯、丑、亥、酉、未、巳 |
| 艮卦 | 辰、午、申、戌、子、寅 |
| 兑卦 | 巳、卯、丑、亥、酉、未 |

### 三、六亲

六亲是六爻断卦的核心要素：

| 六亲 | 含义 | 代表事物 |
|------|------|----------|
| 父母 | 生我者 | 长辈、文书、房屋 |
| 兄弟 | 比和者 | 同辈、朋友、竞争 |
| 子孙 | 我生者 | 晚辈、医药、解忧 |
| 妻财 | 我克者 | 财物、妻子、欲望 |
| 官鬼 | 克我者 | 官职、丈夫、鬼怪 |

<Tip title="六亲取用原则">
六亲的取用以卦宫五行为基准，按照五行生克关系确定。如乾宫属金，则土为父母，水为子孙，木为妻财，火为官鬼，金为兄弟。
</Tip>

## 起卦方法

### 一、金钱起卦法

金钱起卦是最常用的起卦方法：

**准备工作**：
- 三枚铜钱或硬币
- 安静的环境
- 诚心专注

**起卦步骤**：

| 步骤 | 操作 |
|------|------|
| 1 | 凝神静气，默念所问之事 |
| 2 | 将三枚硬币合于掌心，摇动后掷出 |
| 3 | 记录硬币正反面组合 |
| 4 | 重复六次，从下往上记录 |

**爻的确定**：

| 组合 | 记录 | 爻的性质 |
|------|------|----------|
| 三正 | O | 阳爻动 |
| 三反 | X | 阴爻动 |
| 二正一反 | -- | 阴爻静 |
| 二反一正 | — | 阳爻静 |

### 二、时间起卦法

根据起卦时间来起卦：

**方法**：
- 上卦 = (年支数 + 月数 + 日数) ÷ 8 的余数
- 下卦 = (年支数 + 月数 + 日数 + 时数) ÷ 8 的余数
- 动爻 = (年支数 + 月数 + 日数 + 时数) ÷ 6 的余数

**数字对应八卦**：

| 数字 | 八卦 |
|------|------|
| 1 | 乾 |
| 2 | 兑 |
| 3 | 离 |
| 4 | 震 |
| 5 | 巽 |
| 6 | 坎 |
| 7 | 艮 |
| 8 | 坤 |

<Warning title="起卦注意事项">
起卦时心要诚，意要专。一事一卦，不可反复询问同一事项。起卦环境要安静，避免干扰。
</Warning>

## 断卦方法

### 一、世应分析

世爻代表求测者自己，应爻代表对方或所问之事：

| 关系 | 含义 |
|------|------|
| 世应相生 | 有情，顺利 |
| 世应相克 | 无情，阻碍 |
| 世应比和 | 势均力敌 |
| 世旺应衰 | 我强彼弱 |
| 世衰应旺 | 我弱彼强 |

### 二、用神选取

用神是断卦的关键，不同事项选取不同用神：

| 预测事项 | 用神 |
|----------|------|
| 求财 | 妻财爻 |
| 求官 | 官鬼爻 |
| 考试 | 父母爻 |
| 疾病 | 官鬼爻（忌）或子孙爻（喜） |
| 婚姻 | 男测以妻财为用，女测以官鬼为用 |
| 出行 | 父母爻 |

### 三、原神、忌神、仇神

| 名称 | 定义 | 作用 |
|------|------|------|
| 原神 | 生用神之爻 | 助旺用神 |
| 忌神 | 克用神之爻 | 损伤用神 |
| 仇神 | 克原神之爻 | 间接伤害用神 |

### 四、月破与日破

| 概念 | 定义 | 影响 |
|------|------|------|
| 月破 | 被月令冲克之爻 | 大凶，诸事不宜 |
| 日破 | 被日辰冲克之爻 | 凶，但较月破为轻 |
| 旬空 | 甲子旬中戌亥空等 | 有力无力之分 |

> "月破之爻，诸事皆凶；若得日辰生合，尚有一线生机。"——《增删卜易》

### 五、动爻分析

动爻是卦中变化的关键：

| 动爻特点 | 说明 |
|----------|------|
| 动变 | 动爻会发生变化，阳变阴，阴变阳 |
| 动生克 | 动爻有生克权，能主动发挥作用 |
| 动逢合 | 动爻逢合则绊住，难以发挥作用 |
| 动逢冲 | 动爻逢冲则更旺或被冲散 |

<Tip title="动爻要点">
动爻是卦的灵魂，变化的关键。吉神发动则吉，凶神发动则凶。但需看整体配合，不可孤立判断。
</Tip>

## 实际应用示例

### 求财预测

**用神**：妻财爻

**判断要点**：

| 要点 | 说明 |
|------|------|
| 财爻旺相 | 求财可得 |
| 财爻休囚 | 求财困难 |
| 财爻持世 | 财运亨通 |
| 财爻生世 | 得财容易 |
| 财爻克世 | 求财辛苦 |
| 财爻空破 | 求财落空 |

### 求官预测

**用神**：官鬼爻

**判断要点**：

| 要点 | 说明 |
|------|------|
| 官爻旺相 | 官运亨通 |
| 官爻持世 | 官职在身 |
| 官生世爻 | 贵人相助 |
| 官克世爻 | 压力大或灾祸 |
| 官爻空破 | 求官无望 |

## 学习要点总结

1. **六爻占卜以周易为基础**，结合五行六亲进行判断
2. **起卦要诚心专注**，一事一卦
3. **用神选取是断卦关键**，不同事项用神不同
4. **要综合分析世应、月日、动爻等因素**
5. **判断要客观全面**，避免主观臆断

> "易有太极，是生两仪，两仪生四象，四象生八卦。"——《周易·系辞》

六爻占卜是一门需要大量实践的技艺，希望读者在学习理论的同时，多做实际练习，不断提升断卦水平！
