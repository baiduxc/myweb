神煞是八字命理中的辅助判断体系，源自古代星象学与命理学的融合。神煞虽不作为断命的主要依据，但在格局分析的基础上辅以神煞，可以显著提高断命的精细度。

## 神煞概述

### 神煞的性质

| 类型 | 特点 | 影响 |
|------|------|------|
| 吉神 | 主福气、助力 | 增强命局的吉利方面 |
| 凶煞 | 主灾厄、阻碍 | 加重命局的不利因素 |
| 中性神煞 | 因格局而异 | 吉凶取决于组合 |

> 神煞为"辅助"，不可本末倒置。格局为体，神煞为用。

## 主要吉神

### 一、天乙贵人

天乙贵人是八字中最尊贵的吉神，主逢凶化吉、遇难呈祥。

**查法口诀：**

> 甲戊庚牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸兔蛇藏，六辛逢马虎。

| 日干 | 贵人地支 |
|------|----------|
| 甲、戊、庚 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 壬、癸 | 卯、巳 |
| 辛 | 午、寅 |

**天乙贵人的作用：**
- 人际关系良好，容易得到贵人相助
- 遇到困难时容易化险为夷
- 事业上有提携之人

### 三、驿马

驿马主动、主变，代表出行、迁移、变动。

**查法口诀：**

> 申子辰马在寅，寅午戌马在申，巳酉丑马在亥，亥卯未马在巳。

| 年支或日支 | 驿马位 |
|------------|--------|
| 申、子、辰 | 寅 |
| 寅、午、戌 | 申 |
| 巳、酉、丑 | 亥 |
| 亥、卯、未 | 巳 |

**驿马的作用：**
- 主出行、远行、出差
- 主工作变动、迁移
- 驿马逢冲，变动加速

华盖主孤高、才华、宗教哲学倾向。

**查法：**

**华盖的作用：**
- 主聪明好学，偏重哲学宗教
- 性格孤高，不善交际
- 适合从事学术、艺术、宗教事业

## 神煞应用原则

### 神煞判断的注意事项

| 原则 | 说明 |
|------|------|
| 格局为先 | 格局定高低，神煞定细节 |
| 吉煞逢冲减力 | 吉神被冲，福力减弱 |
| 凶煞有制化权 | 凶煞有制，反成助力 |
| 多煞汇聚加重 | 同类神煞多见则加重影响 |
| 不可独断 | 不可仅凭神煞断命 |

## 学习要点总结

1. **辅助定位**：神煞是辅助判断工具，不可取代格局分析
2. **天乙最吉**：天乙贵人是八字中最尊贵的吉神
3. **羊刃需制**：羊刃有制化为权柄，无制则凶
4. **驿马主动**：驿马主变动，逢冲变动加剧
5. **桃花双面**：桃花有利有弊，需看在何宫何位
6. **华盖主孤**：华盖之人聪明但孤高，适合学术艺术
