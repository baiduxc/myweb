## 一、奇门遁甲概述

奇门遁甲是中国古代最高层次的预测术，被誉为"帝王之学"，与太乙神数、六壬神课并称"三式"。

### 1. 名称解析

> "奇"指三奇（乙丙丁），"门"指八门，"遁"指隐藏，"甲"指天干之首。

| 组成 | 含义 |
|------|------|
| 奇 | 三奇——乙为日奇、丙为月奇、丁为星奇 |
| 门 | 八门——休生伤杜景死惊开 |
| 遁 | 隐藏——甲木隐于六仪之下 |
| 甲 | 天干之首——百神之长 |

### 2. 历史渊源

| 时期 | 传说与记载 |
|------|------------|
| 上古 | 黄帝战蚩尤，得奇门之助 |
| 周代 | 姜太公运用奇门辅佐周王 |
| 汉代 | 张良习得奇门，辅佐刘邦 |
| 三国 | 诸葛亮善用奇门 |
| 现代传承 | 多派并存，各有体系 |

### 3. 奇门遁甲的特点

| 特点 | 说明 |
|------|------|
| 天人合一 | 结合天时地理人事 |
| 时空整合 | 时间与空间统一分析 |
| 精确细致 | 可知何人何时何方何事 |
| 趋吉避凶 | 择时择方，选最佳方案 |
