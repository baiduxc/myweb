## 一、八门概述

八门是奇门遁甲预测体系中的核心要素，代表人事活动的各种状态。自古有"八门定吉凶"之说，八门的判断直接决定事情成败。

### 1. 八门的排列位置

| 八门 | 原始宫位 | 五行 | 阴阳 |
|------|----------|------|------|
| 休门 | 坎一宫（北） | 水 | 阳 |
| 生门 | 艮八宫（东北） | 土 | 阳 |
| 伤门 | 震三宫（东） | 木 | 阳 |
| 杜门 | 巽四宫（东南） | 木 | 阴 |
| 景门 | 离九宫（南） | 火 | 阴 |
| 死门 | 坤二宫（西南） | 土 | 阴 |
| 惊门 | 兑七宫（西） | 金 | 阴 |
| 开门 | 乾六宫（西北） | 金 | 阳 |

### 2. 八门的吉凶分类

| 分类 | 八门 | 性质 |
|------|------|------|
| 上吉门 | 休门、生门、开门 | 宜求事 |
| 中平门 | 杜门、景门 | 可用可不用 |
| 下凶门 | 伤门、死门、惊门 | 不宜求事 |

## 二、上吉门详解

### 1. 休门（北方水）

**基本性质**：
- 五行：水
- 方位：北方
- 含义：休息、养生、安逸

| 方面 | 表现 |
|------|------|
| 吉象 | 休养、调整、蓄势、和平 |
| 凶象 | 懒散、不进、停滞、安逸 |
| 适合 | 养生、求和、调解、休息 |
| 不宜 | 积极、创业、进攻 |

**休门的五行生克**：

| 落宫 | 关系 | 吉凶判断 |
|------|------|----------|
| 坎宫 | 伏吟 | 平和安定 |
| 震巽宫 | 休生木 | 泄气，需力 |
| 离宫 | 水克火 | 克出，宜静不宜动 |
| 乾兑宫 | 金生水 | 得助，亨通 |

### 2. 生门（东北土）

**基本性质**：
- 五行：土
- 方位：东北
- 含义：生育、生发、财富

| 方面 | 表现 |
|------|------|
| 吉象 | 生发、生机、财富、新始 |
| 凶象 | 生乱、泛滥、无节制 |
| 适合 | 经商、生育、创业、生财 |
| 不宜 | 杀伐、消亡、终结 |

**生门的重要性**：
> "生门为财门，求财者必看生门，生门有气，财运亨通；生门受制，财运阻滞。"

**生门的五行生克**：

| 落宫 | 关系 | 吉凶判断 |
|------|------|----------|
| 艮坤宫 | 伏吟比和 | 稳定守成 |
| 乾兑宫 | 土生金 | 泄气，转换 |
| 坎宫 | 土克水 | 克出，进取 |
| 震巽宫 | 木克土 | 受克，不宜 |

### 3. 开门（西北金）

**基本性质**：
- 五行：金
- 方位：西北
- 含义：开启、开通、升迁

| 方面 | 表现 |
|------|------|
| 吉象 | 开通、升迁、通达、发展 |
| 凶象 | 散露、浪费、过度开放 |
| 适合 | 开业、升职、出行、疏通 |
| 不宜 | 隐藏、保守、收敛 |

**开门的五行生克**：

| 落宫 | 关系 | 吉凶判断 |
|------|------|----------|
| 乾宫 | 伏吟 | 稳健 |
| 兑宫 | 金金比和 | 助力 |
| 坎宫 | 金生水 | 泄气，分散 |
| 离宫 | 火克金 | 受克，阻滞 |

## 三、中平门详解

### 4. 杜门（东南木）

**基本性质**：
- 五行：木
- 方位：东南
- 含义：杜绝、隐藏、修炼

| 方面 | 表现 |
|------|------|
| 吉象 | 隐藏、修炼、技术、专注 |
| 凶象 | 阻塞、不通、孤僻 |
| 适合 | 修炼、学习、技术、躲藏 |
| 不宜 | 公开、外露、活动 |

**杜门的特殊用法**：
> "杜门宜隐藏，不宜公开。求事者杜门落宫，多主事不外露，需耐等待。"

### 5. 景门（南方火）

**基本性质**：
- 五行：火
- 方位：南方
- 含义：景色、名望、文书

| 方面 | 表现 |
|------|------|
| 吉象 | 名望、文书、宴会、辉煌 |
| 凶象 | 血光、火灾、炫耀过度 |
| 适合 | 求名、文书、宴会、表演 |
| 不宜 | 深藏、低调、收敛 |

**景门的注意事项**：
> 景门虽为中平门，但与血光相关，需结合宫内其他符号判断。

## 四、下凶门详解

### 6. 伤门（东方木）

**基本性质**：
- 五行：木
- 方位：东方
- 含义：伤害、伤痛、争夺

| 方面 | 表现 |
|------|------|
| 吉象 | 进取、争斗、开拓、突破 |
| 凶象 | 伤残、伤害、破财、争斗 |
| 适合 | 竞争、突破、动武 |
| 不宜 | 和平、柔和、养生 |

**伤门的辩证应用**：
> "伤门为凶门，但如需竞争、突破，伤门反而有助。凡事在于配合得当。"

### 7. 死门（西南土）

**基本性质**：
- 五行：土
- 方位：西南
- 含义：死亡、终结、衰败

| 方面 | 表现 |
|------|------|
| 吉象 | 收尾、终结、肃杀、清理 |
| 凶象 | 死亡、病灾、破败、终结 |
| 适合 | 审判、整理、清结算账 |
| 不宜 | 创业、生育、发展 |

**死门的严重性**：
> "死门为八门中最凶之门，求事者死门落宫，多主事不成，需特别谨慎。"

### 8. 惊门（西方金）

**基本性质**：
- 五行：金
- 方位：西方
- 含义：惊恐、官非、口舌

| 方面 | 表现 |
|------|------|
| 吉象 | 警觉、清醒、应变 |
| 凶象 | 惊恐、官司、口舌、是非 |
| 适合 | 应变、警觉、察看 |
| 不宜 | 平稳、安逸、求和 |

**惊门的应用**：
> "惊门主官非口舌，求测官司惊门落宫，需看门与宫的生克关系判断结果。"
