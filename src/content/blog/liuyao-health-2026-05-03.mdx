## 六爻测病的基本原理

疾病预测是六爻占卜的重要应用之一。通过分析卦中官鬼爻的旺衰、所临地支、与世爻的关系等，可以推断疾病的位置、性质、轻重及发展趋势。

### 一、官鬼爻与疾病

#### 官鬼爻的基本含义

在六爻预测中，官鬼爻代表：

| 含义 | 说明 |
|------|------|
| 疾病 | 主要代表病痛和病邪 |
| 病因 | 致病的原因和性质 |
| 病位 | 发病的部位和器官 |
| 医生 | 也代表医生和治疗 |

> **要点：** 官鬼爻是测病的主爻，但需综合分析，不可单看一爻。

#### 官鬼爻的旺衰与病情

| 官鬼状态 | 病情判断 |
|----------|----------|
| 官鬼旺相 | 病情严重，来势凶猛 |
| 官鬼休囚 | 病情较轻，容易康复 |
| 官鬼动变 | 病情有变化，需关注动向 |
| 官鬼空亡 | 病情不实或暂时稳定 |
| 官鬼入墓 | 病情缠绵难愈 |

| 卯 | 肝、手指、神经 | 肝病、手抖、神经痛 |
| 辰 | 胃、胸、肩 | 胃病、胸闷、肩周炎 |
| 巳 | 心、眼、咽喉 | 心脏病、眼疾、咽炎 |
| 午 | 心、眼、舌 | 心脏病、眼疾、舌病 |
| 未 | 脾、胃、脊椎 | 脾胃病、脊椎问题 |
| 申 | 大肠、肺、骨骼 | 肠炎、肺炎、骨病 |

### 三、断卦技法详解

#### 世爻与病人

世爻代表病人自身：

| 世爻状态 | 病人情况 |
|----------|----------|
| 世爻旺相 | 正气足，抵抗力强 |
| 世爻休囚 | 正气弱，抵抗力差 |
| 世爻受克 | 病情加重 |
| 世爻得生 | 有助康复 |
| 世爻空亡 | 精神不振或诊断有误 |

#### 用神的选取

不同情况选取不同用神：

| 测事类型 | 用神 | 说明 |
|----------|------|------|
| 自测病 | 世爻 | 世为自身 |
| 代人测 | 按六亲取 | 父母测子取子孙等 |
| 测医生 | 应爻 | 应为对方 |
| 测药效 | 子孙爻 | 子孙制官鬼 |

#### 卦象吉凶判断

**吉兆：**

| 情况 | 说明 |
|------|------|
| 官鬼休囚空亡 | 病邪无力 |
| 子孙爻旺相 | 药效好，能治病 |
| 世爻旺相得生 | 正气足，有恢复力 |
| 卦逢六合 | 病情稳定 |

**凶兆：**

| 情况 | 说明 |
|------|------|
| 官鬼旺相克世 | 病情严重 |
| 世爻休囚受克 | 正气虚弱 |
| 子孙爻空破 | 药无效，治疗效果差 |
| 卦逢六冲 | 病情反复多变 |

### 四、特殊情况处理

#### 官鬼多现

如果卦中出现多个官鬼：

- 官鬼多现，病情复杂
- 需判断何爻发动为主
- 动爻代表当前主要问题
- 静爻代表潜在问题

#### 游魂卦与归魂卦

| 卦类型 | 特点 | 应对 |
|--------|------|------|
| 游魂卦 | 病情游走不定 | 全面检查 |
| 归魂卦 | 旧病复发 | 注意调理 |

#### 反吟与伏吟

| 情况 | 含义 | 说明 |
|------|------|------|
| 反吟 | 病情反复 | 忽好忽坏 |
| 伏吟 | 病情缠绵 | 久治不愈 |

### 五、疾病预测案例

#### 案例一：预测感冒

某人自测感冒，得卦：

```
【本卦】山水蒙
　　　　　　白虎　　　　　　父母寅木　━　━━
　　　　　　螣蛇　　　　　　官鬼子水　━　━━
　　　　　　勾陈　　　　　　子孙戌土　━　━━　世
　　　　　　朱雀　　　兄弟午火　子孙辰土　━　━━
　　　　　　青龙　　　　　　父母寅木　━　━━
　　　　　　玄武　　　　　　官鬼子水　━　━━　应
```

分析：
- 世爻子孙辰土，月扶日生，旺相
- 官鬼子水在五爻，临螣蛇，主呼吸系统
- 子孙旺相制官鬼，病情轻
- 断：普通感冒，一周内可愈

#### 案例二：预测胃病

某人为父亲测胃病：

```
【本卦】地水师
【变卦】地天师
　　　　　　白虎　　　　　　父母酉金　━　━━　应
　　　　　　螣蛇　　　　　　兄弟亥水　━　━━
　　　　　　勾陈　　　　　　官鬼丑土　━　━━
　　　　　　朱雀　　　　　　官鬼辰土　━　━━　世
　　　　　　青龙　　官鬼丑土×妻财午火　━━━━　
　　　　　　玄武　　　　　　子孙卯木　━　━━
```

分析：
- 父测病，取父母爻为用
- 父母酉金在应爻，月克日泄，休囚
- 二爻官鬼丑土临青龙动，化妻财午火
- 官鬼旺相克父母，病情较重
- 建议：积极治疗，注意饮食调理

### 六、预测注意事项

#### 断卦原则

1. **不可代替医学诊断**
   六爻预测仅供参考，不能替代正规医学检查和治疗。

2. **综合分析**
   不要单看一个爻，需综合世、应、官鬼、子孙等爻全面分析。

3. **注意时间因素**
   月令、日辰对爻的影响很大，需准确判断旺衰。

4. **结合实际情况**
   断卦时需结合病人实际症状，不可脱离实际空断。

#### 伦理规范

| 禁止行为 | 原因 |
|----------|------|
| 不准夸大病情 | 避免造成恐慌 |
| 不准推荐非正规治疗 | 可能延误病情 |
| 不准代替医生诊断 | 涉及医疗责任 |
| 不准收取高额费用 | 有违道德规范 |

### 七、辅助判断技法

#### 六神与疾病

| 六神 | 疾病倾向 |
|------|----------|
| 青龙 | 与酒色、喜事有关的病 |
| 朱雀 | 与情绪、语言有关的病 |
| 勾陈 | 与饮食、消化有关的病 |
| 螣蛇 | 与惊恐、神经有关的病 |
| 白虎 | 与血光、手术有关的病 |
| 玄武 | 与隐私、暗疾有关的病 |

#### 卦身与病位

| 卦身位置 | 病位判断 |
|----------|----------|
| 卦身在上卦 | 病在上身或头部 |
| 卦身在下卦 | 病在下身或四肢 |
| 卦身临官鬼 | 正在病中 |
| 卦身临子孙 | 有康复之象 |

## 学习要点

1. **官鬼主病**：官鬼爻旺衰代表病情轻重
2. **地支病位**：不同地支对应不同人体部位
3. **五行病性**：金木水火土对应不同疾病性质
4. **世应分析**：世为病人，应可代表医生
5. **子孙为药**：子孙制官鬼，代表治疗效果
6. **注意规范**：不可替代医学诊断，需遵守伦理

六爻预测疾病是一门传统技艺，应在遵守伦理的前提下，作为健康管理的辅助参考。
