# 奇门遁甲八门详解：开门休门生门伤门杜门景门死门惊门的奥秘

八门是奇门遁甲"人盘"的核心要素，代表人的活动与行为。八门有吉门凶门之分，配合九宫、九星，可断人事之吉凶成败。本文将全面解析八门的属性与实战应用。

## 一、八门的基本概念

### 1.1 什么是八门

八门，又称人盘，是奇门遁甲三盘（天盘、地盘、人盘）之一，代表人事活动：

> **天时地利人和，八门主人和**

八门在奇门遁甲中的地位：
- 天盘九星：代表天道、时机
- 地盘八宫：代表地利、环境
- 人盘八门：代表人和、行为

|------|------|------|------|----------|
| 开门 | 乾 | 金 | 西北 | 乾六宫 |
| 休门 | 坎 | 水 | 正北 | 坎一宫 |
| 生门 | 艮 | 土 | 东北 | 艮八宫 |
| 伤门 | 震 | 木 | 正东 | 震三宫 |

| 死门 | 坤 | 土 | 西南 | 坤二宫 |
| 惊门 | 兑 | 金 | 正西 | 兑七宫 |

### 1.3 八门吉凶分类

|------|------|----------|
| 三吉门 | 开门、休门、生门 | 主顺遂、吉利、有成 |
| 中平门 | 杜门、景门 | 可吉可凶，需配合看 |
| 三凶门 | 伤门、死门、惊门 | 主破损、灾殃、惊恐 |

## 二、八门详解

### 2.1 开门

**基本属性**：
- 五行：金
- 八卦：乾
- 方位：西北
- 节气：立冬后至大雪前

**开门含义**：

| 方面 | 含义 |
|------|------|
| 字义 | 开启、公开、升迁 |
| 职业 | 官职、公职、领导 |
| 象征 | 权威、名望、开放 |
| 事项 | 升职、开业、公开、外出 |

**开门旺衰表**：

| 宫位 | 旺衰 | 断语 |
|------|------|------|
| 乾宫（本宫） | 大旺 | 官运亨通，得心应手 |
| 兑宫（金） | 旺相 | 有贵人相助，事业顺遂 |
| 坤宫（土） | 相生 | 有根基，可成大业 |
| 离宫（火） | 受克 | 事业受阻，需待时 |
| 震宫（木） | 受泄 | 劳而无功，白费力气 |
| 坎宫（水） | 泄气 | 付出多收获少 |
| 艮宫（土） | 相生 | 有贵人提携 |
| 巽宫（木） | 受泄 | 多有阻滞 |

**开门实战应用**：

| 测事 | 开门临宫断语 |
|------|-------------|
| 求官 | 临吉星，升职有望 |
| 开业 | 临生门或戊，生意兴隆 |
| 出行 | 临九天，远行大利 |
| 考试 | 临文昌，金榜题名 |

## 学习要点

1. **八门主人和**，代表人事活动与行为
2. **三吉门**（开门、休门、生门）用于趋吉，**三凶门**（伤门、死门、惊门）用于避险
3. **八门的旺衰**与宫位、节气密切相关
4. **门与宫的生克关系**决定做事难易
5. 八门需**与九星、八神、天干**配合看
6. 八门临空亡需**分吉凶**——吉空不吉，凶空不凶
7. 实战中**开门测事业、生门测财运、六合测婚姻**是常用法
8. 学习八门需**实战练习**，在实践中体会其变化奥妙
