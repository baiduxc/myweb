## 一、五行概述

五行学说是中国传统文化的哲学基础，贯穿于中医、命理、风水、占卜等各个领域。

### 1. 五行的来源

> "天地之间，一气而已，分为阴阳，衍为五行。"

五行源于古人对自然现象的观察与总结：

| 五行 | 自然现象 |
|------|----------|
| 木 | 植物、风、春天 |
| 火 | 火焰、热、夏天 |
| 土 | 土壤、中心、长夏 |
| 金 | 金属、矿物、秋天 |
| 水 | 液体、寒、冬天 |

### 2. 五行的特性

| 五行 | 特性 | 关键词 |
|------|------|--------|
| 木 | 曲直、生长、舒展 | 生发 |
| 火 | 炎上、光明、热情 | 升腾 |
| 土 | 稼穑、承载、孕育 | 承载 |
| 金 | 从革、肃杀、收敛 | 收敛 |
| 水 | 润下、寒冷、流动 | 流动 |

## 二、五行相生

### 1. 相生顺序

> 木生火，火生土，土生金，金生水，水生木。

| 相生 | 原理 | 现象 |
|------|------|------|
| 木生火 | 木燃烧生火 | 钻木取火 |
| 火生土 | 火烧成灰烬 | 焚烧后的灰 |
| 土生金 | 土中蕴藏金 | 矿石金矿 |
| 金生水 | 金属凝结水 | 露珠凝聚 |
| 水生木 | 水滋润木 | 雨露润物 |

### 2. 相生的深层含义

五行相生代表事物的发展变化：

| 相生 | 象征意义 |
|------|----------|
| 木生火 | 行动创造成果 |
| 火生土 | 成果演变为根基 |
| 土生金 | 根基产生价值 |
| 金生水 | 价值转化为流动 |
| 水生木 | 流动促进新生 |

### 3. 相生的主客关系

| 关系 | 我 | 他 | 含义 |
|------|-----|-----|------|
| 生我者 | 为子 | 为母 | 母生子 |
| 我生者 | 为母 | 为子 | 子得生 |

| 火克金 | 火熔金 | 高温熔化金属 |
| 金克木 | 金伐木 | 斧头砍树 |

### 2. 相克的深层含义

| 相克 | 象征意义 |

| 土克水 | 稳定控制流动 |

## 四、五行生克的外延

### 1. 泄（生者为泄）

**含义**：生者消耗自身能量去生

> 我生者为泄，泄为力量流失。

| 泄的情况 | 影响 |
|----------|------|
| 木生火 | 木泄气 |
| 火生土 | 火泄气 |
| 土生金 | 土泄气 |
| 金生水 | 金泄气 |
| 水生木 | 水泄气 |

### 2. 耗（克者为耗）

**含义**：克者消耗能量去克

> 我克者为耗，耗为能量损失。

| 耗的情况 | 影响 |
|----------|------|
| 木克土 | 木耗费 |
| 土克水 | 土耗费 |
| 水克火 | 水耗费 |
| 火克金 | 火耗费 |
| 金克木 | 金耗费 |

### 3. 五行力量变化图

> 以"木"为例：

| 木与其他五行关系 | 木的力量变化 |
|------------------|--------------|
| 木生火 | 木泄气，力量减 |
| 木克土 | 木耗力，力量减 |
| 水生木 | 木得助，力量增 |
| 金克木 | 木受制，力量减 |

## 五、五行相乘相侮

### 1. 相乘（过度克制）

**含义**：克制太过，超出原本的克制限度

> "乘者，过也。克制过度，犹如欺凌。"

| 相乘情况 | 条件 | 后果 |
|----------|------|------|
| 木乘土 | 木太旺或土太弱 | 土气衰败 |
| 土乘水 | 土太旺或水太弱 | 水气枯竭 |
| 水乘火 | 水太旺或火太弱 | 火气熄灭 |
| 火乘金 | 火太旺或金太弱 | 金气消熔 |
| 金乘木 | 金太旺或木太弱 | 木气夭折 |

### 2. 相侮（反克）

**含义**：被克者反而克制主克者

> "侮者，逆也。弱者反克强者，谓之侮。"

| 相侮情况 | 条件 | 原因 |
|----------|------|------|
| 木侮金 | 木太旺或金太弱 | 强弱逆转 |
| 土侮木 | 土太旺或木太弱 | 克者无力 |
| 水侮土 | 水太旺或土太弱 | 反克主克 |
| 火侮水 | 火太旺或水太弱 | 热量蒸发 |
| 金侮火 | 金太旺或火太弱 | 金多火灭 |

### 3. 乘侮的辩证关系

> 乘侮互为因果，克制太过则可能导致反克。

| 正常相克 | 相乘 | 相侮 |
|----------|------|------|
| 金克木 | 金过旺乘木 | 木过旺侮金 |
| 木克土 | 木过旺乘土 | 土过旺侮木 |
| 土克水 | 土过旺乘水 | 水过旺侮土 |
| 水克火 | 水过旺乘火 | 火过旺侮水 |
| 火克金 | 火过旺乘金 | 金过旺侮火 |

### 3. 旺衰的变化规律

> 春天：木旺，因木应春，生于此时力量最强。

**推演方法**：
- 当季者旺
- 我生者相（泄气）
- 生我者休（得气）
- 克我者囚（受制）
- 我克者死（无力）
