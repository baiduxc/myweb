# 周易六十四卦概述

《周易》以六十四卦为核心，构建了一套完整的宇宙变化模型。从伏羲画卦到文王演易，六十四卦体系历经数千年，至今仍是中华文化的瑰宝。

## 从八卦到六十四卦

### 八卦

八卦是六十四卦的基础单元，由三个爻组成（经卦）：

| 卦名 | 符号 | 自然象 | 属性 | 方位 |
|------|------|--------|------|------|
| 乾 | ☰ | 天 | 刚健 | 西北 |
| 坤 | ☷ | 地 | 顺从 | 西南 |
| 震 | ☳ | 雷 | 动 | 东 |
| 巽 | ☴ | 风 | 入 | 东南 |
| 坎 | ☵ | 水 | 险 | 北 |
| 离 | ☲ | 火 | 明 | 南 |
| 艮 | ☶ | 山 | 止 | 东北 |
| 兑 | ☱ | 泽 | 悦 | 西 |

### 六十四卦的生成

六十四卦由八卦两两相叠而成（别卦）：

> **8 × 8 = 64**

每卦六爻，下卦（内卦）为体，上卦（外卦）为用。六十四卦覆盖了天地人事的种种状态与变化。

## 六十四卦分类

### 按上下卦关系分

#### 八纯卦

上下卦相同，共八卦：

| 卦名 | 卦象 | 含义 |
|------|------|------|
| 乾为天 | 乾上乾下 | 刚健不息 |
| 坤为地 | 坤上坤下 | 厚德载物 |
| 震为雷 | 震上震下 | 惊雷奋进 |
| 巽为风 | 巽上巽下 | 谦逊深入 |
| 坎为水 | 坎上坎下 | 重险在前 |
| 离为火 | 离上离下 | 光明相继 |
| 艮为山 | 艮上艮下 | 知止而安 |
| 兑为泽 | 兑上兑下 | 喜悦和乐 |

#### 错卦与综卦

- **错卦**：阴阳爻全部互换，如乾与坤
- **综卦**：卦象上下颠倒，如屯与蒙

---

### 按卦序分

通行本《周易》的卦序，始于乾坤，终于既济未济，蕴含深意：

| 序号区间 | 卦名 | 主题 |
|----------|------|------|
| 1-2 | 乾、坤 | 天地定位 |
| 3-6 | 屯、蒙、需、讼 | 初始阶段 |
| 7-10 | 师、比、小畜、履 | 社会形成 |
| 11-12 | 泰、否 | 通泰与闭塞 |
| 63-64 | 既济、未济 | 完成与新生 |

## 卦的基本结构

每个卦由以下要素组成：

### 六爻

从下往上依次为：初爻、二爻、三爻、四爻、五爻、上爻。

- 阳爻记为"九"（如初九、九二）
- 阴爻记为"六"（如初六、六二）

### 卦辞与爻辞

| 组成 | 说明 |
|------|------|
| **卦辞** | 整卦的判断语，概述卦的吉凶 |
| **爻辞** | 每一爻的判断语，描述具体情境 |

### 爻位含义

| 爻位 | 含义 |
|------|------|
| 初爻 | 事物起始，根基 |
| 二爻 | 内部中位，臣 |
| 三爻 | 内卦之极，转折 |
| 四爻 | 外卦之始，近君 |
| 五爻 | 外卦中位，君 |
| 上爻 | 事物终了，极变 |

## 重要的卦

### 乾坤二卦

乾坤为六十四卦之门户：

> **乾**：元亨利贞。天行健，君子以自强不息。
> **坤**：元亨，利牝马之贞。地势坤，君子以厚德载物。

乾代表刚健、创造、主动；坤代表柔顺、承载、包容。

### 既济与未济

> **既济**：事已成功，但需守成。
> **未济**：事未完成，但蕴含希望。

六十四卦以"未济"结尾，暗示宇宙永无止境的变化循环。

---

## 卦变与变卦

### 变爻

占卜时，老阳（九）和老阴（六）为变爻：

- 老阳（⚊→⚋）：阳变阴
- 老阴（⚋→⚊）：阴变阳

变爻产生变卦（之卦），反映事物的变化趋势。

### 互卦

取二三四爻为下卦，三四五爻为上卦，组成互卦。互卦反映事物的中间过程和内在因素。
