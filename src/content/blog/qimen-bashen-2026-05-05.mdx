奇门遁甲八神是盘局中最重要的信息符号之一，代表着天时、神灵、贵人等无形力量。八神虽无形无象，却能在预测中发挥关键作用，揭示事物背后的神秘因素。

## 一、八神概述

### 1. 八神的构成

八神共八位，分为阴阳两组：

| 阳遁八神（顺布） | 阴遁八神（逆布） |
|----------------|----------------|
| 值符 | 值符 |
| 腾蛇 | 九地 |
| 太阴 | 九天 |
| 六合 | 朱雀（玄武） |
| 白虎 | 勾陈（白虎） |
| 玄武 | 九地 |
| 九地 | 九天 |
| 九天 | 九地 |

> **注**：在实际应用中，八神按照阳顺阴逆的方式排列于九宫之上。

### 2. 八神的作用

八神在奇门预测中的作用：
- 代表无形的影响因素
- 揭示事情背后的神秘力量
- 显示贵人与小人
- 指示事情的成败关键

## 二、八神详解

### 1. 值符（首领之神）

**基本象义**：
- 性质：吉神，八神之首
- 象征：领导、贵人、权威、正气
- 五行：中央土
- 方位：中央

**应用判断**：

| 落宫情况 | 判断 |
|---------|-----|
| 值符落旺相之宫 | 贵人有力，事情顺遂 |
| 值符落休囚之宫 | 贵人无力，需自力更生 |
| 值符逢庚 | 贵人受阻，办事有碍 |
| 值符逢辛 | 有罪责官非之虞 |

**实际运用**：
- 求人办事，看值符落宫
- 测事业，值符为领导、上司
- 测诉讼，值符为法官、执法者
- 值符所落之宫，往往有贵人相助

### 2. 腾蛇（变化之神）

**基本象义**：
- 性质：凶神，主惊恐、怪异
- 象征：惊恐、怪异、缠绕、变动、虚惊
- 五行：火
- 特点：虚幻不实，变化多端

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 腾蛇临吉门吉星 | 怪事化为平事 |
| 腾蛇临凶门凶星 | 惊恐怪异，诸事不顺 |
| 腾蛇临玄武 | 有欺诈、被骗之象 |
| 腾蛇逢空 | 虚惊一场 |

**实际运用**：
- 测出行，腾蛇临门，有惊恐之事
- 测疾病，腾蛇临身，怪病难断
- 测婚姻，腾蛇临宫，感情纠葛
- 腾蛇主缠绕，遇之需防反复

### 3. 太阴（隐蔽之神）

**基本象义**：
- 性质：吉神，主隐匿、阴谋
- 象征：隐私、隐蔽、阴谋、女性贵人
- 五行：金
- 方位：西方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 太阴临吉门 | 隐秘之事有成 |
| 太阴临凶门 | 阴私之事败露 |
| 太阴合吉星 | 有女性贵人相助 |
| 太阴逢空 | 计划落空 |

**实际运用**：
- 测合作，太阴临宫，暗中有人相助
- 测谋事，太阴临门，可暗中行事
- 测疾病，太阴临身，病因隐秘
- 女性求测，太阴为用神

### 4. 六合（和合之神）

**基本象义**：
- 性质：吉神，主和合、团聚
- 象征：合作、婚姻、中介、介绍人
- 五行：木
- 方位：东方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 六合临吉门吉星 | 合作顺利，婚姻美满 |
| 六合临凶门 | 合作破裂，婚姻不顺 |
| 六合逢庚 | 合作有阻，婚姻有碍 |
| 六合逢空 | 合而不合，聚而复散 |

**实际运用**：
- 测合作，六合临宫为用神
- 测婚姻，六合临夫妻宫，感情和美
- 测中介生意，六合临门，有成交之象
- 六合主人际关系，遇之多有贵人牵线

### 5. 白虎（凶煞之神）

**基本象义**：
- 性质：凶神，主血光、凶灾
- 象征：血光、官司、凶灾、阻隔、道路
- 五行：金
- 方位：西方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 白虎临吉门 | 凶中有吉，灾祸减轻 |
| 白虎临凶门 | 大凶之象 |
| 白虎逢庚 | 官司口舌 |
| 白虎逢死门 | 有伤灾血光 |

**实际运用**：
- 测出行，白虎临门，防车祸凶灾
- 测疾病，白虎临身，病重有手术之象
- 测官司，白虎临宫，官司缠身
- 白虎虽凶，遇吉门可化解

### 6. 玄武（欺诈之神）

**基本象义**：
- 性质：凶神，主欺诈、盗窃
- 象征：欺诈、盗窃、小人、暧昧、桃花
- 五行：水
- 方位：北方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 玄武临吉门 | 欺诈之事不遂 |
| 玄武临凶门 | 被骗被盗 |
| 玄武逢庚 | 盗贼上门 |
| 玄武逢空 | 骗局落空 |

**实际运用**：
- 测交易，玄武临门，防被骗
- 测失物，玄武临宫，被盗之象
- 测婚姻，玄武临夫妻宫，有外遇之嫌
- 玄武主人际中的小人，遇之需防

### 7. 九地（积蓄之神）

**基本象义**：
- 性质：吉神，主积蓄、稳重
- 象征：积蓄、稳重、隐蔽、防守、等待
- 五行：土
- 方位：坤方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 九地临吉门 | 积蓄有成，稳中求进 |
| 九地临凶门 | 隐伏之灾 |
| 九地逢冲 | 安稳不保 |
| 九地临旺宫 | 大吉 |

**实际运用**：
- 测投资，九地临宫，宜守不宜攻
- 测求职，九地临门，稳中有进
- 测疾病，九地临身，病情稳定
- 九地主稳，不宜急躁行事

### 8. 九天（远行之神）

**基本象义**：
- 性质：吉神，主远行、张扬
- 象征：远行、张扬、志向、高升、宣传
- 五行：金
- 方位：乾方

**应用判断**：

| 情况 | 判断 |
|-----|-----|
| 九天临吉门 | 远行有利，志向得遂 |
| 九天临凶门 | 远行有阻，张扬招祸 |
| 九天逢空 | 志向难酬 |
| 九天临旺宫 | 声名远扬 |

**实际运用**：
- 测出行，九天临门，利远行
- 测事业，九天临宫，志向远大
- 测宣传，九天临门，效果显著
- 九天主动，宜进不宜守

## 三、八神组合判断

### 吉神组合

| 组合 | 象义 |
|-----|-----|
| 值符+太阴 | 贵人暗中相助 |
| 值符+六合 | 贵人牵线搭桥 |
| 六合+太阴 | 暗中合作成功 |
| 九地+九天 | 进退有度 |

### 凶神组合

| 组合 | 象义 |
|-----|-----|
| 腾蛇+玄武 | 连续被骗 |
| 白虎+腾蛇 | 惊恐血光 |

| 玄武+腾蛇 | 欺诈缠身 |

## 学习要点

1. **八神有吉凶**：值符、太阴、六合、九地、九天为吉；腾蛇、白虎、玄武为凶
2. **吉凶可转化**：吉神临凶门可减吉，凶神临吉门可减凶
3. **值符最尊贵**：代表领导、贵人、正义力量
4. **白虎主血光**：遇之需防凶灾
5. **玄武主欺诈**：遇之需防被骗
6. **八神需结合门星**：综合判断方为准确
7. **八神应验快**：在短时间预测中作用明显

## 结语

八神是奇门遁甲中最具神秘色彩的部分，代表着无形的力量。在预测实践中，八神往往能揭示事情背后的关键因素，为判断提供重要参考。

掌握八神，需要理解其象征意义，灵活运用，不可生搬硬套。真正的高手，能从八神中读出事情的深层信息，做出精准判断。
