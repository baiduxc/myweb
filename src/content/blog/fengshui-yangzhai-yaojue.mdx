阳宅风水是堪舆学中与人们日常生活最密切相关的分支，研究人居环境与自然能量的和谐关系。好的阳宅风水能旺丁旺财，差的则可能导致疾病、破财、家庭不和。

## 阳宅三要

阳宅风水有三个核心要素，简称"阳宅三要"：

| 要素 | 含义 | 重要性 |
|------|------|--------|
| **门** | 大门——气口 | 第一要位 |
| **主** | 主卧/主房——居者之所 | 第二要位 |
| **灶** | 厨房灶位——养命之源 | 第三要位 |

> 门主灶三者相生比和为吉，相克为凶。此为阳宅风水第一原则。

## 阳宅外部形局

### 四灵山诀

理想的阳宅外部形局应具备"四灵"护卫：

| 灵兽 | 方位 | 要求 | 作用 |
|------|------|------|------|
| **青龙** | 左（东） | 高耸蜿蜒 | 主贵人、男性运势 |
| **白虎** | 右（西） | 低于青龙 | 主女性、财运 |
| **朱雀** | 前（南） | 开阔明亮 | 主前途、名声 |
| **玄武** | 后（北） | 厚实靠山 | 主靠山、根基 |

> **左青龙宜高，右白虎宜低，前朱雀宜明，后玄武宜实。**

### 外部形煞

| 煞名 | 描述 | 影响 |
|------|------|------|
| 路冲 | 道路直冲大门 | 血光、意外 |
| 角煞 | 屋角对冲 | 口舌、疾病 |
| 天斩煞 | 两楼夹缝对冲 | 破财、灾厄 |
| 反弓煞 | 道路/河流反弓 | 退财、人散 |
| 穿心煞 | 大门正对走廊/楼梯 | 漏财、不安 |
| 孤阳煞 | 附近有高压电塔 | 烦躁、失眠 |
| 独阴煞 | 附近有殡仪馆/医院 | 阴气、健康 |

## 阳宅内部布局

### 大门风水

大门是阳宅的气口，决定气的进出：

| 要点 | 说明 |
|------|------|
| 大门宜大 | 气口大则纳气多 |
| 大门宜亮 | 光线充足则阳气旺 |
| 忌对后门 | 前后门直通，气直泄不聚 |
| 忌对电梯 | 电梯开合如剪刀，割断宅气 |
| 忌对楼梯 | 楼梯向下泄气，向上冲气 |
| 门色宜吉 | 根据宅卦选门色 |

### 客厅风水

客厅为明堂，是家宅的面子：

| 要点 | 说明 |
|------|------|
| 宜方正 | 方正聚气，缺角泄气 |
| 宜明亮 | 明厅暗房，阳气充足 |
| 沙发靠实墙 | 有靠山，主安稳 |
| 不宜横梁压顶 | 压运压健康 |
| 财位宜生 | 客厅对角线位置为财位 |

### 卧室风水

卧室关系健康与感情：

| 要点 | 说明 |
|------|------|
| 床头靠实墙 | 有靠山，睡眠安稳 |
| 床不对门 | 睡不安稳，缺乏隐私 |
| 床上无横梁 | 横梁压顶，影响健康 |
| 镜不对床 | 夜间惊魂，影响感情 |
| 不宜过多电器 | 电磁干扰，影响睡眠 |

### 厨房风水

厨房为养命之源：

| 要点 | 说明 |
|------|------|
| 灶不对门 | 对门则财气外泄 |
| 水火不冲 | 灶与水龙头不宜正对 |
| 灶后有靠 | 靠墙则主有靠山 |
| 保持整洁 | 脏乱则病从口入 |

### 卫生间风水

卫生间为排污之处：

| 要点 | 说明 |
|------|------|
| 不在中宫 | 污秽居中，影响全家 |
| 不对大门 | 对门则财气被污 |
| 不对厨房 | 水火相冲 |
| 保持通风 | 浊气需排出 |

## 八宅风水

八宅法是阳宅风水最常用的流派之一。

### 东四宅与西四宅

| 类型 | 卦名 | 适合命卦 |
|------|------|----------|
| **东四宅** | 震（东）、巽（东南）、坎（北）、离（南） | 东四命 |
| **西四宅** | 乾（西北）、坤（西南）、兑（西）、艮（东北） | 西四命 |

### 命卦计算

**男性**：(100-出生年后两位) ÷ 9，余数为命卦

**女性**：(出生年后两位-4) ÷ 9，余数为命卦

余数对应卦：1坎、2坤、3震、4巽、5坤（男）/艮（女）、6乾、7兑、8艮、9离

### 四吉方与四凶方

| 方位 | 含义 | 适合 |
|------|------|------|
| **生气** | 大吉 | 卧室、大门 |
| **天医** | 次吉 | 厨房、卧室 |
| **延年** | 吉 | 客厅、卧室 |
| **伏位** | 小吉 | 起居、书房 |
| **祸害** | 小凶 | 储物间 |
| **六煞** | 凶 | 卫生间 |
| **五鬼** | 大凶 | 不宜使用 |
| **绝命** | 最凶 | 避开不用 |
