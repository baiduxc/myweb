## 一、五行旺衰的基本概念

五行旺衰是指五行在不同时空条件下的强弱变化，是命理分析的核心技术。

### 1. 旺衰的定义

> "当令者旺，我生者相，生我者休，克我者囚，我克者死。"

| 状态 | 五行 | 含义 |
|------|------|------|
| 旺 | 当季五行 | 最强 |
| 相 | 旺所生者 | 次强 |
| 休 | 生旺者 | 中等 |
| 囚 | 克旺者 | 较弱 |
| 死 | 旺所克者 | 最弱 |

### 2. 旺衰的来源

五行旺衰来源于季节变化：

| 季节 | 主气 | 旺衰状态 |
|------|------|----------|
| 春 | 木 | 木旺 |
| 夏 | 火 | 火旺 |
| 长夏 | 土 | 土旺 |
| 秋 | 金 | 金旺 |
| 冬 | 水 | 水旺 |

## 四、五行旺衰的记忆方法

### 1. 口诀记忆法

> **旺相休囚死口诀**：
> 
> 当令者旺我生相，生我为休克我囚。
> 我克为死是真机，五季循环莫忘记。

### 2. 推导方法

从当令五行开始，按生克顺序推导：

**以春季为例**：
1. 春天木当令 → 木旺
2. 木生什么？木生火 → 火相
3. 什么生木？水生木 → 水休
4. 什么克木？金克木 → 金囚
5. 木克什么？木克土 → 土死

## 七、八字中的五行旺衰判断

### 1. 综合判断方法

判断一个八字的五行旺衰，需综合考察：

| 考察要素 | 内容 |
|----------|------|
| 月令 | 最重要，占50% |
| 天干 | 考察得令与否 |
| 地支 | 考察根气 |
| 合化 | 考察变化 |
| 生克 | 考察助益损耗 |

### 2. 月令的重要性

> "得令者旺，失令者衰。月令为提纲，是判断五行旺衰的关键。"

| 情况 | 判断 |
|------|------|
| 日主得月令 | 身旺 |
| 日主失月令 | 身弱 |
| 月令有根 | 更有力 |
| 月令无根 | 需他助 |

### 3. 实例分析

**示例**：甲木日主生于寅月

| 分析 | 结论 |
|------|------|
| 日主为甲木 | 木主 |
| 生于寅月 | 木当令 |
| 甲木得令 | 身旺 |

**示例**：丙火日主生于子月

| 分析 | 结论 |
|------|------|
| 日主为丙火 | 火主 |
| 生于子月 | 水当令 |
| 火不得令 | 身弱 |
