# 六爻卦身详解：卦身的取法与断卦应用

卦身是六爻预测中一个重要概念，它可以帮助判断事物的真假、虚实、成败以及应期。掌握卦身的取法和应用，可以提高断卦的准确性。

## 一、什么是卦身

### 1. 卦身的定义

卦身，又称月卦身、身宫，是根据卦象推算出来的一个特殊爻位。它代表：
- 事物的本体
- 事情的主体
- 预测的核心

> 《增删卜易》："卦身者，乃一卦之主，主事之爻也。"

### 2. 卦身的作用

| 作用 | 说明 |
|-----|------|
| **定真假** | 卦身旺相为真，休囚为假 |
| **判虚实** | 卦身有气为实，无气为虚 |
| **看成败** | 卦身生合为成，冲克为败 |
| **推应期** | 卦身临值或受冲为应期 |

## 二、卦身的取法

### 1. 口诀法

**取卦身口诀**：
> "阴世则从午月起，阳世还从子月生，欲得卦身名月卦，初爻起数至世停。"

**解析**：
- 阳爻持世：从子月起数
- 阴爻持世：从午月起数
- 从初爻开始数，数到世爻停止

### 2. 操作步骤

**步骤**：
1. 确定世爻是阳爻还是阴爻
2. 阳爻持世，从子月开始；阴爻持世，从午月开始
3. 从初爻开始往上数
4. 数到世爻为止，对应的月支即为卦身

**示例**：

假设某卦世爻在五爻，为阳爻：

| 爻位 | 月支 |
|-----|------|
| 初爻 | 子 |
| 二爻 | 丑 |
| 三爻 | 寅 |
| 四爻 | 卯 |
| 五爻（世爻）| 辳（卦身）|

### 3. 查表法

**卦身速查表**（阳爻持世）：

| 世爻位置 | 卦身 |
|---------|------|
| 初爻 | 子 |
| 二爻 | 丑 |
| 三爻 | 寅 |
| 四爻 | 卯 |
| 五爻 | 辳 |
| 六爻 | 巳 |

**卦身速查表**（阴爻持世）：

| 世爻位置 | 卦身 |
|---------|------|
| 初爻 | 午 |
| 二爻 | 未 |
| 三爻 | 申 |
| 四爻 | 酉 |
| 五爻 | 戌 |
| 六爻 | 亥 |

## 三、卦身的意义

### 1. 代表事物主体

- 卦身旺：事物实在、可行
- 卦身弱：事物虚假、难成

### 2. 与用神的关系

| 关系 | 含义 |
|-----|------|
| 卦身生用神 | 事情顺利，有助益 |
| 用神生卦身 | 事情需依赖他人 |
| 卦身克用神 | 有阻滞，不顺遂 |
| 用神克卦身 | 事情有阻力 |
| 卦身比和用神 | 事情平稳 |

### 3. 卦身临六亲

| 六亲 | 意义 |
|-----|------|
| 卦身临父母 | 与文书、长辈、房屋有关 |
| 卦身临兄弟 | 与同辈、竞争、阻隔有关 |
| 卦身临子孙 | 与晚辈、福气、解灾有关 |
| 卦身临妻财 | 与钱财、女性、感情有关 |
| 卦身临官鬼 | 与工作、男性、疾病有关 |

## 四、卦身的应用

### 1. 判断真假

**真事**：
- 卦身旺相有气
- 卦身临贵人吉神
- 卦身不受冲克

**假事/骗局**：
- 卦身休囚无气
- 卦身空亡
- 卦身受严重冲克

### 2. 判断应期

**应期原则**：

| 情况 | 应期 |
|-----|------|
| 卦身旺相 | 卦身临值之月日 |
| 卦身空亡 | 卦身出空之月日 |
| 卦身入墓 | 卦身冲墓之月日 |
| 卦身受克 | 卦身得救之月日 |

### 3. 判断成败

**成事条件**：
- 卦身旺相
- 卦身生合用神
- 卦身不受冲克

**败事条件**：
- 卦身休囚
- 卦身冲克用神
- 卦身严重受克

## 五、卦身与世身的关系

### 1. 世身

世身是以世爻为主计算的：
- 阳爻持世，从子上起数
- 阴爻持世，从午上起数
- 方法与卦身相同

### 2. 区别

| 项目 | 卦身 | 世身 |
|-----|------|------|
| **计算依据** | 卦象 | 世爻 |
| **代表** | 事物主体 | 占者本人 |
| **应用** | 判事物真假 | 判个人状态 |

### 3. 配合使用

- 世身旺：占者有信心、有能力
- 卦身旺：事情实在、有希望
- 两者皆旺：人强事强，必成
- 世旺卦弱：有能力但事情难
- 世弱卦旺：事情好但能力不足

## 六、实例分析

### 案例：求财

某卦：
- 卦身在辰，临妻财
- 卦身旺相得日生
- 用神妻财与卦身同

**分析**：
- 卦身临财，代表此财实在
- 卦身旺相，财可求得
- 应期在辰月或辰日

**断语**：此财可求，且数额不少。

## 七、学习要点总结

1. **卦身定义**：根据世爻阴阳和位置推算，代表事物主体
2. **取卦身法**：阳爻持世从子起，阴爻持世从午起
3. **主要作用**：判真假、定虚实、看成败、推应期
4. **临六亲**：可判断事情性质
5. **与用神关系**：生合为吉，冲克为凶
6. **配合世身**：综合分析人与事的状态

卦身是六爻断卦的重要辅助工具，熟练掌握可以增加断卦的维度，提高预测准确率。但需注意，卦身是辅助判断，不可替代用神的主体地位。
