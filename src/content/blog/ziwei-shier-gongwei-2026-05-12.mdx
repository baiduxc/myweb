## 十二宫位概述

紫微斗数将人生划分为十二个宫位，每个宫位代表人生的一个特定领域，共同构成完整的命运蓝图。

### 十二宫位一览

| 宫位 | 俗称 | 代表领域 |
|------|------|----------|
| 命宫 | 本命宫 | 自我、性格、格局 |
| 兄弟宫 | 手足宫 | 兄弟姐妹、同辈关系 |
| 夫妻宫 | 配偶宫 | 婚姻、感情、配偶 |
| 子女宫 | 子息宫 | 子女、晚辈、生育 |
| 财帛宫 | 财运宫 | 财富、收入、理财 |
| 疾厄宫 | 健康宫 | 身体、疾病、意外 |
| 迁移宫 | 出外宫 | 出行、在外、社交 |
| 交友宫 | 奴仆宫 | 朋友、下属、人际 |
| 官禄宫 | 事业宫 | 事业、工作、名声 |
| 田宅宫 | 房产宫 | 不动产、家宅、环境 |
| 福德宫 | 精神宫 | 精神、心态、享受 |
| 父母宫 | 长辈宫 | 父母、长辈、上司 |

## 命宫：命运的核心

### 命宫的含义

命宫是十二宫的核心，代表：

> "命宫者，一身之主，统摄十二宫"

| 层面 | 具体内容 |
|------|----------|
| 性格 | 内在性格、行事风格 |
| 格局 | 人生高度、发展潜力 |
| 外貌 | 相貌特征、气质风度 |
| 潜能 | 天赋特长、发展方向 |

### 命宫主星影响

| 主星类型 | 性格特点 |
|----------|----------|
| 紫微系 | 尊贵、领导、稳重 |
| 天机系 | 智慧、机敏、变化 |
| 太阳系 | 热情、光明、外向 |
| 武曲系 | 刚毅、财务、执行 |
| 天同系 | 温和、享受、协调 |
| 廉贞系 | 精明、进取、复杂 |
| 天府系 | 稳重、保守、守成 |
| 太阴系 | 阴柔、内敛、细腻 |
| 贪狼系 | 欲望、变化、多才 |
| 巨门系 | 口才、是非、怀疑 |
| 天相系 | 正派、辅佐、调和 |
| 天梁系 | 老成、清高、助人 |
| 七杀系 | 刚强、果断、突破 |
| 破军系 | 变动、开创、破坏 |

## 宫位关系：三方四正

### 三方四正概念

> "观命先看本宫，次看三方四正"

**三方：** 对宫及左右邻宫

**四正：** 本宫加上三方共四宫

| 命宫 | 三方四正宫位 |
|------|--------------|
| 命宫 | 迁移、福德、官禄 |
| 财帛宫 | 福德、命宫、官禄 |
| 夫妻宫 | 官禄、福德、迁移 |

### 宫位互动效应

| 情况 | 效应 |
|------|------|
| 本宫吉，三方吉 | 好上加好 |
| 本宫吉，三方凶 | 吉凶参半 |
| 本宫凶，三方吉 | 可以弥补 |
| 本宫凶，三方凶 | 雪上加霜 |

## 学习要点总结

1. **宫位本质**：每个宫位代表人生一个特定领域

2. **核心命宫**：命宫统领其他宫位，是分析起点

3. **三方四正**：理解宫位之间的相互影响

4. **动态分析**：结合大限流年看宫位变化

5. **综合判断**：不能只看一个宫位，要整体把握
