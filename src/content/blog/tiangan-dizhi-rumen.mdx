# 天干地支入门：认识十天干与十二地支

## 什么是天干地支

天干地支，简称"干支"，是中国古代用来纪年、纪月、纪日、纪时的符号系统。十天干与十二地支相配，组成六十甲子，循环往复，用以记录时间的流转。

## 十天干

十天干为：**甲、乙、丙、丁、戊、己、庚、辛、壬、癸**。

| 天干 | 五行 | 阴阳 | 意象 |
|------|------|------|------|
| 甲 | 木 | 阳 | 参天大树，刚健正直 |
| 乙 | 木 | 阴 | 花草藤蔓，柔韧灵活 |
| 丙 | 火 | 阳 | 太阳之火，光明磊落 |
| 丁 | 火 | 阴 | 灯烛之火，温暖细腻 |
| 戊 | 土 | 阳 | 高山厚土，沉稳包容 |
| 己 | 土 | 阴 | 田园之土，滋养万物 |
| 庚 | 金 | 阳 | 刀剑之金，刚毅果断 |
| 辛 | 金 | 阴 | 珠宝之金，精致秀气 |
| 壬 | 水 | 阳 | 江河之水，奔放不息 |
| 癸 | 水 | 阴 | 雨露之水，润泽无声 |

## 十二地支

十二地支为：**子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥**。

每个地支藏有天干，且与十二生肖对应。地支不仅是时间的标记，更蕴含着五行之气的消长变化。

## 干支与五行的关系

五行——木、火、土、金、水，是干支体系的核心。天干地支各有五行归属，它们之间的**生克制化**关系，构成了命理分析的基础。

### 五行相生
- 木生火 → 火生土 → 土生金 → 金生水 → 水生木

### 五行相克
- 木克土 → 土克水 → 水克火 → 火克金 → 金克木

## 干支纪时

在四柱八字中，一个人出生的年、月、日、时各用一组干支表示，共四组八个字，故称"八字"。这四组干支之间的相互作用，便是命理分析的根据。

> **学习要点**：天干地支是八字的基石。熟记十天干、十二地支的五行属性和阴阳特征，是深入学习八字的第一步。
