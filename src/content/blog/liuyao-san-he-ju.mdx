# 六爻三合局详解：卦中合局的判断与应用技法

在六爻预测中，三合局是一种重要的卦象结构，能够显著改变卦中五行的力量对比，对预测结果产生重要影响。掌握三合局的判断与应用，是提升六爻预测水平的关键技法。

## 一、三合局的基本概念

### 什么是三合局

三合局是指三个地支按照特定规则组合，形成强大的合局力量。三合局共有四种：

| 三合局 | 组成地支 | 合化五行 | 方位 |
|-------|---------|---------|------|
| 申子辰合水局 | 申、子、辰 | 水 | 北方 |
| 寅午戌合火局 | 寅、午、戌 | 火 | 南方 |
| 巳酉丑合金局 | 巳、酉、丑 | 金 | 西方 |
| 亥卯未合木局 | 亥、卯、未 | 木 | 东方 |

### 三合局的形成原理

三合局的形成遵循"长生、帝旺、墓库"的组合原则：

| 局类型 | 长生支 | 帝旺支 | 墓库支 | 合化结果 |
|-------|-------|-------|-------|---------|
| 水局 | 申（水长生） | 子（水帝旺） | 辰（水墓库） | 水气旺盛 |
| 火局 | 寅（火长生） | 午（火帝旺） | 戌（火墓库） | 火气旺盛 |
| 金局 | 巳（金长生） | 酉（金帝旺） | 丑（金墓库） | 金气旺盛 |
| 木局 | 亥（木长生） | 卯（木帝旺） | 未（木墓库） | 木气旺盛 |

> **《增删卜易》云：** "三合局者，势如破竹，其力最大。"

## 二、三合局的形成条件

### 基本条件

三合局的形成需要满足以下条件：

| 条件 | 具体要求 |
|-----|---------|
| 地支齐全 | 三支必须同时出现 |
| 位置要求 | 可在同爻、动爻、变爻中 |
| 动静要求 | 至少有一支发动 |
| 时令要求 | 需得时令之助 |

### 详细条件分析

**1. 三支俱全**

卦中必须同时出现组成三合局的三个地支，缺一不可。

**2. 动静配合**

三合局的三支中，有以下几种情况：

| 情况 | 是否成局 | 说明 |
|-----|---------|------|
| 三支皆静 | 不成局 | 无动力，不能合化 |
| 一支发动 | 可成局 | 动爻引化成局 |
| 两支发动 | 可成局 | 力量更强 |
| 三支皆动 | 大局 | 力量最强 |

**3. 时令辅助**

三合局的成败与时令密切相关：

| 时令 | 对合局的影响 |
|-----|------------|
| 生助合化五行 | 合局力量增强 |
| 克制合化五行 | 合局力量减弱 |
| 合化五行当令 | 合局力量最强 |

## 三、三合局的判断方法

### 判断步骤

**第一步：找出三支**

在卦中找出可能组成三合局的地支。

**第二步：检查动静**

确认三支中是否有发动之爻。

**第三步：分析时令**

查看时令是否生助合局。

**第四步：判断合化**

综合判断合局是否成功。

### 合局成功的标志

| 标志 | 说明 |
|-----|------|
| 动爻引化 | 发动之爻引动合局 |
| 时令生助 | 月建、日辰生助合局 |
| 无冲破 | 无其他地支冲破合局 |
| 用神有利 | 合局生助用神 |

### 合局失败的标志

| 标志 | 说明 |
|-----|------|
| 三支不齐 | 缺少一个地支 |
| 全静无动 | 三支皆静，不能成局 |
| 时令克制 | 月建、日辰克制合局 |
| 被冲破 | 其他地支冲破合局 |

## 四、三合局在预测中的应用

### 对用神的影响

三合局形成后，会对用神产生重要影响：

| 用神与合局关系 | 影响 |
|--------------|------|
| 合局生用神 | 大吉，用神力量大增 |
| 合局克用神 | 大凶，用神力量大减 |
| 合局助用神 | 吉，用神得力 |
| 用神入合局 | 需看合化结果 |

### 具体占断实例

**占财运：**

若三合财局成功：
- 水局合化克火财：财运受阻
- 火局合化生土财：财运大好
- 金局合化生水财：财运顺遂
- 木局合化克土财：财运不利

**占婚姻：**

若三合局涉及官爻或财爻：
- 合局生官/财：姻缘顺利
- 合局克官/财：姻缘受阻
- 合局与世爻有情：感情和谐
- 合局与世爻相克：感情波折

**占事业：**

若三合局形成：
- 合局生官爻：事业有成
- 合局生父爻（文书）：考试、升迁有利
- 合局克官爻：事业受挫

## 五、三合局的特殊形态

### 半合局

当三合局中只有两支出现时，称为半合局：

| 半合类型 | 组成 | 力量 |
|---------|------|------|
| 长生半合 | 长生支+帝旺支 | 较强 |
| 墓库半合 | 帝旺支+墓库支 | 较弱 |
| 拱合 | 长生支+墓库支 | 最弱 |

> **注意：** 半合局力量远不如三合局，需谨慎判断。

### 合局与变卦

变卦中出现三合局的情况：

| 情况 | 影响 |
|-----|------|
| 本卦合局 | 原有基础，力量稳定 |
| 变卦合局 | 后天形成，变化之象 |
| 本变皆合 | 力量最大，事必成功 |

### 合局与空亡

合局遇空亡的情况：

| 情况 | 判断 |
|-----|------|
| 合局地支入空亡 | 合局无力，待出空方验 |
| 部分地支入空亡 | 合局力量减弱 |
| 空亡地支出空时 | 合局开始发挥作用 |

## 六、三合局与六合的关系

### 六合基础知识

六爻预测中的六合：

| 六合 | 组合 |
|-----|------|
| 子丑合 | 化土 |
| 寅亥合 | 化木 |
| 卯戌合 | 化火 |
| 辰酉合 | 化金 |
| 巳申合 | 化水 |
| 午未合 | 化火（或土） |

### 三合与六合的区别

| 对比项 | 三合局 | 六合 |
|-------|-------|------|
| 组成 | 三支 | 两支 |
| 力量 | 强大 | 较弱 |
| 条件 | 需动爻引化 | 静亦能合 |
| 持久性 | 稳定持久 | 容易被冲破 |

### 三合六合同时出现

当三合与六合同时出现时的处理：

| 情况 | 处理原则 |
|-----|---------|
| 三合六合不冲突 | 各自成立 |
| 地支同时参与三合六合 | 优先三合 |
| 六合被三合冲破 | 六合失效 |

## 七、实例分析

### 案例一：三合财局成功

**占问：** 近期财运如何？

**卦象分析：**
- 卦中申、子、辰三支俱全
- 其中子水发动
- 月建为子，日辰为亥，时令生助水局
- 水局成功合化
- 用神为火财

**判断：** 水局克火财，财运受阻，近期不宜投资。

**结果：** 求测者近期投资失利，验证了预测。

### 案例二：三合局不成

**占问：** 考试能否通过？

**卦象分析：**
- 卦中寅、午出现，缺戌
- 虽有动爻，但三支不齐
- 半合局力量较弱
- 用神官爻受时令克制

**判断：** 三合火局不成，用神无力，考试困难。

**结果：** 考试未通过。

## 学习要点总结

1. **三合局类型：** 申子辰水、寅午戌火、巳酉丑金、亥卯未木
2. **形成条件：** 三支俱全、有动爻、得时令
3. **判断步骤：** 找三支、查动静、析时令、断合化
4. **对用神影响：** 生用神吉、克用神凶
5. **特殊形态：** 半合局力量弱，空亡待出空
6. **与六合关系：** 三合优先于六合，力量更大
