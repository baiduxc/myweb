人生难免遇到纷争，当矛盾无法调和时，诉讼成为解决纠纷的最后手段。六爻预测可以为官司诉讼提供重要参考，帮助当事人判断胜负、选择策略、把握时机。

## 一、诉讼预测的用神取法

### 1. 基本用神

| 用神 | 代表意义 | 吉凶取向 |
|-----|---------|---------|
| 世爻 | 求测者本人 | 旺相为吉 |
| 应爻 | 对方当事人 | 休囚为吉 |
| 官鬼爻 | 官司、法律 | 宜静不宜动 |
| 父母爻 | 文书、证据 | 有力为佳 |
| 妻财爻 | 诉讼费用 | 不宜空亡 |

### 2. 六亲角色

| 六亲 | 诉讼角色 | 吉凶作用 |
|-----|---------|---------|
| 父母 | 证据、判决书 | 有力则证据充分 |
| 兄弟 | 证人、帮手 | 得用则有人相助 |
| 子孙 | 和解、律师 | 克官鬼，利和解 |
| 妻财 | 金钱利益 | 财产诉讼关键 |
| 官鬼 | 官司、法官 | 太旺则官司缠身 |

## 二、胜负判断的方法

### 1. 世应关系

| 情况 | 判断 | 说明 |
|-----|-----|-----|
| 世强应弱 | 我方有利 | 自身有力，对方弱势 |
| 世弱应强 | 我方不利 | 力量对比悬殊 |
| 世克应爻 | 我方能胜 | 主动权在我 |
| 应克世爻 | 对方占优 | 需谨慎应对 |
| 世应相生 | 可能和解 | 双方有退让之意 |
| 世应相冲 | 斗争激烈 | 难以调和 |

### 2. 动爻判断

**官鬼发动**：

| 情况 | 判断 |
|-----|-----|
| 官鬼发动生世 | 官司胜诉，或官方支持 |
| 官鬼发动克世 | 官司败诉，或官方不利 |
| 官鬼空亡 | 官司不成立，或不了了之 |

**子孙发动**：

| 情况 | 判断 |
|-----|-----|
| 子孙发动克官 | 利于和解，官司可息 |
| 子孙发动生世 | 有贵人调解 |
| 子孙空亡 | 无调解可能，诉讼到底 |

## 三、六神判断

| 六神 | 诉讼象义 |
|-----|---------|
| 青龙 | 官司有理，光明正大 |
| 朱雀 | 口舌是非，文书重要 |
| 勾陈 | 拖延日久，难以决断 |
| 螣蛇 | 惊恐怪异，情况复杂 |
| 白虎 | 血光凶灾，后果严重 |
| 玄武 | 暗昧不明，有隐情 |

## 四、诉讼类型的具体判断

### 1. 经济纠纷

**用神**：妻财爻为主

| 情况 | 判断 |
|-----|-----|
| 财爻旺相生世 | 追债成功 |
| 财爻被应爻劫 | 被对方赖账 |
| 财爻入墓 | 钱财难追 |

### 2. 合同纠纷

**用神**：父母爻（合同）、官鬼（官司）

| 情况 | 判断 |
|-----|-----|
| 父母旺相有力 | 合同有效，证据充分 |
| 父母空亡 | 合同无效或丢失 |
| 父母受冲 | 合同有争议 |

## 五、诉讼结果判断

### 1. 胜诉判断

| 要点 | 判断 |
|-----|-----|
| 世旺应衰 | 我方占优 |
| 世克应爻 | 我方能胜 |
| 官鬼生世 | 判决有利 |
| 父母有力 | 证据确凿 |

### 2. 败诉判断

| 要点 | 判断 |
|-----|-----|
| 世衰应旺 | 我方劣势 |
| 应克世爻 | 对方占优 |
| 官鬼克世 | 判决不利 |

### 3. 和解判断

| 要点 | 判断 |
|-----|-----|
| 子孙发动 | 有和解机会 |
| 世应相合 | 双方可达成一致 |
| 官鬼入墓 | 官司可息 |

## 六、诉讼时机

### 1. 宜诉时机

| 条件 | 说明 |
|-----|-----|
| 世爻旺相 | 自身有力，宜主动出击 |
| 应爻休囚 | 对方弱势，趁机起诉 |
| 官鬼生世 | 法理在我，有理可据 |
| 父母有力 | 证据充分，胜诉把握大 |

### 2. 不宜诉时机

| 条件 | 说明 |
|-----|-----|
| 世爻休囚 | 自身无力，不宜硬碰 |
| 官鬼克世 | 法理不利，易败诉 |
| 应爻旺相 | 对方强势，难有胜算 |

## 学习要点

1. 用神准确：世应官鬼为核心
2. 力量对比：世应旺衰定胜负
3. 动爻关键：动爻揭示变化
4. 六神参考：白虎螣蛇需警惕
5. 类型区分：不同纠纷判断有别
6. 时机把握：择时而动
7. 理性建议：预测为辅，法律为主

## 结语

诉讼是解决纠纷的最后手段，六爻预测可以为诉讼决策提供参考。但需谨记，预测只是参考，最终还是要依法维权。

建议：能和解尽量和解，诉讼费时费力；必须诉讼时，做好充分准备，依法维护自身权益。

和为贵——诉讼虽可预测，和解方为上策。
