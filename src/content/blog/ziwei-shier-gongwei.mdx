# 紫微斗数十二宫位详解

紫微斗数以十二宫位构建了一套完整的人生分析框架。十二宫如同一张人生的"地图"，每个宫位代表一个特定的人生领域，星曜落入其中，揭示了该领域的运势与特点。

## 十二宫概览

| 宫位 | 简称 | 代表领域 |
|------|------|----------|
| 命宫 | 命 | 本命、性格、才能 |
| 兄弟宫 | 兄 | 兄弟姐妹、同辈关系 |
| 夫妻宫 | 夫 | 婚姻、配偶、感情 |
| 子女宫 | 子 | 子女、晚辈、生殖 |
| 财帛宫 | 财 | 财富、收入、理财 |
| 疾厄宫 | 疾 | 健康、疾病、身体 |
| 迁移宫 | 迁 | 外出、旅行、社会 |
| 交友宫 | 交 | 朋友、下属、人际 |
| 官禄宫 | 官 | 事业、工作、官位 |
| 田宅宫 | 田 | 房产、家宅、环境 |
| 福德宫 | 福 | 精神、享乐、福气 |
| 父母宫 | 父 | 父母、长辈、遗传 |

---

## 各宫位详解

### 一、命宫

命宫是十二宫之首，代表命主的本命特质。

| 方面 | 含义 |
|------|------|
| 性格 | 先天性格、气质 |
| 才能 | 主要才能与优缺点 |
| 容貌 | 外貌特征 |
| 格局 | 整体人生格局高低 |

命宫为"体"，其他宫为"用"。命宫好坏决定了命主的先天禀赋。

---

### 二、兄弟宫

代表兄弟姐妹的情况，也扩展到同辈、同事关系。

| 方面 | 含义 |
|------|------|
| 兄弟数量 | 有几人、感情如何 |
| 互助程度 | 能否得到手足帮助 |
| 同辈关系 | 与同事、同学的关系 |
| 合作伙伴 | 合伙生意的伙伴 |

---

### 三、夫妻宫

代表婚姻感情的核心宫位。

| 方面 | 含义 |
|------|------|
| 婚姻状况 | 结婚迟早、婚姻质量 |
| 配偶特质 | 配偶的性格、外貌 |
| 感情模式 | 感情中的表现 |
| 桃花运 | 异性缘、外遇可能 |

> 夫妻宫有化忌，感情多波折；有化禄，婚姻顺遂。

---

### 四、子女宫

代表子女情况，也关联生殖能力。

| 方面 | 含义 |
|------|------|
| 子女数量 | 有无子女、几人 |
| 亲子关系 | 与子女的感情 |
| 子女成就 | 子女的发展前景 |
| 晚辈关系 | 与学生、下属晚辈的关系 |
| 生殖健康 | 生育能力、性能力 |

---

### 五、财帛宫

代表财富运势的核心宫位。

| 方面 | 含义 |
|------|------|
| 财运类型 | 正财还是偏财 |
| 赚钱能力 | 理财能力、收入水平 |
| 财富来源 | 财富的获得方式 |
| 消费观念 | 花钱的态度 |

> 财帛宫与命宫、官禄宫合称"三方四正"，为事业财运的核心三角。

---

### 六、疾厄宫

代表身体健康状况。

| 方面 | 含义 |
|------|------|
| 体质强弱 | 先天体质 |
| 易患疾病 | 身体薄弱环节 |
| 意外伤害 | 受伤的可能性 |
| 心理状态 | 心理健康程度 |

> 疾厄宫需结合星曜五行，分析具体脏腑问题。

---

### 七、迁移宫

代表外出、社会活动的运势。

| 方面 | 含义 |
|------|------|
| 外出运 | 离乡发展的运势 |
| 旅游运 | 出行的顺逆 |
| 社会表现 | 在外界的形象 |
| 贵人运 | 外出遇贵人的可能 |

> 迁移宫与命宫对冲，代表在外与在家的对比。

---

### 八、交友宫（奴仆宫）

代表人际关系，尤其是朋友、下属。

| 方面 | 含义 |
|------|------|
| 朋友运 | 朋友的数量质量 |
| 下属运 | 部属的忠诚度 |
| 人际关系 | 社交能力 |
| 小人运 | 是否易遇小人 |

---

### 九、官禄宫（事业宫）

代表事业发展的核心宫位。

| 方面 | 含义 |
|------|------|
| 事业方向 | 适合的行业类型 |
| 工作能力 | 职场表现 |
| 升迁运 | 晋升的可能性 |
| 创业运 | 自行创业的成败 |

> 官禄宫与命宫、财帛宫形成"三方四正"，是判断事业的核心。

---

### 十、田宅宫

代表不动产、家宅环境。

| 方面 | 含义 |
|------|------|
| 房产运 | 有无房产、多少 |
| 居住环境 | 住家的条件 |
| 祖产继承 | 能否继承祖业 |
| 家庭氛围 | 家庭生活状态 |

---

### 十一、福德宫

代表精神世界、享受能力。

| 方面 | 含义 |
|------|------|
| 精神状态 | 内心的满足感 |
| 享受能力 | 懂不懂享受 |
| 兴趣爱好 | 精神追求 |
| 晚年运 | 晚年的生活状态 |

> 福德宫是"精神仓库"，与命宫共同决定幸福程度。

---

### 十二、父母宫

代表父母长辈的情况。

| 方面 | 含义 |
|------|------|
| 父母状况 | 父母的健康、事业 |
| 亲子关系 | 与父母的感情 |
| 遗传因素 | 从父母继承的特质 |
| 上司关系 | 与领导的关系 |

---

## 三方四正

每个宫位都有其"三方四正"：

| 本宫 | 三方 | 对宫 |
|------|------|------|
| 命宫 | 财帛、官禄 | 迁移 |
| 兄弟宫 | 疾厄、田宅 | 交友 |
| 夫妻宫 | 子女、福德 | 官禄 |
| 子女宫 | 夫妻、官禄 | 田宅 |
| 财帛宫 | 命宫、迁移 | 福德 |
| 疾厄宫 | 兄弟、交友 | 父母 |
| 迁移宫 | 财帛、官禄 | 命宫 |
| 交友宫 | 兄弟、疾厄 | 父母 |
| 官禄宫 | 命宫、财帛 | 夫妻 |
| 田宅宫 | 兄弟、子女 | 子女 |
| 福德宫 | 财帛、田宅 | 财帛 |
| 父母宫 | 疾厄、交友 | 疾厄 |

> 分析某宫时，需看本宫及三方四正的星曜综合判断。

---

## 宫位的应用原则

1. **本宫为主**：本宫星曜决定该宫基本特质
2. **三方四正辅助**：其他宫位提供补充信息
3. **四化影响**：化禄、化权、化科、化忌改变宫位吉凶
4. **大运流年**：大运流年宫位叠加本命宫位分析

---

> **学习要点**：十二宫位是紫微斗数的"舞台"，星曜是"演员"。学习时先理解各宫位的基本含义，再结合星曜特性综合分析。记住"三方四正"的概念，这是紫微斗数解盘的核心方法。
