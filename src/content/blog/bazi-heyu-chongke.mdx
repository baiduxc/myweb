天干地支之间的合化、冲克、刑害关系，是八字命局内部互动的基本语法。这些关系决定了命局中各元素如何相互影响，是一切断命分析的基础。

## 天干关系

### 天干五合

| 合化 | 条件 | 化后五行 |
|------|------|----------|
| 甲己合 | 甲己相邻，土旺 | 化土 |
| 乙庚合 | 乙庚相邻，金旺 | 化金 |
| 丙辛合 | 丙辛相邻，水旺 | 化水 |
| 丁壬合 | 丁壬相邻，木旺 | 化木 |
| 戊癸合 | 戊癸相邻，火旺 | 化火 |

> 合而不化的条件：两干虽合，但月令不助化神，或被他干冲破，则合而不化，只论合绊。

### 合的吉凶

| 情况 | 吉凶 |
|------|------|
| 合化为喜用 | 大吉 |
| 合化为忌神 | 大凶 |
| 合绊喜用神 | 凶 |
| 合绊忌神 | 吉 |
| 争合（一干合两干） | 感情纠纷，事多牵绊 |

### 天干相冲

| 对冲组 | 五行关系 |
|--------|----------|
| 甲庚冲 | 金木冲 |
| 乙辛冲 | 金木冲 |
| 丙壬冲 | 水火冲 |
| 丁癸冲 | 水火冲 |

> 天干相冲力量不如地支冲大，但主变动、冲突。

## 地支关系

### 地支六合

| 合化 | 化后五行 |
|------|----------|
| 子丑合 | 化土 |
| 寅亥合 | 化木 |
| 卯戌合 | 化火 |
| 辰酉合 | 化金 |
| 巳申合 | 化水 |
| 午未合 | 化火/土 |

> 六合以合绊为主，化气需条件。合绊则两支都受牵制，不能自由发挥作用。

### 地支三合局

| 三合局 | 组成 | 化后五行 |
|--------|------|----------|
| 申子辰 | 水局长生帝旺墓 | 化水 |
| 亥卯未 | 木局长生帝旺墓 | 化木 |
| 寅午戌 | 火局长生帝旺墓 | 化火 |
| 巳酉丑 | 金局长生帝旺墓 | 化金 |

> 三合局力量大于六合。三合局成则改变命局五行力量对比，影响巨大。

**三合局成局条件**：
1. 三个地支同时出现
2. 或两支出现，大运流年补齐第三支
3. 中间字（子/卯/午/酉）不可空亡

### 地支三会局

| 三会局 | 组成 | 化后五行 |
|--------|------|----------|
| 寅卯辰 | 东方木 | 化木 |
| 巳午未 | 南方火 | 化火 |
| 申酉戌 | 西方金 | 化金 |
| 亥子丑 | 北方水 | 化水 |

> 三会局力量最强，大于三合局。会成则一方五行力量极旺。

### 地支六冲

| 冲组 | 五行关系 | 主事 |
|------|----------|------|
| 子午冲 | 水火冲 | 情绪波动、变动 |
| 丑未冲 | 土土冲 | 土地、房产纠纷 |
| 寅申冲 | 金木冲 | 行走、迁移、冲突 |
| 卯酉冲 | 金木冲 | 破损、分离 |
| 辰戌冲 | 土土冲 | 官司、争斗 |
| 巳亥冲 | 水火冲 | 反复、变动 |

> 冲主动、主变、主散。逢冲则动，不动则伤。

### 地支三刑

| 刑名 | 组成 | 含义 |
|------|------|------|
| 无恩之刑 | 寅巳申 | 忘恩负义、恩将仇报 |
| 恃势之刑 | 丑戌未 | 仗势欺人、官司 |
| 无礼之刑 | 子卯 | 不守礼法、暧昧 |
| 自刑 | 辰辰、午午、酉酉、亥亥 | 自寻烦恼、自残 |

> 刑主刑罚、纠纷、伤灾。但刑不等于凶，需看刑的是用神还是忌神。

### 地支六害

| 害组 | 含义 |
|------|------|
| 子未害 | 骨肉分离 |
| 丑午害 | 官非口舌 |
| 寅巳害 | 是非争斗 |
| 卯辰害 | 朋友背叛 |
| 申亥害 | 交通意外 |
| 酉戌害 | 伤身破财 |

> 害主损害、暗伤。害不如冲剧烈，但多为暗中伤害。

### 地支相破

| 破组 | 含义 |
|------|------|
| 子酉破 | 感情破裂 |
| 丑辰破 | 兄弟不和 |
| 寅亥破 | 事业阻碍 |
| 卯午破 | 夫妻不和 |
| 巳申破 | 朋友背叛 |
| 未戌破 | 合伙纠纷 |

> 破主破坏、分裂，力量较冲刑为轻。

## 合冲刑害的优先级

当命局中多种关系同时存在时，判断优先级：

| 优先级 | 关系 | 原因 |
|--------|------|------|
| 1 | 三会局 | 力量最大 |
| 2 | 三合局 | 力量次大 |
| 3 | 六冲 | 冲则动，优先处理 |
| 4 | 六合 | 合则绊，需看化否 |
| 5 | 三刑 | 刑主纠纷 |
| 6 | 六害 | 害主暗损 |
| 7 | 相破 | 力量最轻 |

> **重要原则**：贪合忘冲。若地支有合有冲，先论合，合住则不冲。但若冲力过大，则冲破合。

## 实战应用

### 命局分析步骤

1. 先看有无三合三会——力量最大，改变全局
2. 再看六冲——冲则动，定变动方向
3. 然后看六合——合则绊，定牵制关系
4. 最后看刑害破——辅助判断细节

### 合化对命局的影响

| 影响类型 | 说明 |
|----------|------|
| 改变五行力量 | 合化后五行改变，需重新论旺衰 |
| 牵制地支 | 合而不化，两支皆被绊住 |
| 引动事件 | 大运流年补齐三合局，引动重大事件 |

---

> **学习要点**：合冲刑害是八字命局内部互动的基本语法。初学时应先记住天干五合、地支六合六冲，再逐步掌握三合三会与刑害。实战中，先抓大的关系（三合三会六冲），再看小关系（六合刑害），由大及小，方不混乱。
