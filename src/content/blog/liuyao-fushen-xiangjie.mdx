# 六爻伏神详解：隐爻显化的断卦技法

六爻预测中，卦象并非总能呈现完整的信息。有时，关键的用神或原神藏而不显，需要通过伏神的方法来寻找。本文详细讲解伏神的原理与应用。

## 一、什么是伏神

伏神，又称隐爻，是指在本宫八卦中没有出现，而隐藏在本宫其他卦中的爻。当卦中缺少某个六亲时，就需要找伏神。

> 《增删卜易》云："卦无是用神，看用神伏于何爻之下。"

### 伏神的本质

伏神是：
- 被压在明爻之下的隐性能量
- 暂时未显化但实际存在的因素
- 需要特定条件才能发挥作用的力量

## 二、伏神的查找方法

### 1. 确定本宫所属

首先需明确卦的本宫：

| 卦名 | 本宫 | 卦名 | 本宫 |
|-----|------|-----|------|
| 乾为天 | 乾宫 | 坤为地 | 坤宫 |
| 兑为泽 | 兑宫 | 艮为山 | 艮宫 |
| 离为火 | 离宫 | 坎为水 | 坎宫 |
| 震为雷 | 震宫 | 巽为风 | 巽宫 |

### 2. 查找伏神位置

**口诀**：
> "一二三六为内卦，四五为神记心中。"

具体方法：
1. 看卦中缺什么六亲
2. 在本宫八卦中找到该六亲所在的卦
3. 确定其所伏的爻位

**示例**：

假设占到`天风姤`卦（乾宫）：
- 上卦为乾，下卦为巽
- 卦中只有父母、兄弟、子孙
- 缺妻财和官鬼
- 查乾宫八卦，妻财在乾为天卦初爻（甲子），官鬼在乾为天卦三爻（甲辰）
- 故妻财伏于初爻之下，官鬼伏于三爻之下

## 三、伏神的取用原则

### 1. 用神上卦优先

> 原则：有用神不用伏神，无才寻伏。

- 若用神本卦中有，直接取用
- 只有本卦无时才考虑伏神

### 2. 伏神可用条件

伏神能否发挥作用，需看：

| 条件 | 说明 |
|-----|------|
| **有气** | 伏神得日月旺相 |
| **得生** | 被飞神生合 |
| **冲开** | 日月冲开飞神 |
| **出透** | 伏神透出之时 |

### 3. 伏神难用情况

以下情况伏神难用：
- 伏神休囚无气
- 被飞神克制
- 空亡且无救应
- 月破且无解救

## 四、飞神与伏神的关系

### 飞神

飞神是指伏神所藏之处的明爻。

**关系类型**：

| 飞伏关系 | 影响 |
|---------|------|
| 飞生伏 | 伏神容易出头，力量增强 |
| 飞克伏 | 伏神受压，难以出头 |
| 伏生飞 | 伏神泄气，力量减弱 |
| 伏克飞 | 伏神虽弱，但不服飞神 |
| 飞伏比和 | 互相帮助，力量均衡 |

> 《增删卜易》："飞来生伏得长生，伏去生飞为泄气。飞来克伏反伤身，伏去克飞为出暴。"

## 五、伏神的实际应用

### 案例1：求财卦无财

占得某卦，用神妻财本卦无：
1. 查找伏神位置
2. 看飞伏关系
3. 判断伏神是否有气
4. 若飞神生伏，伏神有气，待出透时可得财

### 案例2：占病无官鬼

官鬼为病神，卦无官鬼：
1. 寻伏神官鬼
2. 看飞伏关系
3. 若伏神受克，疾病较轻
4. 若伏神得生，疾病沉疴

## 六、伏神出透的时机

伏神要在特定时机才能发挥作用：

### 出透方式：

1. **日辰冲开**：日辰冲动飞神，伏神浮现
2. **月建冲开**：月建冲动飞神
3. **动爻冲开**：卦中有爻发动冲开飞神
4. **时令到临**：伏神值令之月日

### 判断口诀：

> "伏居空亡事恐迟，伏临月建事如期。
> 日辰若也来冲克，伏神出透自无疑。"

## 七、学习要点总结

1. **伏神定义**：本宫八卦中隐伏的六亲，卦中缺少时使用
2. **查找方法**：先定本宫，再查所缺六亲位置
3. **取用原则**：用神优先上卦，无时方用伏神
4. **飞伏关系**：生、克、比和各有不同影响
5. **出透条件**：日月冲开、动爻冲开、时令到临
6. **实际应用**：需综合判断飞伏关系和伏神状态，不可执一

掌握伏神技法，能更全面地解读卦象信息，提高断卦准确性。但需多加实践，方能运用自如。
