## 八字格局概述

八字格局是四柱命理学的核心概念，它决定了一个人命局的层次高低和基本特征。

<Note title="格局的意义">
格局好比一个人的"命格蓝图"，决定了命主的基本发展方向和人生层次。相同的五行组合，若格局不同，命运可能天壤之别。
</Note>

## 格局的基本分类

### 正格与外格

| 类别 | 含义 | 主要格局 |
|------|------|----------|
| 正格 | 日主与月令配合形成的格局 | 正官格、偏官格、正印格、偏印格、食神格、伤官格、正财格、偏财格 |
| 外格 | 特殊组合形成的格局 | 从格、化气格、建禄格、羊刃格 |

### 正格十神格局

#### 一、官杀格局

**正官格**

| 要素 | 说明 |
|------|------|
| 成格条件 | 月令正官透干或本气正官当令 |
| 正官含义 | 代表名誉、地位、官职、丈夫（女命） |
| 喜忌 | 喜印星护官、财星生官；忌伤官克官 |
| 命主特征 | 正直守信、循规蹈矩、有责任感 |

> "正官格最贵，必要透出天干，方为真格。"——《子平真诠》

**偏官格（七杀格）**

| 要素 | 说明 |
|------|------|
| 成格条件 | 月令七杀透干或本气七杀当令 |
| 七杀含义 | 代表权力、魄力、压力、偏夫（女命） |
| 喜忌 | 喜食神制杀、印星化杀；忌无制无化 |

<Tip title="官杀格局要点">
官杀格最讲究制化。正官需保护，七杀需制约。无制化的七杀容易导致灾祸，有制化的七杀则能成就大业。
</Tip>

#### 二、印绶格局

**正印格**

| 要素 | 说明 |
|------|------|
| 成格条件 | 月令正印透干或本气正印当令 |
| 正印含义 | 代表学业、名誉、母亲、贵人 |
| 喜忌 | 喜官星生印、比劫护印；忌财星坏印 |

#### 三、财星格局

**正财格**

| 要素 | 说明 |
|------|------|
| 成格条件 | 月令正财透干或本气正财当令 |
| 正财含义 | 代表正当收入、薪水、妻子（男命） |
| 喜忌 | 喜身旺担财、官星护财；忌比劫夺财 |

<Warning title="伤官格注意">
伤官见官，为祸百端。伤官格最怕见正官，容易招惹是非官灾。但伤官生财、伤官佩印也是好格局。
</Warning>

## 格局的判定方法

### 步骤一：确定日主

日主是八字的中心，代表命主本人。日主即日柱天干。

### 步骤二：分析月令

月令是格局判定最重要的依据：

1. 查看月支本气
2. 检查月令藏干透出情况
3. 确定格神

### 步骤三：判断日主强弱

| 日主状态 | 判断标准 | 格局取向 |
|----------|----------|----------|
| 身旺 | 得令、得地、得势 | 取克泄耗为用 |
| 身弱 | 失令、失地、失势 | 取生扶为用 |

## 格局高低论断

### 高格特征

1. **格清**：格局纯粹，不杂乱
2. **局正**：气势正宗，配合得当
3. **有情**：日主与格神有情
4. **有力**：格神有根有气

| 格局层次 | 特点 | 人生表现 |
|----------|------|----------|
| 上格 | 格清局正，用神有力 | 富贵双全，事业有成 |
| 中格 | 格局中等，用神有情 | 衣食无忧，有成就 |
| 下格 | 格局破败，用神无力 | 贫穷困顿，多灾多难 |

## 学习要点总结

1. **格局分为正格和外格**，正格以月令十神为基础
2. **格局判定以月令为核心**，需综合考虑日主强弱
3. **用神是格局的关键**，决定了格局层次
4. **格局高低看格神的清浊、正偏、有情无力**
