紫微斗数的命盘由十二宫位组成，每个宫位代表人生的一个特定领域。理解宫位的含义及其相互关系，是学习紫微斗数的基础。

## 一、十二宫位概述

### 宫位体系

| 宫位 | 别名 | 代表领域 |
|-----|-----|---------|
| 命宫 | - | 自我、性格、命运基调 |
| 兄弟宫 | 手足宫 | 兄弟姐妹、合作关系 |
| 夫妻宫 | 配偶宫 | 婚姻、感情、配偶特质 |
| 子女宫 | 男女宫 | 子女、晚辈、下属 |
| 财帛宫 | 财富宫 | 财运、收入来源 |
| 疾厄宫 | 健康宫 | 身体、疾病、灾厄 |
| 迁移宫 | 出外宫 | 外出、旅游、搬迁 |
| 交友宫 | 奴仆宫 | 朋友、下属、人际 |
| 官禄宫 | 事业宫 | 事业、工作、成就 |
| 田宅宫 | 不动产宫 | 房产、土地、家宅 |
| 福德宫 | 精神宫 | 心理、精神、嗜好 |
| 父母宫 | 相貌宫 | 父母、长辈、相貌 |

## 二、各宫位详解

### 1. 命宫

**核心含义**：代表一个人的本命，是整个命盘的核心。

| 分析维度 | 说明 |
|---------|-----|
| 性格特质 | 内在性格与外在表现 |
| 先天禀赋 | 与生俱来的能力与局限 |
| 命运基调 | 一生运势的大方向 |
| 格局高低 | 整体命盘的层次 |

### 2. 兄弟宫

**核心含义**：代表兄弟姐妹及平辈关系。

| 分析维度 | 说明 |
|---------|-----|
| 手足情缘 | 与兄弟姐妹的关系 |
| 合作关系 | 合伙、团队协作 |
| 人际关系 | 平辈之间的相处 |

### 3. 夫妻宫

**核心含义**：代表婚姻感情及配偶情况。

| 分析维度 | 说明 |
|---------|-----|
| 配偶特质 | 另一半的性格特征 |
| 感情运势 | 感情的顺利程度 |
| 婚姻质量 | 婚姻生活的状态 |

### 4. 子女宫

**核心含义**：代表子女及晚辈情况。

| 分析维度 | 说明 |
|---------|-----|
| 子女数量 | 子女的多寡 |
| 子女特质 | 子女的能力与性格 |
| 亲子关系 | 与子女的互动方式 |
| 下属情况 | 部属、晚辈的关系 |

### 5. 财帛宫

**核心含义**：代表财运与收入。

| 分析维度 | 说明 |
|---------|-----|
| 进财方式 | 收入来源与方式 |
| 理财能力 | 金钱管理的能力 |
| 财富积累 | 存钱致富的可能性 |

### 6. 疾厄宫

**核心含义**：代表健康与疾病。

| 分析维度 | 说明 |
|---------|-----|
| 身体强弱 | 整体健康状况 |
| 易患疾病 | 需要注意的病症 |
| 意外灾厄 | 可能遭遇的灾祸 |

### 7. 迁移宫

**核心含义**：代表外出运与活动力。

| 分析维度 | 说明 |
|---------|-----|
| 外出运势 | 出门在外的运气 |
| 活动能力 | 行动力与适应性 |
| 旅行搬迁 | 迁徙变动的机会 |

### 8. 交友宫

**核心含义**：代表人际关系与下属。

| 分析维度 | 说明 |
|---------|-----|
| 朋友运 | 朋友的数量与质量 |

| 人际关系 | 社交活动的状况 |

### 9. 官禄宫

**核心含义**：代表事业与工作。

| 分析维度 | 说明 |
|---------|-----|
| 事业运势 | 工作发展的前途 |
| 适合职业 | 有利的工作领域 |

### 10. 田宅宫

**核心含义**：代表不动产与居住。

| 分析维度 | 说明 |
|---------|-----|
| 房产运势 | 拥有不动产的机会 |
| 居住环境 | 居家的品质 |
| 祖业遗产 | 承继祖产的可能 |

### 11. 福德宫

**核心含义**：代表精神层面与享受。

| 分析维度 | 说明 |
|---------|-----|
| 精神状态 | 内心的快乐程度 |
| 嗜好兴趣 | 个人的兴趣爱好 |
| 福报深浅 | 享受的多少 |

### 12. 父母宫

**核心含义**：代表父母与长辈。

| 分析维度 | 说明 |
|---------|-----|
| 父母关系 | 与父母的互动方式 |
| 长辈缘分 | 贵人、长辈的帮助 |
| 相貌特征 | 外表形象的参考 |

## 三、宫位关系

| 关系 | 说明 |
|-----|-----|
| 三方 | 本宫、对宫及两个三合宫 |
| 四正 | 本宫与对宫相互影响 |

## 结语

十二宫位构成了紫微斗数的框架，理解宫位含义是学习紫微斗数的基础。每个宫位都不是孤立的，它们相互影响、相互制约，共同构成了人生的全貌。

宫位有定，人生无定——紫微斗数的智慧在于理解规律，而非宿命论断。
