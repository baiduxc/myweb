大运与流年是八字预测中动态分析运势的核心工具。命局如车，大运如路，流年如天气。命局定格局高低，大运定运势起伏，流年定吉凶应期。

## 大运排法

### 起运数计算

**阳年男命、阴年女命**——顺排大运

**阴年男命、阳年女命**——逆排大运

> 阳年：甲丙戊庚壬年；阴年：乙丁己辛癸年。

### 起运年龄计算

从出生日向前（逆排）或向后（顺排）数到最近的节气：

- 每三天折为一年
- 每一天折为四个月
- 每个时辰折为十天

> 顺排者从出生日数到下一个节气，逆排者数到上一个节气。

### 大运排布

以月干支为基准：

- **顺排**：从月柱下一个干支开始顺行
- **逆排**：从月柱上一个干支开始逆行

**示例**：甲寅月柱，顺排大运为乙卯、丙辰、丁巳、戊午……

**示例**：甲寅月柱，逆排大运为癸丑、壬子、辛亥、庚戌……

### 大运周期

每步大运管十年：

| 运序 | 年龄段 | 干支 |
|------|--------|------|
| 第一步 | 起运岁～+9 | 第1柱 |
| 第二步 | +10～+19 | 第2柱 |
| 第三步 | +20～+29 | 第3柱 |
| …… | …… | …… |

> 天干管前五年，地支管后五年，但需整体看。

## 流年推算

### 流年确定

流年即当年的干支，以立春为界：

> 立春前为上一年干支，立春后为本年干支。

### 流年与命局的关系

流年与命局发生以下作用：

| 作用类型 | 说明 | 效力 |
|----------|------|------|
| 流年与日干 | 天干与日干论十神 | 直接 |
| 流年与大运 | 流年与运干支生克 | 重要 |
| 流年与命局 | 流年与四柱天干地支的关系 | 基础 |

### 流年作用层次

> 命局为静，大运为动，流年为触。流年先触大运，再及命局。

作用优先级：

1. **流年冲克大运**——当年的突发变化
2. **流年合化命局**——当年的人际合作
3. **流年与命局刑冲**——当年的冲突事件

## 大运断法

### 看大运的总体方向

| 判断维度 | 内容 |
|----------|------|
| 运干十神 | 十年运势的主导力量 |
| 运支藏干 | 十年运势的内在因素 |
| 运与命局生克 | 大运对命局的帮扶或抑制 |
| 运与用神关系 | 用神得力则吉，用神受制则凶 |

### 大运吉凶判断

| 情况 | 吉凶 |
|------|------|
| 大运生扶用神 | 吉 |
| 大运克泄用神 | 凶 |
| 大运克制忌神 | 吉 |
| 大运生助忌神 | 凶 |
| 大运与命局天合地合 | 化吉则吉，化凶则凶 |
| 大运与命局天克地冲 | 动荡变化之年 |

## 流年断法

### 流年十神断事

| 流年十神 | 可能发生之事 |
|----------|-------------|
| 比肩 | 兄弟朋友、竞争、投资 |
| 劫财 | 争夺、破财、合作失败 |
| 食神 | 饮食、才华、子女 |
| 伤官 | 变动、口舌、创意 |
| 偏财 | 意外之财、父事、投资 |
| 正财 | 工资收入、婚姻、务实 |
| 七杀 | 压力、官非、意外 |
| 正官 | 升迁、考试、约束 |
| 偏印 | 学业、偏门、孤寂 |
| 正印 | 文书、母亲、庇护 |

### 流年与四柱的作用

| 作用位置 | 影响范围 |
|----------|----------|
| 流年与年柱 | 祖上、长辈、远方 |
| 流年与月柱 | 父母、工作、事业 |
| 流年与日柱 | 自己、配偶、身体 |
| 流年与时柱 | 子女、晚辈、投资 |

### 流年应期判断

> 应期是八字预测的难点，即判断事情在何年何月发生。

常用方法：

1. **流年引动命局**——命局中有合待冲、冲待合的组合，流年来触发
2. **流年填实**——命局中有空亡，流年来填实
3. **流年天干主事**——流年天干决定事件性质
4. **流年地支主方**——流年地支决定事件方位

## 大运流年综合分析

### 分析框架

1. **先定命局格局用忌**
2. **看大运对用忌的帮扶或抑制**
3. **看流年与大运的配合**
4. **找出关键的吉凶流年**

### 实例分析思路

假设某命局日主偏弱，以印比为用：

| 大运 | 运干支 | 与用神关系 | 总体评价 |
|------|--------|------------|----------|
| 甲寅 | 比肩+比肩 | 大力帮身 | 佳运 |
| 乙卯 | 劫财+比肩 | 帮身但有争 | 尚可 |
| 丙辰 | 食神+偏印 | 食神泄身，辰土晦火 | 平运 |
| 丁巳 | 伤官+偏财 | 伤官泄身生财 | 较差 |
| 戊午 | 偏财+正财 | 财旺身弱难担 | 差运 |

> 关键：身弱之人逢帮身之运则兴，逢泄耗之运则衰。

## 注意事项

1. **大运重地支**——地支为根基，天干为表象
2. **流年重天干**——天干决定事件性质
3. **先命后运再年**——命局是根本，大运是环境，流年是触发
4. **运年不可割裂**——大运流年需结合分析
5. **注意岁运并临**——大运流年干支相同，力量倍增
