# 奇门遁甲用神与断卦法：精准预测的核心技法

奇门遁甲排盘只是基础，用神选取与断卦才是关键。用神取错，断卦必谬；用神取准，断卦方灵。本文将系统讲解奇门遁甲中用神的选取与断卦的核心技法。

## 一、用神的基本概念

### 1.1 什么是用神

用神，即在一局奇门遁甲中代表所测事物或人的符号。选取正确的用神是断卦的前提：

> **用神是奇门遁甲断卦的灵魂，取准用神等于成功了一半**

用神的选取原则：
- 一个事物对应一个用神
- 用神取最直接、最明显的符号
- 必要时可取多个用神综合分析

### 1.2 用神的分类

| 分类 | 说明 | 示例 |
|------|------|------|
| 日干用神 | 以日干代表求测者本人 | 甲日生人以甲为用神 |
| 时干用神 | 以时干代表所测之事 | 时干临宫断事情发展 |
| 年命用神 | 以出生年干代表本人 | 甲子年生人以甲为用神 |
| 专有用神 | 以特定符号代表特定事物 | 开门代表事业、生门代表财运 |

## 三、断卦的基本步骤

### 3.1 五步断卦法

**第一步：审局**

审视整体格局，判断此局的总体层次：

| 审局内容 | 要点 |
|----------|------|
| 值符值使 | 值符所临之宫为全局关键 |
| 吉凶格局 | 是否有吉格凶格 |
| 马星空亡 | 是否逢马星或空亡 |
| 伏吟反吟 | 是否为伏吟或反吟局 |

**第二步：取用神**

根据所测之事，选取对应的用神符号。

**第三步：看落宫**

分析用神落在何宫，该宫的星、门、神如何：

| 分析要素 | 内容 |
|----------|------|
| 用神落宫 | 在何宫位 |
| 天盘星 | 何星临此宫 |
| 人盘门 | 何门临此宫 |
| 神盘神 | 何神临此宫 |
| 地盘干 | 地盘天干是什么 |
| 天盘干 | 天盘天干是什么 |

**第四步：判吉凶**

综合分析用神落宫的各种因素，判断吉凶：

| 吉凶等级 | 条件 |
|----------|------|
| 大吉 | 吉格+吉星+吉门+旺相 |
| 吉 | 吉格+吉星或吉门 |
| 平 | 无吉无凶 |
| 凶 | 凶格+凶星或凶门 |
| 大凶 | 凶格+凶星+凶门+休囚 |

**第五步：定时应**

预测事情发生的时间：

| 方法 | 说明 |
|------|------|
| 旺相断近期 | 用神旺相，事情快则应 |
| 休囚断远期 | 用神休囚，事情慢则应 |
| 冲合断期 | 冲则散，合则成 |
| 马星断期 | 马星动则有变 |

### 3.2 断卦的注意事项

1. **不可主观臆断**：要严格依据用神落宫分析
2. **不可断一件不论全局**：需全面审视
3. **不可执着于吉凶**：吉中有凶，凶中有吉
4. **不可一概而论**：需结合求测者的实际情况
5. **不可轻断生死**：涉及重大事项需慎重

## 六、断卦实战案例

### 案例：求测事业

**时间**：2024年甲辰年四月辛日丙时
**排盘**：阳遁三局

**断卦过程**：

1. **取用神**：日干辛为求测者，开门为事业
2. **看日干**：辛落在离九宫，天盘辛+地盘丙→"丙辛合"，为合局
3. **看开门**：开门落在坤二宫，临天任星+太阴
4. **关系判断**：离九宫生坤二宫（火生土），日干生开门
5. **结论**：求测者目前对事业很上心，愿意付出，但尚未得到事业回报。天任星临开门，事业需勤恳踏实。太阴临之，有贵人暗中帮助。
6. **时应**：开门在坤宫旺相，近期可有望事业好转。

## 学习要点

1. **用神是断卦的灵魂**，取准用神等于成功一半
2. **五步断卦法**（审局→取用神→看落宫→判吉凶→定时应）是基本流程
3. **专项断卦**各有特定用神，需熟记用神表
4. **格局配合断卦**可提高准确率——吉格加吉、凶格加凶
5. **断卦需全面**，不可执着于一点
6. **定时应是难点**，需结合旺衰、冲合、马星等综合判断
7. 实战断卦需**勤加练习**，从简单案例入手
8. 断卦的最终目的是**趋吉避凶、指导行动**
