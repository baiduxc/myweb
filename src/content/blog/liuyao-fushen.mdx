## 引言

在六爻预测中，有时卦象中的六亲（父母、兄弟、妻财、子孙、官鬼）不全，缺少某一亲，这时就需要从本宫中查找"伏神"。伏神如同隐藏的能量，平时不显，但在特定条件下会发挥作用。理解伏神的含义与应用，是六爻预测进阶的重要内容。本文将系统介绍伏神的查法、作用与应用方法。

## 伏神的基本概念

### 什么是伏神

伏神是指卦中缺少的六亲，需要从本卦所属的宫中查找：

| 概念 | 解释 |
|-----|------|
| 本宫 | 八卦中每一卦所属的宫，如乾宫、坎宫等 |
| 飞神 | 卦中已有的六亲爻，伏神藏于飞神之下 |
| 伏神 | 本宫中对应缺少六亲的爻，隐藏在飞神之下 |

> 《增删卜易》云："卦有六亲，或缺一亲，则从本宫查之，此名伏神，藏于飞神之下。"

### 六亲不全的情况

六亲不全时需查伏神：

```
六亲查法举例：
例：乾为天卦（乾宫）
- 初爻：子水（父母）
- 二爻：寅木（兄弟）
- 三爻：辰土（妻财）
- 四爻：午火（官鬼）
- 五爻：申金（子孙）
- 六爻：戌土（妻财）

若卦中缺少某亲，如缺少子孙爻，
则从乾宫八卦中查找子孙爻的位置作为伏神。
```

## 伏神的查法

### 八宫归属

伏神需从本宫查找，八宫归属如下：

| 八卦 | 所属宫 | 实性质 |
|-----|-------|-------|
| 乾为天 | 乾宫 | 金 |
- 坤为地 | 坤宫 | 土 |
| 坎为水 | 坎宫 | 水 |
| 离为火 | 离宫 | 火 |
| 震为雷 | 震宫 | 木 |
| 巽为风 | 巽宫 | 木 |
| 艮为山 | 艮宫 | 土 |
| 兑为泽 | 兑宫 | 金 |

### 伏神查法步骤

查找伏神的具体步骤：

```
步骤一：确定卦象所属宫位
步骤二：观察卦中六亲是否齐全
步骤三：若缺少某亲，从本宫八卦中查找对应爻
步骤四：确定伏神位置，记录伏神属性
步骤五：分析伏神与飞神的关系
```

### 伏神查法实例

伏神查法的具体实例：

| 情况 | 卦象 | 缺少六亲 | 伏神位置 |
|-----|-----|---------|---------|
| 缺父母 | 坎卦某爻卦 | 父母爻 | 从坎宫查父母爻为伏神 |
| 缺妻财 | 震卦某爻卦 | 妻财爻 | 从震宫查妻财爻为伏神 |
| 缺官鬼 | 巽卦某爻卦 | 官鬼爻 | 从巽宫查官鬼爻为伏神 |

> 《卜易正宗》："凡卦缺亲，必查本宫，得其伏神。伏神虽不现于卦中，然有事则出，无事则伏。"

## 伏神的作用机制

### 伏神与飞神的关系

伏神藏于飞神之下，二者关系决定伏神能否发挥作用：

| 关系类型 | 含义 | 伏神能否出伏 |
|---------|------|-------------|
| 飞神生伏神 | 飞神五行生伏神五行 | 易出伏，有助力 |
| 飞神克伏神 | 飞神五行克伏神五行 | 难出伏，受压制 |
| 伏神生飞神 | 伏神五行生飞神五行 | 被泄气，作用减弱 |
| 伏神克飞神 | 伏神五行克飞神五行 | 有冲克，关系复杂 |

### 伏神出伏的条件

伏神在何种条件下可以发挥作用：

```
伏神出伏有利条件：
✓ 飞神生伏神（有助力）
✓ 日月建生助伏神（外力支持）
✓ 伏神逢动爻冲开飞神（冲破压制）
✓ 飞神旬空（压制解除）
✓ 飞神被日月克破（压制失效）

伏神出伏不利条件：
✗ 飞神克伏神（压制强）
✗ 伏神休囚无力（自身弱）
✗ 无外力生助（孤立无援）
✗ 飞神旺相有力（压制持续）
```

## 伏神在预测中的应用

### 求财预测中的伏神

求财预测中妻财伏神的应用：

| 情况 | 判断要点 | 结论 |
|-----|---------|------|------|
| 卦无妻财爻，伏神为妻财 | 查伏神能否出伏 | 出伏则有财，不出则无财 |
| 妻财伏神受飞神克 | 求财困难 | 需等时机，或需努力争取 |
| 妻财伏神受飞神生 | 求财顺利 | 财运可得，时机有利 |
| 妻财伏神逢日月助 |财运来临 | 该日该月有财运 |

> 《六爻精义》："求财若无财爻，查伏神财爻。伏神出伏则得财，不出伏则不得财。出伏时机看日月动爻。"

### 求官预测中的伏神

求官预测中官鬼伏神的应用：

```
官鬼伏神判断：
✓ 官鬼伏神出伏：职位可得，时机将至
✓ 官鬼伏神受压制：职位难得，阻力重重
✓ 官鬼伏神逢贵人助：有贵人推荐，职位有望
✗ 官鬼伏神不出伏：职位无缘，另寻他途
```

### 求婚姻预测中的伏神

婚姻预测中的伏神应用：

| 婚姻相关伏神 | 判断要点 | 婚姻影响 |
|-------------|---------|---------|
| 妻财伏神（男测婚） | 是否出伏，旺衰如何 | 妻财出伏旺相，婚姻可得 |
| 官鬼伏神（女测婚） | 是否出伏，旺衰如何 | 官鬼出伏旺相，婚姻可得 |
| 子孙伏神 | 子息缘分 | 子孙出伏，子女缘分有 |

### 健康预测中的伏神

健康预测中官鬼伏神的应用：

- **官鬼为病**：官鬼伏神代表潜在疾病
- **子孙为药**：子孙伏神代表治疗效果
- **伏神出伏**：病发或有治疗方法出现

## 伏神的特殊应用

### 伏神代表隐藏事物

伏神可代表隐藏、隐秘的事物：

| 伏神类型 | 隐藏含义 | 应用场景 |
|---------|---------|---------|
| 妻财伏神 | 隐藏的财源、暗中收入 | 查额外收入、隐秘财务 |
| 官鬼伏神 | 隐藏的职位、暗中安排 | 查暗箱操作、幕后安排 |
| 子孙伏神 | 隐藏的解决方案、潜在途径 | 查备选方案、潜在机会 |
| 父母伏神 | 隐藏的信息、幕后支持 | 查隐秘信息、背后助力 |

### 伏神与旬空

伏神与旬空的关系：

> 《增删卜易》："飞神旬空，则伏神出伏；伏神旬空，则虽出伏亦无力。"

飞神旬空与伏神旬空的判断：

```
飞神旬空情况：
- 飞神旬空，压制解除
- 伏神可出伏发挥作用
- 但需观察伏神自身状态

伏神旬空情况：
- 伏神旬空，虽出伏亦无力
- 需待出空之时方能作用
- 旬空期间暂时无效
```

## 伏神实战案例分析

### 案例一：求财卦中妻财伏神

某人测财运，卦中无妻财爻：

```
卦象分析：
- 卦为震宫某卦，缺少妻财爻
- 从震宫查妻财伏神为卯木
- 飞神为辰土，辰土克卯木
- 飞神克伏神，妻财受压制

判断结论：
- 求财困难，财源受阻碍
- 需等辰土受克或卯木逢助之时
- 暂时不宜投资，待时机成熟

后续验证：
- 当月辰土被日支酉冲破
- 妻财伏神出伏，求财有得
```

### 案例二：求职卦中官鬼伏神

某人测求职，卦中无官鬼爻：

| 分析要点 | 具体情况 | 判断 |
|---------|---------|------|------|
| 伏神查法 | 从本宫查官鬼伏神 | 官鬼伏神为午火 |
| 飞神状态 | 飞神为巳火，巳火生午火 | 飞神生伏神 |
| 日月影响 | 日建为寅木，生助午火 | 日建生伏神 |
| 综合判断 | 伏神有助力，可出伏 | 职位可得 |

> 《六爻实战录》："官鬼伏神若逢飞神生助，又有日月外力，则出伏有力，职位必得。"

## 学习要点总结

1. **伏神概念**：卦中缺少六亲时，从本宫查找的隐藏爻为伏神，藏于飞神之下
2. **伏神查法**：确定卦所属宫位→观察六亲是否齐全→从本宫查伏神→分析伏神飞神关系
3. **出伏条件**：飞神生伏神、日月助伏神、飞神旬空等条件下伏神可出伏
4. **应用场景**：求财看妻财伏神，求官看官鬼伏神，婚姻看相应伏神
5. **隐藏含义**：伏神可代表隐藏的事物，如隐秘收入、暗中安排、潜在方案
6. **实战要点**：综合分析伏神、飞神、日月建的关系，判断伏神能否出伏发挥作用
