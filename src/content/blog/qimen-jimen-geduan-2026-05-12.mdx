## 奇门遁甲概述

奇门遁甲是中国古代最高层次的预测学，被誉为"帝王之学"，通过时空布局来预测和改善命运。

### 奇门遁甲的基本构成

| 构成 | 内容 | 作用 |
|------|------|------|
| 奇 | 乙、丙、丁三奇 | 辅助力量 |
| 门 | 八门 | 行动方式 |
| 仪 | 六仪 | 能量载体 |
| 神 | 八神 | 精神力量 |

### 八门简介

| 门 | 五行 | 吉凶 | 代表 |
|----|------|------|------|
| 开门 | 金 | 吉 | 开创、公开、事业 |
| 休门 | 水 | 吉 | 休养、婚姻、贵人 |
| 生门 | 土 | 吉 | 生财、生育、发展 |
| 伤门 | 木 | 凶 | 伤害、竞争、变动 |
| 杜门 | 木 | 平 | 阻塞、隐藏、技术 |
| 景门 | 火 | 平 | 名誉、文书、计划 |
| 死门 | 土 | 凶 | 死亡、终结、失败 |
| 惊门 | 金 | 凶 | 惊恐、口舌、官司 |

## 择时应用

### 择时原则

| 原则 | 说明 |
|------|------|
| 避凶趋吉 | 避开凶时，选择吉时 |
| 格局优先 | 吉门吉星吉格最佳 |
| 用事相配 | 不同事情用不同格局 |
| 人时相合 | 时辰要有利于当事人 |

### 常用择时表

| 事宜 | 宜用格局 | 忌用格局 |
|------|----------|----------|
| 开业 | 生门+三奇 | 死门+庚 |
| 求职 | 开门+丁奇 | 惊门+癸 |
| 签约 | 休门+乙奇 | 伤门+辛 |
| 出行 | 休门+值符 | 死门+马星 |
| 投资 | 生门+戊 | 伤门+己 |

## 学习要点总结

1. **理解八门**：掌握八门的吉凶属性和代表意义

2. **熟练格局**：牢记主要吉凶格局的条件和效应

3. **用事判断**：不同事情看不同的要素

4. **综合分析**：门、星、神、奇综合判断

5. **择时应用**：学会选择最佳时机办事
