# 周易变卦与互卦：卦象变化的深层逻辑

周易的核心在于"变"。一卦并非静止不变，而是蕴含着变化的可能性。变卦与互卦正是揭示这种变化规律的重要技法，是断卦时不可或缺的分析工具。

## 一、变卦的基本概念

### 1. 何谓变卦

**变卦**，又称"之卦"，是指本卦中的某一爻或几爻发生变化（阳变阴或阴变阳）后所得到的新卦。本卦代表事物的现状，变卦代表事物的趋向或结果。

| 概念 | 含义 | 象征 |
|------|------|------|
| 本卦 | 起卦所得的原卦 | 事物的起始状态 |
| 变卦 | 本卦变爻后所得之卦 | 事物的发展趋向 |
| 变爻 | 发生阴阳变化的爻 | 变化的关键节点 |

### 2. 变爻的来源

变爻的产生有几种方式：

- **蓍草法**：通过大衍之数的推算，得6、7、8、9四个数字。其中6为老阴（变爻），9为老阳（变爻），7为少阳（不变），8为少阴（不变）
- **铜钱法**：三枚铜钱的组合中，三个背（老阳，变爻）和三个字（老阴，变爻）
- **时间起卦法**：根据动爻的判定确定变爻

### 3. 变爻的判断规则

根据变爻的数量不同，断卦的规则也不同：

| 变爻数量 | 断卦规则 | 依据 |
|----------|----------|------|
| 无变爻 | 以本卦卦辞断 | 《系辞》 |
| 一个变爻 | 以本卦变爻爻辞断 | 朱熹法 |
| 两个变爻 | 以本卦两个变爻爻辞断，上爻为主 | 朱熹法 |
| 三个变爻 | 以本卦卦辞为主，变卦卦辞为辅 | 朱熹法 |
| 四个变爻 | 以变卦中两个不变爻辞断，下爻为主 | 朱熹法 |
| 五个变爻 | 以变卦中一个不变爻辞断 | 朱熹法 |
| 六个变爻 | 乾坤看用辞，余卦以变卦卦辞断 | 朱熹法 |

## 二、变卦的深层含义

### 1. 阴阳转化

变卦的本质是阴阳转化。阳极生阴，阴极生阳，这是宇宙运行的根本规律。

> 《道德经》云："反者道之动。"物极必反，盛极而衰，变卦正是这种哲学思想的体现。

### 2. 本卦与变卦的关系

**本卦为体，变卦为用**：

- 本卦反映当前的形势和条件
- 变卦揭示发展的方向和结局
- 从本卦到变卦的过程，就是事物从现状到未来的发展轨迹

### 3. 变卦的吉凶判断

变卦的吉凶需要结合多方面因素：

1. **本卦卦德**：本卦的基本性质决定了事物的基础
2. **变爻位置**：变爻所在的爻位影响变化的性质
3. **变卦卦德**：变卦的性质揭示变化后的状态
4. **卦序关系**：本卦与变卦在六十四卦序中的关系

## 三、互卦理论详解

### 1. 互卦的定义

**互卦**，又称"中间卦"，是从本卦的二三四爻和三四五爻各取一卦而组成的新卦。

### 2. 互卦的取法

以本卦六爻为序：

- **下互卦（内互）**：取本卦的二、三、四爻组成
- **上互卦（外互）**：取本卦的三、四、五爻组成

**举例**：以**水雷屯卦**为例

```
上六 ━ ━ （第六爻）
九五 ━━━ （第五爻）
六四 ━ ━ （第四爻）  ← 外互取三、四、五爻
六三 ━ ━ （第三爻）  ← 内互取二、三、四爻
六二 ━ ━ （第二爻）
初九 ━━━ （第一爻）
```

- 内互卦：二三四爻为坤（☷），阴阴阴
- 外互卦：三四五爻为艮（☶），阴阴阳
- 互卦为**地山谦卦**

### 3. 互卦的意义

互卦代表事物发展过程中**内在的、隐藏的因素**：

| 层次 | 含义 | 时间性 |
|------|------|--------|
| 本卦 | 事物的外在表现和起始状态 | 过去/现在 |
| 互卦 | 事物的内在本质和中间过程 | 现在/过程 |
| 变卦 | 事物的发展趋向和最终结果 | 未来/结果 |

> 互卦揭示的是"事情的本质"，往往比本卦更接近事物的真实面貌。

## 四、互卦的实战应用

### 1. 互卦与疾病判断

在占断疾病时，互卦往往代表：
- 病因的根源所在
- 病情的发展趋势
- 隐藏的并发症

### 2. 互卦与事业判断

在占断事业时：
- 本卦显示当前工作的表面情况
- 互卦揭示工作中的核心问题和人际关系
- 变卦预示事业的发展方向

### 3. 互卦的连环分析

有些派别还主张对互卦再取互卦（二次互卦），层层深入，挖掘事物更深层的内涵。但一般以一次互卦为主，二次互卦为辅。

## 五、变卦与互卦的综合运用

### 综合分析框架

断卦时，应当建立**本卦—互卦—变卦**三位一体的分析框架：

1. **看本卦**：了解当前形势的基本格局
2. **看互卦**：把握内在的、核心的因素
3. **看变卦**：预判未来的发展方向
4. **比较三卦**：观察从本卦经互卦到变卦的变化趋势

### 变化趋势的判断

| 趋势类型 | 表现 | 吉凶 |
|----------|------|------|
| 由吉转吉 | 本卦吉→互卦吉→变卦吉 | 大吉 |
| 由吉转凶 | 本卦吉→互卦平→变卦凶 | 先吉后凶 |
| 由凶转吉 | 本卦凶→互卦平→变卦吉 | 先凶后吉 |
| 由凶转凶 | 本卦凶→互卦凶→变卦凶 | 大凶 |

### 注意事项

1. **不可孤立看变卦**：变卦必须与本卦对照分析，脱离本卦谈变卦没有意义
2. **变爻数量影响权重**：变爻越多，变化越大，变卦的权重越高
3. **互卦为辅不为主**：互卦是辅助工具，不可替代本卦和变卦的核心地位
4. **结合爻辞**：变卦分析不可脱离爻辞，纯理论推导容易失准

## 学习要点总结

1. **变卦**是本卦中变爻翻转后所得的新卦，代表事物的发展趋向

2. **变爻判断规则**遵循朱熹法，根据变爻数量选择不同断法

3. **互卦**由本卦二三四爻和三四五爻分别组成，揭示事物的内在本质

4. **本卦—互卦—变卦**构成完整的分析链条，分别对应起始、过程、结果

5. 变卦的吉凶需与本卦对照分析，趋势判断比单一判断更重要

6. 实践中应以本卦为主、互卦为辅、变卦为参考，综合运用方能精准

变卦与互卦理论是周易预测从静态分析走向动态分析的关键，掌握这一技法，你的断卦水平将实现质的飞跃。
