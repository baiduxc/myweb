# 紫微斗数入门：十四主星概览

## 什么是紫微斗数

紫微斗数，简称"紫微"，是中国传统命理学中的重要流派。它以人的出生年月日时为基础，排出命盘，将诸星分布于十二宫位中，通过星曜的庙旺落陷、吉凶组合来推断命运。

## 十四主星

十四主星是紫微斗数的核心，可分为北斗星系和南斗星系两大体系。

### 北斗星系
| 星名 | 特性 |
|------|------|
| **紫微星** | 帝星，尊贵稳重，有领导力 |
| **天机星** | 智谋之星，聪明善变，思虑缜密 |
| **太阳星** | 光明磊落，热情大方，有博爱之心 |
| **武曲星** | 刚毅果断，重义轻利，善理财 |
| **天同星** | 福星，温和随性，重享受 |
| **廉贞星** | 复杂多变，才华横溢，感情丰富 |

### 南斗星系
| 星名 | 特性 |
|------|------|
| **天府星** | 库星，稳重守成，善管理 |
| **太阴星** | 温柔细腻，内敛含蓄 |
| **贪狼星** | 欲望之星，多才多艺，善交际 |
| **巨门星** | 口舌之星，能言善辩，喜研究 |
| **天相星** | 印星，公正平和，善协调 |
| **天梁星** | 荫星，慈悲为怀，有长者风范 |
| **七杀星** | 将星，果断勇猛，不喜受约束 |
| **破军星** | 耗星，变动创新，敢为人先 |

## 主星的庙旺落陷

主星在不同宫位有不同的力量强弱，从庙、旺、得地、平、陷到落陷，力量差异很大。同一颗星，庙旺时展现正面特性，落陷时则易显负面。

## 学习建议

紫微斗数体系庞大，建议按以下顺序学习：

1. 先熟悉十四主星的基本特性
2. 了解十二宫位的含义
3. 学习吉凶辅星的作用
4. 掌握星曜组合的推断方法

> **提示**：紫微斗数注重星曜的组合和宫位的互动，不可孤立地看某一颗星。
