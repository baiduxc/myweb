# 六爻六亲六神详解

六爻占断以六亲定"所测之事"，以六神定"事的性质"。六亲六神配合使用，构成了六爻判断的核心分析体系。

---

## 六亲详解

六亲以卦宫五行为基准，按五行生克关系推导：

---

### 一、父母爻

**关系**：生我者（卦宫五行所生之爻）

**基本含义**：庇护、承载、文书

| 方面 | 具体代表 |
|------|----------|
| 人物 | 父母、长辈、老师、领导 |
| 物品 | 房屋、衣物、文书、证件、合同 |
| 事情 | 考试、学历、通知、消息 |
| 特性 | 辛苦、操劳、保护、约束 |

**旺相表现**：
- 文书顺利，考试有利
- 有长辈庇护
- 房屋安稳

**休囚表现**：
- 文书不利，考试受阻
- 缺乏庇护
- 房产不安

---

### 二、兄弟爻

**关系**：同我者（与卦宫五行同类之爻）

**基本含义**：竞争、阻隔、帮扶

| 方面 | 具体代表 |
|------|----------|
| 人物 | 兄弟姐妹、朋友、同事、竞争对手 |
| 物品 | 同类物品、竞争品 |
| 事情 | 争夺、阻隔、破财、合伙 |
| 特性 | 争夺、帮扶、消耗 |

**旺相表现**：
- 竞争激烈，破财耗损
- 朋友多但花费大
- 事情有阻隔

**休囚表现**：
- 竞争减少
- 朋友少但花费少
- 阻力较小

> 兄弟爻为"劫财之神"，求财时多为不利。

---

### 三、妻财爻

**关系**：我克者（卦宫五行所克之爻）

**基本含义**：财物、利益、妻室

| 方面 | 具体代表 |
|------|----------|
| 人物 | 妻子（男占）、下属、佣人 |
| 物品 | 钱财、粮食、物资、车辆 |
| 事情 | 财运、生意、理财、婚姻（男） |
| 特性 | 柔顺、利益、可控 |

**旺相表现**：
- 财运亨通，收入丰厚
- 妻子贤惠
- 物资充裕

**休囚表现**：
- 财运不佳，入不敷出
- 婚姻不顺
- 物资匮乏

> 妻财爻为"财源"，求财时最为关键。

---

### 四、子孙爻

**关系**：我生者（卦宫五行所生之爻）

**基本含义**：福气、子嗣、医药

| 方面 | 具体代表 |
|------|----------|
| 人物 | 子女、晚辈、学生 |
| 物品 | 医药、器具、娱乐品 |
| 事情 | 福气、解忧、制官、投资 |
| 特性 | 乐观、释放、消散 |

**旺相表现**：
- 子女有出息
- 有福气，少忧愁
- 医药有效，疾病可愈
- 制官鬼有力

**休囚表现**：
- 子女不顺
- 福气不足
- 医药无效

> 子孙爻为"福神"，制官鬼、解忧愁。占病时子孙为药。

---

### 五、官鬼爻

**关系**：克我者（克卦宫五行之爻）

**基本含义**：官府、灾祸、丈夫

| 方面 | 具体代表 |
|------|----------|
| 人物 | 丈夫（女占）、官员、领导、小人 |
| 物品 | 官府、衙门、武器、凶器 |
| 事情 | 官讼、灾祸、疾病、盗贼 |
| 特性 | 威严、压制、凶险 |

**旺相表现**：
- 当官掌权（求官为吉）
- 疾病严重（占病为凶）
- 官司缠身

**休囚表现**：
- 无官运
- 灾祸减轻
- 疾病减轻

> 官鬼爻为"祸福之神"，求官为吉，占病为凶。

## 六神详解

六神（又称六兽）按日干起排，辅助判断事的性质与状态。

### 排法口诀

> 甲乙起青龙，丙丁起朱雀，戊起勾陈，己起螣蛇，庚辛起白虎，壬癸起玄武。

从初爻起排，依次向上：

| 六神 | 颜色 | 方位 | 主要含义 |
|------|------|------|----------|
| **青龙** | 青/绿 | 东 | 喜庆、吉祥、财运、婚姻 |
| **朱雀** | 红 | 南 | 口舌、是非、文书、消息 |
| **勾陈** | 黄 | 中 | 纠缠、田土、迟滞、牵连 |
| **螣蛇** | 杂 | — | 惊恐、怪异、虚惊、蛇蝎 |
| **白虎** | 白 | 西 | 血光、丧事、凶险、刚烈 |
| **玄武** | 黑 | 北 | 暗昧、欺诈、暧昧、盗贼 |

---

### 六神临六亲的组合判断

| 六神+六亲 | 含义 |
|------------|------|
| 青龙+妻财 | 正当之财，喜事得财 |
| 青龙+父母 | 喜庆文书，吉庆考试 |
| 朱雀+官鬼 | 口舌官非，惊扰不安 |
| 朱雀+兄弟 | 争吵纷争，口角是非 |
| 勾陈+妻财 | 财运迟滞，田土之财 |
| 勾陈+兄弟 | 纠缠不清，拖延阻隔 |
| 螣蛇+官鬼 | 虚惊怪异，噩梦不安 |
| 螣蛇+妻财 | 财来虚幻，投资不实 |
| 白虎+官鬼 | 血光之灾，疾病凶险 |
| 白虎+父母 | 丧事文书，凶讯传来 |
| 玄武+妻财 | 暗中得财，偏门之财 |
| 玄武+官鬼 | 盗贼暗算，阴谋诡计 |

## 用神的选取

用神是断卦的关键。不同事类，用神不同：

| 事类 | 用神 | 忌神 | 原神 |
|------|------|------|------|
| 求财 | 妻财 | 兄弟 | 子孙 |
| 考试 | 父母 | 子孙 | 官鬼 |
| 疾病 | 官鬼(病)/子孙(药) | 子孙(克官)/官鬼(病重) | — |
| 婚姻(男) | 妻财 | 兄弟 | 子孙 |
| 婚姻(女) | 官鬼 | 子孙 | 妻财 |
| 官讼 | 官鬼 | 子孙 | 妻财 |
| 出行 | 父母(交通) | 子孙 | 官鬼 |
| 失物 | 妻财 | 兄弟 | 子孙 |

> **用神**：所测之事的代表爻
> **忌神**：克制用神之爻（不利）
> **原神**：生扶用神之爻（有利）
> **仇神**：生扶忌神之爻（间接不利）

## 六亲旺衰的判断

### 月令为主

| 月令关系 | 旺衰 |
|----------|------|
| 月令生扶六亲 | 旺相 |
| 月令与六亲同类 | 有力 |
| 月令克制六亲 | 休囚 |
| 六亲克月令 | 废退 |
| 六亲生月令 | 泄气 |

### 日辰辅助

日辰对六亲的作用类似于月令，但力量稍弱。日辰可以冲动爻、合住爻、入墓等。

### 动爻影响

动爻对其他爻有直接作用力：
- 动爻生扶 → 增强
- 动爻克制 → 削弱

---

> **学习要点**：六亲是六爻断卦的"骨架"，六神是"血肉"。学习时先掌握六亲的确定方法和基本含义，再结合六神辅助判断。用神的选取是关键，选错用神则全盘皆误。多练习不同事类的用神选取，是提高断卦准确率的基础。
