# 风水八宅法详解：东四命与西四命的住宅风水

八宅风水是传统理气风水的重要流派，以简洁明了著称。它将宅命分为八种，人命也分八种，讲究宅命相配，以八方位论吉凶。本文将详细讲解八宅风水的理论与实践应用。

## 一、八宅风水的基本概念

### 1.1 八宅的定义

八宅风水将住宅按坐向分为八类：

| 宅名 | 坐向 | 五行 |
|------|------|------|
| 坎宅 | 坐北朝南 | 水 |
| 离宅 | 坐南朝北 | 火 |
| 震宅 | 坐东朝西 | 木 |
| 兑宅 | 坐西朝东 | 金 |
| 巽宅 | 坐东南朝西北 | 木 |
| 乾宅 | 坐西北朝东南 | 金 |
| 坤宅 | 坐西南朝东北 | 土 |
| 艮宅 | 坐东北朝西南 | 土 |

### 1.2 东西四宅

八宅又分为东四宅和西四宅：

| 分类 | 宅名 | 共同五行属性 |
|------|------|-------------|
| 东四宅 | 坎、离、震、巽 | 水火木，相生相合 |
| 西四宅 | 乾、坤、艮、兑 | 金土，相生相合 |

> 东四宅宜居东四命之人，西四宅宜居西四命之人，此为宅命相配。

### 1.3 命卦的计算

命卦是根据出生年份计算的八卦属性：

**男命计算法**：
1. 用（100-出生年份后两位）÷ 9
2. 取余数，按表定命卦

**女命计算法**：
1. 用（出生年份后两位-4）÷ 9
2. 取余数，按表定命卦

**命卦对应表**：

| 余数 | 命卦 | 归属 |
|------|------|------|
| 1 | 坎水 | 东四命 |
| 2 | 坤土 | 西四命 |
| 3 | 震木 | 东四命 |
| 4 | 巽木 | 东四命 |
| 5 | 男坤/女艮 | 西四命 |
| 6 | 乾金 | 西四命 |
| 7 | 兑金 | 西四命 |
| 8 | 艮土 | 西四命 |
| 9 | 离火 | 东四命 |

> 注：余数为0时，男命为离（9），女命为离（9）；余数为5时，男命归坤，女命归艮。

## 二、四吉方与四凶方

### 2.1 四吉方详解

八宅风水中，每宅有四吉方：

| 名称 | 含义 | 能量级别 | 主要作用 |
|------|------|----------|----------|
| 生气 | 生机蓬勃 | 上吉 | 财运、事业、生命力 |
| 天医 | 天赐良医 | 次吉 | 健康、贵人、机遇 |
| 延年 | 延年益寿 | 三吉 | 人缘、婚姻、合作 |
| 伏位 | 伏命安身 | 四吉 | 稳定、守成、安宁 |

**生气方**：
- 能量最强，主财运亨通、事业腾飞
- 宜设大门、主卧、办公桌
- 开业动土，选此方最佳

**天医方**：
- 主健康长寿，贵人相助
- 宜设卧室、书房
- 有病之人居此方可助康复

**延年方**：
- 主人缘和美，婚姻顺利
- 宜设主卧（尤其是夫妻房）
- 利于谈判、签约、催桃花

**伏位方**：
- 主平稳安定，守成有余
- 宜设寝室、休息区
- 性格沉稳者宜居此方

### 2.2 四凶方详解

| 名称 | 含义 | 凶险级别 | 主要影响 |
|------|------|----------|----------|
| 绝命 | 断绝生机 | 大凶 | 破财损丁、意外伤亡 |
| 五鬼 | 五鬼闹宅 | 次凶 | 官非口舌、小人暗害 |
| 六煞 | 六亲不认 | 中凶 | 是非纷争、情感劫难 |
| 祸害 | 祸从天降 | 小凶 | 疾病损耗、事业阻滞 |

**绝命方**：
- 最凶之位，主重大灾祸
- 不可设大门、主卧、灶台
- 若为卫生间、仓库可得化解

**五鬼方**：
- 主官非、口舌、小人
- 不宜设卧室、办公桌
- 可设储藏室、次卧

**六煞方**：
- 主情感纠纷、是非口角
- 不宜年轻人居此方
- 可作客房、杂物间

**祸害方**：
- 主疾病缠身、运气不佳
- 不宜久居
- 可设卫生间排气煞气

## 三、八宅游年法

### 3.1 游年星的由来

游年法是确定八宅八方吉凶的方法，其核心是大游年歌：

> **大游年歌**：
> 乾六天五祸绝延生，坎五天生延绝祸六
> 艮六绝祸生延天五，震延生祸绝五天六
> 巽天五六祸生绝延，离六五绝延祸生天
> 坤天延绝生祸五六，兑生祸延绝六五天

### 3.2 游年星的排列规则

游年星按八卦变爻原理排列：

1. 从伏位（本宫）起
2. 每变一爻，得一新卦
3. 新卦对应游年星

**变爻规则表**：

| 变爻数 | 游年星 | 变法 |
|--------|--------|------|
| 上爻变 | 生气 | 一爻变 |
| 中爻变 | 五鬼 | 二爻变 |
| 下爻变 | 天医 | 三爻变 |
| 上中变 | 绝命 | 四爻变 |
| 中下变 | 祸害 | 五爻变 |
| 上下变 | 六煞 | 六爻变 |
| 三爻全变 | 延年 | 七爻变 |
| 不变 | 伏位 | 本宫 |

### 3.3 游年星的实际应用

应用步骤：

1. 确定住宅坐向，定宅卦
2. 根据大游年歌，找出八方游年星
3. 吉方设重要房间，凶方设次要用房
4. 命卦与宅卦相配，协调人与环境

## 四、八宅风水的应用技法

### 4.1 住宅选址

选择住宅时，需考虑：

| 因素 | 要求 |
|------|------|
| 宅命相配 | 东四命选东四宅，西四命选西四宅 |
| 大门方位 | 开在吉方（生气、天医、延年） |
| 主卧方位 | 在吉方，夫妻宜延年方 |
| 灶位 | 坐凶向吉，以煞制煞 |

### 4.2 室内布局

**大门布局**：
- 门开吉方，纳气聚财
- 门不宜冲煞：梯冲、墙冲、路冲
- 门内宜有玄关缓冲

**主卧布局**：
- 居吉方，夫妻宜延年方
- 床头靠墙，不压梁
- 不对卫生间门

**厨房布局**：
- 灶坐凶向吉
- 灶不对门，不压中宫
- 水火不相克

**卫生间布局**：
- 宜设凶方，以煞引煞
- 不压中宫，不正对门
- 保持清洁干燥

### 4.3 办公室风水

| 部门 | 宜设方位 |
|------|----------|
| 老板办公室 | 生气方或延年方 |
| 财务室 | 天医方或伏位方 |
| 业务部 | 生气方 |
| 人事部 | 延年方 |
| 仓库 | 祸害或六煞方 |

八宅风水可与以下流派结合：

- **三元九运**：引入时运因素
- **玄空飞星**：精确计算气场变化
- **峦头形法**：结合地形形势判断

### 5.3 八宅与玄飞结合

在八宅基础上，引入玄空飞星：

1. 先定宅卦，确定八宅八方吉凶
2. 再排飞星盘，看当运旺衰
3. 八宅吉方+飞星旺星=上吉
4. 八宅凶方+飞星衰星=大凶

## 六、八宅风水实例分析

### 实例：坎宅风水布局

**基本情况**：
- 住宅：坐北朝南坎宅
- 宅主：1980年庚申年生，男命，命卦坤（西四命）

**问题分析**：
- 坎宅为东四宅，宅主为西四命
- 宅命不相配，需加强调理

**八宅八方分布**：
| 方位 | 游年星 | 用途建议 |
|------|--------|----------|
| 东南 | 生气 | 大门、主卧首选 |
| 东 | 天医 | 书房、次卧 |
| 南 | 延年 | 夫妻房、客厅 |
| 北 | 伏位 | 稳定房间 |
| 西南 | 绝命 | 卫生间 |
| 西 | 六煞 | 储藏室 |
| 西北 | 五鬼 | 杂物间 |
| 东北 | 祸害 | 次要房间 |

**调理建议**：
1. 将大门改在东南生气方
2. 主卧设在南延年方
3. 西南绝命方设卫生间
4. 命卦与宅卦不配，可佩戴开运饰品化解
