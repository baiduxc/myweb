# 六爻占卜基础入门

六爻占卜，又称"纳甲占法"或"火珠林法"，是周易预测学中最常用的占卜方法之一。它以铜钱摇卦获取卦象，通过六亲、六神、世应、动变等要素综合推断事物的吉凶。

---

## 六爻占卜的基本流程

```
占卜问事 → 摇卦起卦 → 装卦纳甲 → 分析判断 → 得出结论
```

---

## 一、起卦方法

### 传统铜钱法

准备三枚铜钱（或硬币），诚心默念所问之事，摇动后掷出，记录结果：

| 结果 | 判定 | 爻象 |
|------|------|------|
| 三枚字面朝上 | 老阴（变爻） | ⚋ → ⚊ |
| 两字一背 | 少阴 | ⚋ |
| 一字两背 | 少阳 | ⚊ |
| 三枚背面朝上 | 老阳（变爻） | ⚊ → ⚋ |

> 摇六次，从下往上记录，得到完整的六爻卦。

### 数字起卦法

以问事时的年月日时数字，通过特定公式起卦。适合无法摇卦时使用。

---

## 二、装卦纳甲

得到卦象后，需要"装卦"——为每一爻配置天干地支、六亲、六神。

### 纳甲法

将天干地支配入六爻：

| 卦象 | 内卦天干 | 外卦天干 |
|------|----------|----------|
| 乾、震 | 甲 | 壬 |
| 坤、巽 | 乙 | 癸 |
| 坎 | 戊 | 戊 |
| 离 | 己 | 己 |
| 艮 | 丙 | 丙 |
| 兑 | 丁 | 丁 |

地支按固定顺序配入：

- 阳卦：子、寅、辰、午、申、戌（顺行）
- 阴卦：未、巳、卯、丑、亥、酉（逆行）

### 安世应

| 卦种 | 世爻位置 | 应爻位置 |
|------|----------|----------|
| 八纯卦 | 上爻（六爻） | 三爻 |
| 一世卦 | 初爻 | 四爻 |
| 二世卦 | 二爻 | 五爻 |
| 三世卦 | 三爻 | 上爻 |
| 四世卦 | 四爻 | 初爻 |
| 五世卦 | 五爻 | 二爻 |
| 游魂卦 | 四爻 | 初爻 |
| 归魂卦 | 三爻 | 上爻 |

> **世爻**代表问卦者本人，**应爻**代表对方或所问之事。

---

## 三、六亲系统

六亲是六爻判断的核心要素，以卦宫五行为基准推导：

| 六亲 | 关系 | 代表含义 |
|------|------|----------|
| **父母** | 生我者 | 文书、证件、长辈、房屋 |
| **兄弟** | 同我者 | 兄弟、朋友、竞争、阻隔 |
| **妻财** | 我克者 | 钱财、妻子、物资、粮食 |
| **子孙** | 我生者 | 子女、福气、医药、解忧 |
| **官鬼** | 克我者 | 官府、丈夫、疾病、盗贼 |

### 六亲旺衰

六亲的力量取决于月令和日辰：

- 月令生扶 → 旺
- 月令克制 → 衰
- 日辰冲克 → 弱

---

## 四、六神（兽）

六神是辅助判断的参考要素，按日干推排：

| 六神 | 日干起位 | 特性 |
|------|----------|------|
| **青龙** | 甲日起 | 喜庆、吉祥、财运 |
| **朱雀** | 乙日起 | 口舌、是非、文书 |
| **勾陈** | 丙日起 | 纠缠、拖延、田土 |
| **螣蛇** | 丁日起 | 惊恐、怪异、虚惊 |
| **白虎** | 戊日起 | 血光、丧事、凶险 |
| **玄武** | 己日起 | 暗昧、欺诈、暧昧 |

---

## 五、动爻与变卦

### 动爻

老阴和老阳的爻为动爻，代表事物变化的关键点。

- **动爻**：当前状态与变化趋势
- **变爻**：变化后的结果

### 变卦

动爻阴阳互变后得到的新卦，反映事物的变化方向。

> 动爻是断卦的关键，有动则有变，有变则有吉凶。

---

## 六、断卦基本步骤

### 第一步：审卦

确定卦名、卦宫、世应位置。

### 第二步：定用神

根据所问之事，确定用神（关键六亲）：

| 所问之事 | 用神 |
|----------|------|
| 求财 | 妻财爻 |
| 考试 | 父母爻 |
| 疾病 | 官鬼爻（忌神）或子孙爻（药） |
| 婚姻 | 妻财爻（男）/官鬼爻（女） |
| 官讼 | 官鬼爻 |
| 出行 | 父母爻（交通工具） |

### 第三步：看旺衰

用神在月令日辰下的旺衰状态：

- **旺相**：有力，吉
- **休囚**：无力，凶
- **入墓**：受困，难成
- **空亡**：落空，不成

### 第四步：看动变

动爻对用神的作用：

- 动爻生扶用神 → 有利
- 动爻克制用神 → 不利
- 用神发动 → 事情主动变化

### 第五步：综合判断

综合月令、日辰、动变、六神，给出吉凶判断与时间推断。

---

## 常用断语

| 状况 | 断语 |
|------|------|
| 用神旺相不动 | 事情平稳，按部就班 |
| 用神旺动 | 事情快速变化，主动争取 |
| 用神休囚受克 | 事情不利，多阻难成 |
| 用神空亡 | 暂时难成，待出空可成 |
| 忌神发动克用 | 有阻力，事情不利 |
| 原神发动生用 | 有助力，事情顺利 |
| 用神入墓 | 事情受困，待冲墓可解 |

---

## 占卜注意事项

1. **心诚则灵**：占卜时需诚心专注，心不诚则卦不准
2. **一事一卦**：每次占卜只问一件事
3. **不疑不占**：心中有疑才占，无端占卜不可取
4. **时间限制**：同一件事短期内不宜反复占卜
5. **不可迷信**：占卜结果仅供参考，命运最终掌握在自己手中

---

> **学习要点**：六爻占卜的核心在于"用神"的确定和旺衰的判断。初学时建议从简单的事情开始练习，如占天气、占失物等，逐步积累经验。多实践、多总结，断卦水平会逐步提高。
